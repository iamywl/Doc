# BK21 CS 분야 우수학술대회 등급 체계

## 1. 등급 체계 개요

BK21(Brain Korea 21) 사업은 CS 분야 학술대회를 **2개 구분(Top-tier / 2nd-Tier)**, **4단계 인정 IF 점수(4, 3, 2, 1)** 로 분류한다.

| 인정 IF | 구분 | 한국어 통칭 | 설명 | 학술대회 수 |
|--------|---------|-----------|------|-----------|
| **4** | Top-tier | 최우수 | CS 분야 최정상급 학술대회 | 39개 |
| **3** | Top-tier | 우수 | CS 분야 상위급 학술대회 | 32개 |
| **2** | 2nd-Tier | 우량 | 국제적 수준의 학술대회 | 57개 |
| **1** | 2nd-Tier | 일반 우수 | 인정 대상 학술대회 | 60개 |

**총 188개 학술대회** (BK21 플러스 사업 CS 분야 우수국제학술대회 목록, 2018.3.1 기준)

## 2. 논문 유형별 점수 적용

| 논문 유형 | 점수 산정 |
|----------|----------|
| Regular 논문 | 인정 IF 그대로 부여 |
| Short 논문 | 인정 IF - 1점 |
| Spotlight 논문 | 인정 IF - 2점 |
| 워크숍/포스터 | IF 미부여 (0점) |

## 4. 저널(학술지) 평가

BK21은 저널에 대해 별도의 인정 IF 목록을 관리하지 않으며, **JCR(Journal Citation Reports) Impact Factor 원값**을 그대로 사용한다.

| 저널명 | SCI/SCIE | 대략적 IF | 비고 |
|--------|---------|----------|------|
| ACM Computing Surveys | SCIE | ~23.8 | CS 분야 최상위 서베이 저널 |
| IEEE TKDE | SCIE | ~8.9 | DB/데이터마이닝 최상위 저널 |
| VLDB Journal | SCIE | ~4.2 | DB 분야 최상위 저널 |
| ACM TODS | SCIE | ~2.2 | DB 분야 전통 최상위 저널 |
| IEEE TPDS | SCIE | ~5.6 | 병렬/분산 시스템 최상위 저널 |
| ACM TOCS | SCIE | ~2.1 | OS/시스템 분야 전통 최상위 저널 |
| CACM | SCIE | ~11.1 | CS 분야 대표 종합 저널 |
