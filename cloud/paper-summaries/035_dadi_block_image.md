# DADI: Block-Level Image Service

- **참고문헌 번호:** 35
- **저자:** Xu et al.
- **학회/출처:** ATC, 2020
- **논문 링크:** [Google Scholar](https://scholar.google.com/scholar?q=DADI+Block-Level+Image+Service)
- **분류:** 보완, 블록 수준 지연 로딩

## 요약

컨테이너 이미지를 블록 수준에서 지연 로딩(lazy loading)하여 배포 속도를 4배 향상시키는 이미지 서비스이다. 기존 레이어 기반 파일 시스템 이미지 대신 블록 디바이스로 이미지를 관리하며, 필요한 블록만 온디맨드로 로드하여 컨테이너 시작 시간을 대폭 단축한다.

## 핵심 기여

- 블록 수준 컨테이너 이미지 포맷 설계로 4배 빠른 배포 달성
- 온디맨드 지연 로딩으로 불필요한 데이터 전송 제거
- 기존 레이어 기반 이미지 대비 효율적인 스토리지 활용
- 대규모 컨테이너 클러스터에서의 확장성 검증

## Firecracker와의 관계

Firecracker microVM의 루트 파일 시스템 이미지 로딩을 최적화하는 데 적용 가능한 보완 기술이다. 블록 수준 지연 로딩은 Firecracker의 빠른 부팅 시간 목표와 결합하여 콜드 스타트를 더욱 단축할 수 있다.
