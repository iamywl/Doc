# CKI: Container Kernel Isolation using PKS

- **참고문헌 번호:** 81

- **학회/출처:** EuroSys 2025

- **분류:** 반대

## 요약

CKI는 Intel PKS(Protection Keys for Supervisor)를 활용하여 커널 수준에서 컨테이너를 격리하는 시스템이다. VM 기반 격리 없이 하드웨어 보호 키를 통해 커널 메모리 영역을 컨테이너별로 분리하여, 경량이면서도 강력한 격리를 제공한다.

## 핵심 기여

- Intel PKS를 활용한 커널 수준 컨테이너 격리 메커니즘
- VM 없이 하드웨어 기반 강력한 격리 달성
- 커널 메모리 영역의 컨테이너별 분리
- 기존 컨테이너 대비 최소한의 성능 오버헤드

## Firecracker와의 관계

Firecracker가 하이퍼바이저를 통한 VM 기반 격리를 사용하는 반면, CKI는 Intel PKS를 통한 커널 수준 격리라는 근본적으로 다른 접근법을 제시한다. VM 오버헤드 없이 하드웨어 기반 격리를 달성하므로, Firecracker의 대안적 격리 기술로 볼 수 있다.
