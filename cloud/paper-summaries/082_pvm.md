# PVM: Shadow Paging for Nested Secure Containers

- **참고문헌 번호:** 82
- **저자:** Huang et al.
- **학회/출처:** SOSP 2023
- **논문 링크:** https://scholar.google.com/scholar?q=PVM+Shadow+Paging+for+Nested+Secure+Containers
- **분류:** 확장

## 요약

PVM은 중첩 가상화(nested virtualization) 환경에서 섀도우 페이징 기법을 사용하여 보안 컨테이너를 효율적으로 지원하는 시스템이다. 중첩 가상화의 페이지 테이블 관리 오버헤드를 섀도우 페이징으로 최적화하여 보안 컨테이너의 성능을 개선한다.

## 핵심 기여

- 중첩 가상화 환경을 위한 최적화된 섀도우 페이징 메커니즘
- 보안 컨테이너의 페이지 테이블 관리 오버헤드 감소
- 중첩 VM 환경에서의 메모리 가상화 효율성 향상
- 보안과 성능을 동시에 고려한 중첩 컨테이너 설계

## Firecracker와의 관계

Firecracker microVM이 중첩 가상화 환경에서 실행될 때 발생하는 성능 저하를 PVM의 섀도우 페이징 기법으로 완화할 수 있다. 특히 클라우드 환경에서 Firecracker를 중첩 VM으로 실행하는 시나리오에서 메모리 관리 효율성을 높이는 데 기여한다.
