# UPM: User-guided Page Merging for Memory Deduplication

- **참고문헌 번호:** 78
- **저자:** Qiu et al.
- **학회/출처:** IEEE BigData 2023
- **논문 링크:** https://scholar.google.com/scholar?q=UPM+User-guided+Page+Merging+for+Memory+Deduplication
- **분류:** 확장

## 요약

UPM은 madvise 시스템 콜을 기반으로 사용자 주도 페이지 공유를 수행하는 메모리 중복 제거 시스템이다. 커널 수준의 자동 중복 제거(KSM) 대비 더 효율적으로 동작하며, 55%의 메모리 감소와 2배의 VM 밀도를 달성한다.

## 핵심 기여

- madvise 기반 사용자 주도 페이지 공유 메커니즘
- 55% 메모리 사용량 감소
- 호스트당 VM 밀도 2배 향상
- KSM 대비 효율적인 중복 제거 전략

## Firecracker와의 관계

Firecracker microVM의 밀도를 높이기 위해 UPM의 메모리 중복 제거 기법을 직접 활용할 수 있다. 특히 동일한 게스트 OS와 런타임을 사용하는 다수의 Firecracker microVM에서 공통 메모리 페이지를 공유하면 호스트당 VM 밀도를 크게 높일 수 있다.
