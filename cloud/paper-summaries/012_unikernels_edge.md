# Are Unikernels Ready for Serverless on the Edge?

- **참고문헌 번호:** 12
- **저자:** (Various Authors)
- **학회/출처:** IC2E, 2024
- **논문 링크:** [Google Scholar](https://scholar.google.com/scholar?q=Are+Unikernels+Ready+for+Serverless+on+the+Edge)
- **분류:** 유사

## 요약

이 연구는 엣지 컴퓨팅 환경에서 유니커널, Firecracker, Docker를 체계적으로 비교 분석한다. 리소스가 제한된 엣지 환경에서 각 기술의 부팅 시간, 메모리 사용량, 성능, 격리 수준을 벤치마크하여 유니커널의 엣지 서버리스 환경 적합성을 평가한다.

## 핵심 기여

- 엣지 환경에서의 유니커널 vs Firecracker vs Docker 비교 벤치마크
- 리소스 제약 환경에서의 각 기술 적합성 분석
- 엣지 서버리스를 위한 기술 선택 가이드라인 제시
- 유니커널의 엣지 환경 준비도(readiness) 평가

## Firecracker와의 관계

Firecracker를 엣지 환경에서의 주요 비교 대상으로 활용한다. 엣지의 리소스 제약 조건에서 Firecracker의 경량 VM 접근법과 유니커널의 초경량 접근법 간의 트레이드오프를 분석하는 직접적인 비교 연구이다.
