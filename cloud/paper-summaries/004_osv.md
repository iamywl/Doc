# OSv: Optimizing the Operating System for Virtual Machines

- **참고문헌 번호:** 4

- **학회/출처:** ATC, 2014
- **논문 링크:** [Google Scholar](https://scholar.google.com/scholar?q=OSv+Optimizing+the+Operating+System+for+Virtual+Machines)
- **분류:** 유사

## 요약

OSv는 클라우드 VM 환경에 최적화된 게스트 OS로, 기존 범용 OS의 불필요한 중복 기능을 제거하여 성능을 향상시켰다. 단일 애플리케이션 실행에 특화되어 있으며, 하이퍼바이저와의 중복 기능(스케줄링, 메모리 관리 등)을 최소화하여 오버헤드를 줄인다.

## 핵심 기여

- VM 환경에 최적화된 게스트 OS 설계
- 호스트-게스트 간 OS 기능 중복 제거
- 단일 주소 공간에서의 효율적인 애플리케이션 실행
- 기존 Linux 애플리케이션과의 바이너리 호환성 유지

## Firecracker와의 관계

Firecracker가 VMM 측면에서 경량화를 추구한다면, OSv는 게스트 OS 측면에서 VM 최적화를 추구한다. 두 기술은 상호 보완적으로 조합될 수 있으며, Firecracker MicroVM 위에서 OSv를 게스트로 실행하여 추가적인 성능 향상을 기대할 수 있다.
