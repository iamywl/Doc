# 제10장: 스마트팩토리 및 AI/빅데이터 연계

---

## 10.1 개요

현대오토에버는 H-Cloud 인프라를 기반으로 스마트팩토리(Smart Factory) 솔루션과 AI/빅데이터 플랫폼을 운영하고 있다[52][53]. 클라우드 기반의 스마트 MES(Manufacturing Execution System), AI, 디지털 트윈 등의 서비스를 통해 현대자동차그룹의 제조 혁신을 주도하고 있다.

---

## 10.2 스마트팩토리 솔루션

### 10.2.1 SFaaS (Smart Factory as a Service)

SFaaS는 현대오토에버가 개발한 클라우드 기반의 스마트팩토리 서비스이다[52]. 고객 중심형 온디맨드(On-Demand) 방식으로 제조 혁신 IT 서비스를 제공하며, 스마트 MES와 지능형 제조 플랫폼을 통합 서비스로 제공한다[52].

SFaaS의 핵심 구성요소는 다음과 같다:

- **스마트 MES**: 생산 계획, 공정 관리, 품질 관리, 설비 관리 등 제조 실행 시스템을 클라우드 기반으로 제공한다.
- **AI 기반 품질 관리**: 머신러닝 기반의 품질 예측 및 이상 탐지 기능을 제공한다.
- **디지털 트윈**: 물리적 공장의 디지털 복제본을 생성하여, 시뮬레이션 및 최적화를 수행한다.
- **IoT 플랫폼**: 공장 내 설비에서 수집되는 센서 데이터를 실시간으로 처리한다[52].

### 10.2.2 네오팩토리 (NNNEO Factory)

현대오토에버는 2025년 '소프트웨어 정의 공장(SDF, Software Defined Factory)' 개념을 도입한 차세대 스마트팩토리 솔루션 '네오팩토리(NNNEO Factory)'를 발표하였다[53][54]. 네오팩토리는 AI를 활용하여 생산, 품질, 데이터 관리 등 모든 제조 시스템을 소프트웨어 기반으로 통합하고 제어하는 솔루션이다.

네오팩토리의 핵심 특징은 다음과 같다[53][54]:

- **소프트웨어 정의 공장(SDF)**: 물리적 설비의 제어를 소프트웨어로 추상화하여, 유연한 생산 라인 재구성이 가능하다.
- **AI 통합**: AI를 활용한 생산 최적화, 품질 예측, 설비 예측 정비 등의 기능을 제공한다.
- **VR/AR 활용**: 가상현실(VR) 및 증강현실(AR) 기술을 활용한 작업자 교육 및 원격 지원을 제공한다[54].
- **클라우드 네이티브**: H-Cloud 인프라 위에서 클라우드 네이티브 방식으로 운영된다.

### 10.2.3 Dell/Intel 제조 엣지 파트너십

2024년 4월 Dell Technologies, 현대오토에버, Intel은 제조 엣지 솔루션 확장을 위한 3자 파트너십을 체결하였다[134]. 이 파트너십의 핵심 내용은 다음과 같다:

- **NeoFactory IoT Blueprint**: 현대오토에버의 NeoFactory IoT 소프트웨어가 Dell Validated Design for Manufacturing Edge와 통합되어, Dell NativeEdge 엣지 운영 플랫폼 위에서 공장 인프라 배포를 단순화한다[134].
- **Dell NativeEdge**: Dell의 엣지 운영 소프트웨어 플랫폼으로, 멀티 애플리케이션 관리와 보안을 유지하면서 엣지 인프라 배포를 자동화한다[135].
- **Intel OpenVINO**: Intel의 AI 개발자 도구가 NativeEdge Blueprint 카탈로그에 포함되어, 엣지 환경에서의 AI/ML 모델 배포 및 추론을 지원한다[134].

이를 통해 공장 현장에서 IT/OT 인프라의 통합, AI 기반 공정 최적화, 장비 성능 실시간 모니터링, 이상 징후 감지, 예측 유지보수가 가능해졌다.

## 10.3 IoT 플랫폼

### 10.3.1 공장 IoT

현대오토에버의 IoT 플랫폼은 공장 내 설비에서 수집되는 센서 데이터를 실시간으로 수집, 처리, 분석한다[52]. 이 플랫폼의 주요 기능은 다음과 같다:

- **실시간 데이터 수집**: 공장 내 각종 설비, 센서, 로봇에서 발생하는 데이터를 실시간으로 수집한다.
- **빅데이터 분석**: 수집된 데이터를 분석하여 설비 이상, 고장 원인, 고장 패턴 등을 파악한다[52].
- **예측 정비**: 이상 징후가 포착되면 즉각적으로 대응하여 안전하고 지속적인 운영이 가능하게 한다[52].
- **데이터 레이크**: 공장 내 모든 데이터를 한 곳으로 모아 AI 기반으로 지속 개선하고 최적의 의사결정을 내도록 지원한다[52].

## 10.4 AI/빅데이터 플랫폼

### 10.4.1 통합 AI 플랫폼

현대오토에버는 고객이 빅데이터를 분석, 학습, 시뮬레이션할 수 있는 통합 AI 플랫폼을 제공하고 있다[14]. 고성능 엔진을 기반으로 대규모 데이터의 분석과 AI 모델 학습을 지원한다.

### 10.4.2 H Chat

현대오토에버는 생성형 AI를 활용한 기업용 AI 챗봇 서비스 'H Chat'을 제공하고 있다[55]. H Chat은 현대자동차그룹 내부 업무 지원을 위한 AI 어시스턴트로, H-Cloud 인프라 위에서 운영된다.

### 10.4.3 AI GPU 인프라

AI/ML 워크로드의 증가에 따라 H-Cloud의 AI GPU 인프라 투자가 확대되고 있다[8]. GPU 기반 베어메탈(HMetal) 서비스를 통해 딥러닝 모델 학습, 추론 등의 고성능 AI 워크로드를 처리한다. AI GPU 인프라 투자는 클라우드 매출 성장의 핵심 동인 중 하나이다[8].

## 10.5 클라우드와 스마트팩토리의 연계

H-Cloud와 스마트팩토리의 연계는 다음과 같은 구조로 이루어진다:

```
┌─────────────────────────────────────────────────┐
│                    H-Cloud                       │
│  ┌─────────┐  ┌──────────┐  ┌─────────────────┐ │
│  │ AI/ML   │  │ 데이터    │  │ SFaaS           │ │
│  │ 플랫폼  │  │ 레이크    │  │ (네오팩토리)     │ │
│  └────┬────┘  └─────┬────┘  └────────┬────────┘ │
│       │             │               │            │
│       └─────────────┼───────────────┘            │
│                     │                            │
└─────────────────────┼────────────────────────────┘
                      │
              ┌───────┴───────┐
              │               │
      ┌───────┴───────┐ ┌────┴──────────┐
      │   IoT 플랫폼   │ │  엣지 컴퓨팅   │
      │ (데이터 수집)   │ │  (실시간 처리) │
      └───────┬───────┘ └────┬──────────┘
              │              │
      ┌───────┴──────────────┴───────┐
      │                              │
      │     스마트팩토리 (공장)         │
      │  설비 | 센서 | 로봇 | PLC      │
      │                              │
      └──────────────────────────────┘
```

공장의 설비와 센서에서 수집된 데이터는 IoT 플랫폼과 엣지 컴퓨팅을 통해 전처리된 후, H-Cloud의 데이터 레이크에 저장된다. AI/ML 플랫폼이 이 데이터를 분석하고 학습하여, SFaaS를 통해 품질 예측, 설비 예측 정비, 생산 최적화 등의 인텔리전스를 공장에 제공한다.

## 10.6 차세대 ERP

현대자동차그룹은 차세대 ERP 시스템의 클라우드 전환을 추진하고 있다[8][24]. 현대제철의 SAP ERP 클라우드 전환 사례가 대표적이며[24], 이를 통해 다음과 같은 효과를 달성하고 있다:

- **비용 절감**: 온프레미스 대비 인프라 운영 비용을 절감하였다[24].
- **확장성 향상**: 업무량 변동에 따른 탄력적 자원 확장이 가능해졌다.
- **운영 효율성**: Rancher Prime과 SLES를 통해 IT 운영 프로세스가 간소화되었다[24].
- **마이크로서비스 전환**: 물류 분야에서 마이크로서비스 파일럿 프로젝트를 진행 중이며, 향후 MES 시스템으로 확대할 계획이다[24].
