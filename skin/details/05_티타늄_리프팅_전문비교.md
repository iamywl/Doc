## 1. 소프라노 티타늄이란?

- **장비**: 소프라노 티타늄 (이스라엘 ALMA社)
- **파장**: 755nm + 810nm + 1064nm 동시 조사 (1초 150회)
- **효과**: 리프팅 + 피부톤 개선 + 모공축소 + 제모
- **시술시간**: 약 30분 (풀페이스)
- **마취**: 없음
- **다운타임**: 없음 (즉시 일상생활 가능)
- **효과 유지**: 약 6개월~1년
- **권장 주기**: 4주 간격, 3~5회 이상

## 4. 제안받은 조건 분석 (60kJ / 12회 / 200만원)

### 단가 비교

| 비교 대상 | 1회 가격 | 에너지 | 비고 |
|-----------|----------|--------|------|
| **제안받은 가격** | **약 16.7만원** | 60kJ | 12회 200만원 |
| 전국 시세 (behealthy4u) | 75만원 | 60kJ | 체험 1회 |
| 리프트성형외과 (대구, 정품) | 60만원 | 풀페이스 | 할인가 |
| 톡스앤필 (이벤트 구독) | ~46.7만원 | 풀페이스 | 6회 기준 |
| 대구엣지성형외과 | 33만원 | 40kJ | 하프페이스 |
| 빌리브피부과 | 9.9만원 | 10kJ | 체험가 |

### 판단

- 60kJ 1회 시세: **60만~75만원**
- 제안 1회당: **16.7만원** → **시세의 22~28% 수준**
- **극단적으로 저렴** → 아래 사항 반드시 확인

### ⚠️ 반드시 확인할 사항

1. **정품 장비 여부**
   - 소프라노 티타늄 정품인지 확인
   - ALMA 정품 인증 스티커 확인
   - 소프라노 티타늄 공식 사이트 인증병원 확인: https://sopranotitanium.co.kr/?page_id=5784

2. **실제 에너지 세팅**
   - 시술 시 모니터에서 60kJ 맞는지 직접 확인
   - 너무 빠르게 끝나면 에너지 낮을 가능성

3. **시술 범위**
   - 풀페이스인지 확인 (하프/턱라인만이면 가격 다름)
   - 풀페이스 = 이마+양볼+턱라인+목

4. **시술 시간**
   - 풀페이스 기준 약 30분 소요
   - 20분 이하면 에너지 낮을 가능성 높음

5. **VAT 포함 여부**
   - 부가세 별도이면 실질 220만원

6. **시술 간격 및 기간**
   - 4주 간격 권장
   - 12회면 약 1년 소요

7. **계약 조건**
   - 환불 규정 확인
   - 중도 해지 시 위약금

## 5. 추천 행동

1. **리프트성형외과** (정품 인증병원)에 전화 → 60kJ 12회 패키지 가격 문의
2. **빌리브피부과**에 40kJ, 60kJ 가격 문의 (10kJ 9.9만원이니 단가 파악 가능)
3. **예쁨주의쁨의원**에 티타늄 60kJ 가격 문의 (메뉴에 티타늄 있음)
4. **유앤아이의원**에 티타늄 패키지 가격 문의 (60kJ 기준)
5. 제안받은 병원에 **정품 인증서/에너지 세팅 화면** 확인 요청
6. **강남언니 앱**에서 대구 지역 "티타늄" 필터링하여 추가 비교
