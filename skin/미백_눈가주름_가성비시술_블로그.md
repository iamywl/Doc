# 피부과 시술, 돈값 하는 게 뭘까? 미백 & 눈가 주름 가성비 시술 총정리

> 실제 피부과 시술명 + 논문 수치로 검증한 가성비 가이드 | 2026.03.22

---

피부과 메뉴판을 보면 피코슈어, 레블라이트, 리쥬란, 백옥주사... 종류가 너무 많죠. **"도대체 뭘 해야 돈값을 하지?"** 실제 피부과에서 받을 수 있는 시술명에 맞춰, 논문에서 보고된 **구체적인 수치**를 기반으로 가성비를 따져봤습니다.

---

## Part 1. 미백 시술 가성비 랭킹

### 1위. 처방 미백크림 (트리플 콤비네이션)

| 항목 | 내용 |
|------|------|
| 비용 | 월 2~5만원 (처방전 필요) |
| 근거 수준 | **Level I (최고)** |
| 효과 | 중등도~중증 기미에 가장 효과적 (Rivas & Pandya, 2013) |

하이드로퀴논 + 트레티노인 + 약한 스테로이드를 섞은 처방 크림입니다. 놀랍게도 **수십만원짜리 레이저 시술보다 이 크림의 근거 수준이 더 높습니다.** 초반에 약 40%에서 홍반이나 각질이 생길 수 있지만, 비용 대비 효과로는 압도적 1위.

### 2위. 화학 필링 (아쿠아필, 글리콜산 필링)

| 항목 | 내용 |
|------|------|
| 비용 | 회당 1.8~6만원 (아쿠아필 18,000원~) |
| 근거 수준 | Level II |
| 효과 | 기미 2차 치료 골드 스탠다드 (Indian Pigmentary Expert Group 컨센서스) |

2024년 메타분석에서 **레이저보다 부작용(색소침착)이 유의하게 적다**는 결과가 나왔습니다. 아시아인 피부에서도 안전. 크림이 안 먹힐 때 다음 단계로 가장 추천되는 시술입니다.

### 3위. 레블라이트 SI 토닝

| 항목 | 내용 |
|------|------|
| 비용 | 회당 7~15만원 / 10회 패키지 70만원~ |
| 근거 수준 | Level II~III |
| 효과 | **MASI 점수 52% 감소** (6.7→3.2), 58.8%가 50% 이상 개선 보고 |

많이들 하시는 레이저 토닝! 2022년 체계적 문헌고찰에서 **기미 점수가 절반 가까이** 떨어지는 효과가 확인됐습니다. 10회 이상 필요한 게 단점이고, 과도한 시술 시 반점성 저색소증(오히려 하얘지는 얼룩) 주의.

**꿀팁:** 2025년 논문에서 **경구 트라넥삼산 병합 시 효과가 유의하게 향상**되는 것으로 확인됐습니다!

### 4위. 피코슈어 토닝 / 피코토닝

| 항목 | 내용 |
|------|------|
| 비용 | 체험가 4.9만원 / 정상가 11~30만원/회 |
| 근거 수준 | Level II |
| 효과 | 1064nm이 755nm보다 유의하게 우수 (24주까지 지속) |

2023년 RCT에서 **피코초 1064nm이 피코슈어(755nm)보다 2회차부터 유의하게 더 효과적**이었습니다. 가격이 레블라이트SI보다 비싸면서 효과 차이가 확실하지 않아 가성비에서는 밀립니다.

### 5위. 백옥주사 (IV 글루타치온) — 주의!

| 항목 | 내용 |
|------|------|
| 비용 | 회당 3~4만원 / 10회 29만원 |
| 근거 수준 | **근거 매우 부족** |
| 효과 | 위약대조 RCT **단 1건**, 효과 불명확 |

**솔직히 말하면, 미백 목적의 백옥주사(IV 글루타치온)는 근거가 거의 없습니다.** 2025년 체계적 문헌고찰에서 "IV 경로의 미백 효과는 임상시험 부족으로 불명확"이라고 결론지었고, 필리핀 FDA는 **간독성·아나필락시스 위험**으로 경고까지 발령했습니다. 가격이 저렴해 보여도, 효과가 검증되지 않은 시술에 돈을 쓰는 건 가성비가 아닙니다.

### 미백 가성비 전략 요약

```
1단계: 처방 미백크림 (월 2~5만원) × 8~12주
  ↓ 부족하면
2단계: 아쿠아필/화학 필링 추가 (1.8~6만원/회) × 5~8회
  ↓ 그래도 부족하면
3단계: 레블라이트SI 토닝 10회 (70만원~) + 트라넥삼산 병합
```

## Part 2. 눈가 주름 시술 가성비 랭킹

### 1위. 주름 보톡스 (가성비 갑)

| 항목 | 내용 |
|------|------|
| 비용 | 국산 1부위 18,000원~ / 체험가 2,000원~ |
| 근거 수준 | **Level I (코크란 리뷰 — 최고 수준)** |
| 효과 | 65개 RCT, 14,919명 분석 → 까마귀발에 일관된 효과, 높은 만족도 |
| 지속 | 3~4개월 |
| 연간 비용 | 국산 기준 5.4~26만원 (연 3~4회) |

2021년 코크란 체계적 문헌고찰에서 **65개 임상시험, 약 15,000명**을 분석. 눈가(까마귀발)에서 안검하수(눈꺼풀 처짐) 부작용도 드물고, 효과는 3~5일 내 시작됩니다.

**연간 비용으로 따지면 가장 저렴하면서 근거가 가장 확실합니다.**

### 2위. 리쥬란 아이 (PDRN/폴리뉴클레오타이드)

| 항목 | 내용 |
|------|------|
| 비용 | 15~30만원/회 |
| 근거 수준 | Level II |
| 효과 | **눈가 주름 61.4% 감소** (218명 아시아인 연구, 12주) |
| 지속 | 6~12개월 |

Phase 3 RCT에서 식염수·HA(물광) 대비 **탄력, 콜라겐, 주름 깊이 모두 유의하게 우수**했습니다. 보톡스가 근육 이완(동적 주름)이라면, 리쥬란은 **콜라겐 재생(정적 주름)**에 강점. 지속 기간이 6~12개월로 길어서, 연간 비용은 보톡스와 비슷할 수 있습니다.

### 3위. 보톡스 + 슈링크 병합

| 항목 | 내용 |
|------|------|
| 비용 | 보톡스 + 슈링크 체험가 활용 시 5~12만원/세트 |
| 근거 수준 | Level II |
| 효과 | 보톡스+레이저 병합이 레이저 단독 대비 **유의하게 우수** (6개월 지속) |

2023년 무작위 맹검 연구에서 확인. 보톡스로 근육을 먼저 이완 → 2~3주 후 슈링크로 콜라겐 리모델링. **즉각 효과 + 장기적 구조 개선**의 시너지.

### 4위. 물광주사 / 스킨부스터

| 항목 | 내용 |
|------|------|
| 비용 | 체험가 29,000원~ / 정상가 30만원 |
| 근거 수준 | Level II~III |
| 효과 | **Lemperle 주름 등급 40% 감소** (3회 시술 후) |

2024년 파일럿 연구에서 주름 40% 개선 + 수분·탄력·광채 개선이 확인됐습니다. 하지만 **눈가 주름 전용으로는 보톡스·리쥬란보다 효과가 약합니다.** "전반적으로 피부 좀 좋아지고 싶다" 할 때 더 적합.

### 가성비 낮은 선택

- **써마지 FLX**: 100~200만원. 콜라겐 리모델링 효과는 있지만, 같은 예산으로 보톡스+리쥬란 여러 번 받는 게 훨씬 가성비 좋습니다.
- **필러**: 눈밑 꺼짐/다크서클에는 OK, 까마귀발 주름에는 부적합.

### 눈가 주름 가성비 전략 요약

```
1단계: 주름 보톡스 (국산 18,000원~) × 연 3~4회
       → 동적 주름(표정 주름) 즉각 개선
  ↓ 정적 주름도 개선하고 싶다면
2단계: 리쥬란 아이 추가 (15~30만원/회) × 연 2~3회
       → 콜라겐 재생, 주름 61.4% 감소, 6~12개월 지속
  또는
2단계: 보톡스 + 슈링크 병합 (체험가 활용)
       → 보톡스 후 2~3주 뒤 슈링크 시행
```

## 꼭 기억하세요

1. **비싼 게 좋은 게 아닙니다.** 처방 크림(Level I)이 피코슈어(Level II)보다, 국산 보톡스(Level I)가 써마지(Level II)보다 근거가 더 확실합니다.

2. **백옥주사는 미백 근거가 거의 없습니다.** 체계적 문헌고찰에서 IV 글루타치온의 위약대조 RCT는 **단 1건**. 미백 목적이면 처방 크림이 훨씬 확실합니다.

3. **단계적으로 접근하세요.** 가장 저렴하고 근거 확실한 것부터. 효과가 부족할 때 다음 단계로.

4. **자외선 차단이 모든 시술의 기본.** 아무리 비싼 시술을 해도 선크림 없으면 효과 반감.

5. **병합이 단독보다 낫습니다.** 토닝+트라넥삼산, 보톡스+슈링크/리쥬란 — 논문에서 검증된 조합입니다.

6. **체험가에 현혹되지 마세요.** 코스 치료가 필요한 시술(토닝 10회 등)은 정상가 기준으로 총 비용을 계산하세요.
