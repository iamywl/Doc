# 에듀커넥트 (EduConnect) — 글로벌 1:1 튜터-학생 실시간 매칭 플랫폼

> **예비창업패키지 사업계획서**
> 작성일: 2026년 3월
> 버전: 1.0

## □ 창업 아이템 개요

**에듀커넥트**는 전 세계 튜터와 학생을 AI로 매칭하는 **글로벌 1:1 실시간 교육 플랫폼**이다. 기존 교육 플랫폼은 언어 과목만 또는 국내만 서비스하는 한계가 있다. 본 플랫폼은 전과목 + 글로벌 양방향으로 "교육의 당근마켓"을 구현한다.

| 요약 | 내용 |
|------|------|
| **문제** | 사교육비 27.1조원(역대 최고), 적합한 튜터 찾기 어려움 72%, 교육 격차 5.2배 |
| **솔루션** | LLM 학습 진단 + 성향 매칭, WebRTC 화상수업, 적응형 학습경로, 녹화+AI 요약 |
| **성장** | 한국 → 한국어/K-문화 글로벌 → 50개국 양방향. 수업료 15-20%. 연매출 200억원 |

## 1. 문제 인식 (Problem)

- **한국 사교육비**: 27.1조원 (2024, 역대 최고, 통계청)
- **적합한 튜터 찾기 어려움**: 학부모 72% (한국교육개발원)
- **소득 격차**: 상위 20% vs 하위 20% 사교육비 5.2배
- **글로벌 교육 접근 불가**: 2.5억명 (UNESCO)

### 시장

| 시장 | 2024년 | 2030년 | CAGR |
|------|--------|--------|------|
| 글로벌 에듀테크 | $340B | $738B | 13.9% |
| 온라인 튜터링 | $12.1B | $36.2B | 20.0% |
| 언어 학습 | $61.6B | $115B | 11.0% |

### 성공 사례

- **italki**: $1B+ 추정, 150개국 30,000+ 튜터, 1,000만+ 학생
- **Preply**: $120M+ 투자, 185개국, 월 100만 수업, AI 매칭으로 선택 시간 70% 단축
- **Superprof**: 50개국, 2,300만+ 튜터, 1,000+ 과목
- **Wyzant**: 80,000+ 튜터, 300+ 과목

|---|--------|--------|--------|-------------|

| 학습 분석 | 없음 | 기본 | 없음 | AI 적응형 학습 |

## 2. 실현 가능성 (Solution)

### 핵심 기능

1. **AI 학습 진단 & 매칭**: 과목별 실력·학습 스타일·목표·예산 → 튜터 5명 추천 → 15분 무료 체험
2. **화상 수업 환경**: WebRTC 1:1 + 공유 화이트보드(tldraw) + 코드 에디터(Monaco) + 녹화 + AI 요약
3. **적응형 학습**: 약점 분석 → 다음 수업 추천 → AI 연습문제 자동 생성
4. **글로벌 매칭**: 한국어/K-POP/수학/코딩 튜터 → 글로벌 학생, 다국어 자막
5. **안전**: 신원 인증, 성범죄 조회, 수업 녹화, 미성년자 보호, 환불 보호

### 기술 스택

Next.js + React Native / NestJS + GraphQL / LLM 매칭 + IRT 학습분석 / Livekit(WebRTC) + tldraw + Monaco / Whisper STT + 번역 LLM / Stripe + 토스페이먼츠

## 3. 성장전략 (Scale-up)

| 수익원 | 비중 |
|--------|------|
| 수업료 수수료 (15-20%) | 55% |
| 프리미엄 구독 (월 9,900원) | 20% |
| B2B 학교/기업 | 20% |
| 콘텐츠 마켓 | 5% |

- **Phase 1**: 한국 수학·영어·코딩, 튜터 5,000명 + 학생 20,000명
- **Phase 2**: 한국어/K-문화 글로벌 + 전과목 확장
- **Phase 3**: 50개국 양방향, 수업 120만건/년, 연매출 500억원

### ESG
- **사회**: 교육 불평등 해소, 개도국 학생 → 선진국 튜터 접근
- **환경**: 통학 탄소배출 절감
- **지배구조**: 튜터 자유 가격, 투명한 리뷰
