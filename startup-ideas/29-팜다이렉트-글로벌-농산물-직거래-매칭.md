### 창업 아이템 개요

**팜다이렉트**는 아프리카·동남아·중남미 개발도상국의 소규모 농가와 선진국(한국·일본·EU·미국)의 레스토랑·소매업체·소비자를 AI로 직접 연결하는 **글로벌 크로스보더 농산물 직거래 매칭 플랫폼**이다. 기존 국제 농산물 무역은 4-7단계의 중간 유통을 거치며 원산지 농가 수취가가 최종 소비자가의 10-25%에 불과하다. 팜다이렉트는 AI 수요예측·가격최적화, 콜드체인 IoT 실시간 모니터링, 블록체인 기반 생산이력 추적을 통해 중간 유통 단계를 획기적으로 줄이고, 개도국 소농의 수입을 2-4배 향상시키면서 선진국 바이어에게는 더 신선하고 추적 가능한 농산물을 합리적 가격에 공급한다. 이는 기존 #20 팜커넥트(한국 내수 직거래)와 달리 **다국가 공급망, 크로스보더 물류, 국제 식품안전 인증(HACCP/GlobalG.A.P./FDA)** 컴플라이언스를 핵심으로 하는 글로벌 모델이다.

### 1.1 시장 현황

글로벌 AgriTech 시장은 개도국 소농의 디지털 전환과 지속가능한 공급망에 대한 수요 증가로 급격히 성장하고 있다.

| 지표 | 수치 | 출처 |
|------|------|------|
| 글로벌 AgriTech 시장 규모 | $22.5B (2024) | MarketsandMarkets, 2024 |
| 2030년 전망 | $48.0B (CAGR 13.4%) | MarketsandMarkets, 2024 |
| 글로벌 농산물 교역 규모 | $1.9T (2024) | WTO, 2024 |
| 개도국 소농 가구 수 | 5.7억 가구 (전 세계 식량의 70% 생산) | FAO, 2024 |
| 크로스보더 식품 e-커머스 | $51B (2024) → $120B (2030) | Statista, 2024 |
| 글로벌 식품 손실·폐기 | 연간 $1.2T, 전체 생산량의 30-40% | FAO, 2024 |
| 개도국 농산물 수출 중 소농 비중 | 2-5% (시장 접근 극히 제한적) | World Bank, 2024 |

그러나 이 거대한 시장에서 **개도국 소규모 농가의 국제 시장 접근은 극히 제한적**이다. 국제 농산물 무역은 대형 중개상·수출입업체·유통업체가 지배하며, 소농이 직접 해외 바이어와 거래하는 것은 사실상 불가능에 가깝다.

### 1.2 문제점 분석

**개도국 농가 측 (공급):**
- 글로벌 소농 5.7억 가구 중 80%가 국제 시장에 직접 접근 불가 (World Bank, 2024)
- 국제 수출 과정에서 4-7단계 중간 유통 → 농가 수취가가 최종가의 10-25%에 불과
- 국제 식품안전 인증(GlobalG.A.P., HACCP, FDA) 획득 비용·절차 부담 → 소농 단독 불가
- 환율 변동·결제 지연(평균 60-120일) → 현금 흐름 불안정
- 수출 규격·포장·콜드체인 요구사항 충족 어려움 → 수출 기회 상실

**선진국 바이어 측 (수요):**
- 레스토랑·소매업체 68%가 "원산지 추적 가능한 농산물 소싱에 어려움" (Deloitte, 2024)
- 유통 과정 불투명 → 식품 안전·윤리적 소싱 검증 어려움
- ESG 규제 강화(EU CSDDD, 미국 FSVP) → 공급망 실사 의무 부담 증가
- 소량·다품목 개도국 특산물(아보카도, 카카오, 커피, 향신료) 직소싱 채널 부재
- 품질 편차·공급 불안정 → 안정적 조달 리스크

**글로벌 구조적 문제:**
- 크로스보더 농산물 유통에서 30-40%가 중간 단계에서 손실·폐기 (FAO, 2024)
- 개도국 소농 빈곤 심화 → SDG 1(빈곤 퇴치), SDG 2(기아 종식) 달성 저해
- 기후변화로 인한 공급망 리스크 증가 → 다변화된 소싱 필요성 확대
- 국제 무역 서류 80%가 여전히 수작업 → 디지털화 시급

### 1.3 사회적 비용 분석

글로벌 농산물 유통의 비효율로 인한 사회적 비용은 천문학적 규모에 달한다.

| 사회적 비용 항목 | 연간 규모 | 산출 근거 | 팜다이렉트 절감 효과 |
|-----------------|----------|----------|-------------------|
| **식품 손실·폐기** | $1.2T | 전 세계 생산량 30-40% 폐기 (FAO) | 수요예측+콜드체인으로 25% 절감 → $300B |
| **소농 소득 손실** | $450B | 중간유통 마진 60-75% 착취, 소농 5.7억 가구 | 직거래로 수취가 2-4배 → $150B 소농 환원 |
| **탄소 배출 (유통)** | $180B | 불필요 유통 단계의 운송·냉장·폐기 탄소발자국 | 유통단계 50% 축소 → 탄소 40% 절감 |
| **식품 안전 사고** | $110B | 오염·부패·위변조로 인한 건강비용+리콜비용 | 블록체인 이력추적+IoT 모니터링 |
| **빈곤 관련 사회비용** | $500B+ | 소농 빈곤 → 교육·의료·이주·치안 사회적 비용 | 소농 소득 향상 → 빈곤 탈출 가속 |
| **무역 비효율 비용** | $80B | 수작업 서류, 중복 검사, 결제 지연 | 디지털화+자동 컴플라이언스 |
| **합계** | **$2.52T+** | | **$500B+ 절감 잠재력** |

> 이 수치는 단순한 경제 지표가 아니다. $2.52T의 비효율은 곧 수억 명의 소농 가족이 빈곤에 갇히는 이유이며, 13억 톤의 식량이 필요한 사람에게 도달하지 못하는 이유이다.

### 1.4 성공 사례 분석

#### Ninjacart (인도, 2015~)
- **기업가치**: $1B+ (유니콘)
- **누적 투자**: $350M+ (Walmart, Tiger Global 등)
- **핵심**: 인도 농가-소매상 B2B 농산물 공급망 플랫폼
- **성과**: 일일 1,400톤 처리, 농가 110,000+, 소매상 180,000+, 8개 도시
- **기술**: AI 수요예측, 자체 콜드체인 네트워크, 동적 가격 알고리즘
- **시사점**: 중간 유통 제거로 농가 수취가 20% 향상, 식품 폐기 5% 미만 달성. 단, 인도 국내 시장에 한정

#### Twiga Foods (케냐, 2014~)
- **기업가치**: $600M+
- **누적 투자**: $250M+ (Goldman Sachs, IFC 등)
- **핵심**: 아프리카 소농-소매상 B2B 농산물 유통 플랫폼
- **성과**: 케냐·우간다 농가 70,000+, 소매상 140,000+ 연결, 일일 500톤 처리
- **기술**: 모바일 머니(M-Pesa) 결제, 수요예측 ML, 자체 물류
- **시사점**: 아프리카 소농 디지털화의 선구자. 농가 소득 30% 향상 검증. 그러나 아프리카 내수 중심

#### Tridge (한국, 2015~)
- **누적 투자**: $100M+
- **핵심**: 글로벌 농산물·식품 원자재 B2B 인텔리전스 + 소싱 플랫폼
- **성과**: 190개국 데이터, 50,000+ 공급자, 30,000+ 바이어 연결
- **기술**: AI 시장 가격 예측, 위성·기상 데이터 기반 수확량 예측
- **시사점**: 글로벌 농산물 데이터 인프라 구축 성공. 그러나 대형 벌크 트레이딩 중심, 소농 직거래에 최적화되지 않음

#### ProducePay (미국, 2014~)
- **누적 투자**: $380M+
- **핵심**: 멕시코-미국 간 농산물 크로스보더 무역 + 농가 선결제 금융
- **성과**: 연간 $4B+ 농산물 거래 중개, 중남미 농가 1,000+
- **기술**: 시장 가격 실시간 분석, 농가 신용평가 AI, 선결제 플랫폼
- **시사점**: 크로스보더 농산물 무역 + 금융을 결합한 모델 검증. 그러나 멕시코-미국 단일 축 중심

#### Cropin (인도, 2010~)
- **누적 투자**: $120M+
- **핵심**: 농업 데이터 SaaS — 위성 영상 기반 작물 모니터링·수확 예측
- **성과**: 56개국, 700만+ 농가 데이터, 580+ 작물 종류 분석
- **기술**: 위성+기상+IoT 데이터 통합 AI, 작물 건강도 분석
- **시사점**: 농업 데이터 인프라의 글로벌 확장 가능성 입증. 그러나 데이터 분석 SaaS로 직거래 매칭 기능 없음

### 2.1 핵심 기능

#### 1) AI 수요예측·가격최적화 엔진
- 글로벌 농산물 시세 데이터 + 기상·위성 데이터 + 바이어 구매 이력 통합 분석
- 국가별·품목별 수요 예측 → 농가에게 "향후 30일 최적 출하 시기·가격" 추천
- 동적 가격 알고리즘: 수확량·물류비·환율·시장 시세를 실시간 반영하여 공정 가격 산출
- 바이어에게 "최적 소싱 시기·원산지·가격대" 추천 → 구매 의사결정 지원

#### 2) 콜드체인 IoT 실시간 모니터링
- GPS + 온도·습도·진동 센서 부착 → 농장→항구→선박/항공→수입항→바이어 전 과정 추적
- 이상 온도 감지 시 자동 알림 + 보험 청구 자동화
- 냉장 컨테이너 공유 매칭: 소농 소량 화물을 그룹핑하여 풀 컨테이너 구성 → 물류비 50% 절감
- 도착 예정 시간(ETA) 실시간 업데이트 → 바이어 재고 관리 최적화

#### 3) 블록체인 기반 생산이력 추적
- 씨앗·비료 투입 → 재배·농약사용 → 수확·선별 → 수출검역 → 운송 → 수입통관 → 최종 판매 전 과정 기록
- QR코드 스캔 → 최종 소비자가 원산지·농가 정보·유통 경로·탄소발자국 확인
- EU CSDDD, 미국 FSVP 등 국제 공급망 실사 규제 자동 대응
- NFT 기반 인증서 발급: 유기농·공정무역·탄소중립 인증의 디지털화

#### 4) 국제 식품안전 자동 컴플라이언스
- 수입국별 식품안전 규격(GlobalG.A.P., HACCP, FDA, EU 유기농 인증) 요구사항 자동 체크리스트 생성
- 농가 현장 데이터(사진·문서) 업로드 → AI가 규격 충족 여부 자동 판정
- 부족 항목 개선 가이드 다국어 제공 (영어·불어·스페인어·스와힐리어·베트남어)
- 인증 기관 연계 → 온라인 인증 프로세스 간소화 (기존 6-12개월 → 2-3개월)

#### 5) 다국어 AI 커뮤니케이션 & 매칭
- 농가(현지어) ↔ 바이어(한국어·일본어·영어) 간 LLM 기반 실시간 자동 번역
- 바이어 수요(품목·물량·품질·납기·예산) → 최적 농가 5개소 자동 매칭
- 샘플 요청·가격 협상·계약 체결·인보이스 발행 프로세스 자동화
- 농가 신뢰도 점수: 과거 납품 이행률·품질 평점·인증 보유 현황 기반 AI 산출

#### 6) 크로스보더 결제·정산
- 모바일머니(M-Pesa, GCash, Dana) + 은행 송금 + 암호화폐 스테이블코인(USDC) 지원
- 농가 즉시정산: 바이어 결제 확인 후 48시간 내 농가 계좌 입금 (기존 60-120일 대비)
- 환율 리스크 헤징: AI 기반 최적 환전 시점 추천 + 환율 보증 옵션
- 에스크로 결제: 품질 확인 후 대금 지급으로 바이어·농가 양측 보호

- **배경**: 줄리안은 3헥타르 아보카도 농장을 운영한다. 연간 40톤을 수확하지만, 지역 중간상에게 kg당 $0.8에 판매 → 유럽 최종 소비자가 $6/kg의 13%만 수취. 4명의 자녀 교육비를 감당하기 어려운 상황. GlobalG.A.P. 인증이 있으면 수출 가능하지만, 인증 비용 $3,000과 절차를 감당할 수 없음.
- **일일 수입**: 약 $8.8 (연 $32,000 / 365일, 가족 6명 부양)
- **꿈**: "큰아들을 나이로비 대학에 보내고 싶다. 등록금이 연 $2,000인데, 지금 형편으로는 불가능하다."
- **인지 단계**: 케냐 농업협동조합 모임에서 KOICA-팜다이렉트 파일럿 프로그램 소개 → "수출 가격의 50%를 직접 받을 수 있다?"
- **관심 단계**: 스와힐리어 앱 설치 (저사양 안드로이드 지원) → 농장 정보·작물·수확 시기 입력 → AI가 "한국·일본 시장 아보카도 수요 3월-6월 피크, 현재 국제 시세 $2.5/kg" 정보 제공
- **고려 단계**: GlobalG.A.P. 인증 AI 자동 가이드 시작 → 기존 6개월 → 2개월로 단축 예상. 인증 비용 팜다이렉트 부담(후불제). 다른 나이바샤 농가 5곳과 컨테이너 공유 매칭 → 물류비 50% 절감
- **전환 단계**: 첫 수출 (일본 바이어 3톤 계약, $2.0/kg) → M-Pesa로 48시간 내 입금 → 기존 대비 소득 2.5배. "이제 큰아들을 대학에 보낼 수 있다"
- **핵심 구매 동인**: 소득 2.5배 향상이라는 **기능적 경제** + 자녀 교육이라는 **감정적 희망** + 착취 없는 공정 거래의 **사회적 정의**

#### 페르소나 B: 사토 유이치 (Sato Yuichi, 48세, 일본 도쿄, 스페셜티 커피 로스터리 대표)

- **배경**: 유이치는 도쿄 시부야에서 프리미엄 커피 로스터리를 운영한다. 에티오피아 예가체프 생두를 쓰지만, 3단계 중간 유통(수출업체→무역상→수입업체)을 거쳐 kg당 ¥4,500에 구매. 원산지 농가 정보도 알 수 없어, 고객에게 "이 커피가 어디서 왔는지" 스토리를 제공하지 못함. EU CSDDD와 유사한 일본 공급망 실사 규제도 걱정.
- **월 원재료비**: ¥900,000 (약 830만원), 연 ¥10,800,000
- **고민**: "고객들이 원산지 스토리를 원하는데, 중간 유통을 거치면 알 수가 없다."
- **인지 단계**: 일본 커피 박람회에서 팜다이렉트 부스 방문 → "에티오피아 농가 직접 소싱 + 블록체인 이력추적"에 관심
- **관심 단계**: 일본어 플랫폼에서 에티오피아 예가체프 생두 직거래 가격 확인 → ¥2,800/kg (기존 대비 38% 저렴). 농가 줄레카(가명)의 프로필, 농장 사진, 재배 방법, 인증 정보 확인
- **고려 단계**: 5kg 샘플 주문 → 콜드체인 IoT로 운송 온도 실시간 모니터링 → 품질 확인 → QR코드에 "에티오피아 시다마 지역 줄레카 농가, 해발 1,900m, 워시드 프로세스" 정보 → 고객 반응 "이런 투명한 커피 좋다!"
- **전환 단계**: 바이어 프리미엄 구독(월 $299) → 월 200kg 정기 주문 → 블록체인 인증서로 공급망 실사 규제 자동 대응 → 연간 원재료비 35% 절감

#### 페르소나 C: 응웬 티 란 (Nguyen Thi Lan, 34세, 베트남 다크락, 커피·후추 복합 농가)

- **디지털 리터러시**: 스마트폰(저가 안드로이드) 사용, Facebook과 Zalo(베트남 메신저) 활용 가능
- **꿈**: "아이들이 농사 대신 좋은 직업을 가졌으면 하는데, 사실 농사도 제대로 대우받으면 나쁘지 않다."
- **인지 단계**: 다크락 농업협동조합의 KOICA 연계 디지털 교육 프로그램에서 팜다이렉트 소개

- **고려 단계**: AI 수요예측이 "3월에 한국 바이어 수요 피크, 후추는 일본 수요 강세" 안내 → 커피 5톤 + 후추 1톤 사전 계약 매칭 → 컨테이너 공유(인근 3농가 합류)로 물류비 분담

## 6. 사회적 문제 공감대 형성

### 6.1 실제 사례 / 스토리텔링

#### 사례 1: "1달러로 하루를 사는 커피 농부" — 글로벌 식품 가치사슬의 불공정
에티오피아 시다마 지역의 아베베(가명, 55세)는 40년간 커피를 재배해왔다. 그가 키운 커피는 도쿄의 카페에서 한 잔에 ¥600(약 5,500원)에 팔린다. 하지만 아베베가 받는 금액은 원두 1kg당 $1.2 — 최종 소비자가의 약 2%에 불과하다. 나머지 98%는 중간상, 수출업체, 수입업체, 로스터, 카페가 가져간다. 아베베는 하루 수입 약 $3로 가족 6명을 먹여 살린다. "평생 커피를 키웠지만, 내 커피를 마시는 사람의 나라 이름도 모른다." 팜다이렉트가 아베베를 도쿄의 사토 유이치와 직접 연결한다면, 아베베의 수취가는 $4.5/kg으로 3.7배 향상되고, 유이치는 38% 저렴하게 더 신선한 원두를 받는다. 모두가 이기는 구조다.

#### 사례 2: "바다에 버려지는 과일" — 식품 손실의 비극
베트남 메콩 델타의 망고 농가 응웬 반 하이(가명, 38세)는 매년 5월 최대 수확기에 30톤의 망고를 수확한다. 하지만 현지 시장이 포화되면 가격이 폭락해 kg당 $0.3까지 떨어진다. 결국 수확량의 40%인 12톤을 폐기한다. "아이들이 학교에서 돌아올 때 트럭에 실린 망고가 쓰레기장으로 가는 걸 보면 마음이 아프다." 만약 AI 수요예측이 "3월에 한국·일본 바이어에게 사전 계약을 체결하세요, 예상 가격 $1.2/kg"이라고 알려주고, 콜드체인 컨테이너 공유로 소량 수출이 가능했다면, 12톤의 망고는 폐기 대신 외화 수입이 되었을 것이다.

#### 사례 3: "QR코드 뒤의 진실" — 소비자가 원하는 투명성
서울 이태원의 레스토랑 셰프 김다은(가명, 35세)은 "농장에서 식탁까지(Farm-to-Table)" 콘셉트로 운영한다. 하지만 수입 아보카도의 실제 원산지를 확인할 방법이 없다. 수입업체가 "멕시코산"이라고 했지만, 실제로는 페루에서 온 것일 수도 있다. "손님에게 이 아보카도가 어디서 왔는지 자신있게 말하고 싶다." 블록체인 이력추적이 있다면, QR코드 스캔 한 번으로 "케냐 나이바샤, 줄리안 농가, 2026년 2월 15일 수확, 해발 1,800m, 유기농 인증, 콜드체인 4°C 유지 확인"이라는 완전한 이력을 제공할 수 있다.

### 6.2 통계의 인간적 해석

- **"글로벌 소농 5.7억 가구 중 80%가 국제 시장 접근 불가"** → 전 세계 식량의 70%를 생산하는 소농이 국제 시장에서 배제되어 있다. 이들이 직접 수출할 수 있다면, 글로벌 빈곤 인구 7억 명 중 상당수가 빈곤에서 벗어날 수 있다.
- **"글로벌 식품 손실 연간 $1.2T"** → 매년 전 세계에서 생산되는 식량의 30-40%가 버려진다. 이는 연간 13억 톤에 해당하며, 이것만으로 전 세계 기아 인구 8.2억 명을 2번 먹일 수 있는 양이다. 수요예측 AI와 콜드체인 모니터링으로 이 손실의 상당 부분을 줄일 수 있다.
- **"중간 유통 마진 60-75%"** → 소비자가 $10에 사는 커피 1봉지 중 $6-7.5가 중간 단계에서 소비된다. 농부에게 가는 것은 $1-2.5뿐이다. 이 구조는 수백 년간 바뀌지 않았지만, 기술로 바꿀 수 있다.
- **"개도국 농산물 수출 중 소농 비중 2-5%"** → 국제 농산물 무역의 95% 이상을 대형 플랜테이션과 다국적 기업이 지배한다. 소농은 규모의 경제, 인증 비용, 물류 인프라 부재로 참여조차 불가능하다.

## 9. 컴퓨터공학과 학생의 기술적 강점 (Why CS Students)

### 왜 컴퓨터공학과 대학생이 이 사업을 해야 하는가?

팜다이렉트는 **블록체인, IoT, 컴퓨터비전, 시계열 예측, 멀티리전 인프라**라는 컴퓨터공학의 최첨단 기술이 총동원되는 프로젝트이다. FAO(2024)에 따르면 글로벌 식품 손실이 연간 $1.2T에 달하며 [2], 이를 기술로 해결하는 것은 SDG(지속가능발전목표) 달성에도 기여한다.

| 핵심 기술 영역 | 컴공과 학생의 역량 | 학습 과목 연관성 |
|---------------|-------------------|----------------|
| **Hyperledger Fabric** | 프라이빗 블록체인, 스마트 컨트랙트 | 분산시스템, 블록체인 |
| **IoT (MQTT/LoRaWAN)** | 센서 데이터 수집·처리 | 임베디드시스템, IoT |
| **YOLOv8 품질 분석** | 객체 감지, 이미지 분류 | 컴퓨터비전, 딥러닝 |
| **시계열 예측** | Prophet, LSTM, 수요예측 | 기계학습, 시계열분석 |
| **멀티리전 인프라** | AWS 4개 리전, CDN, 장애 복구 | 클라우드컴퓨팅, 분산시스템 |
| **다국적 결제** | Stripe + M-Pesa + USDC 통합 | 소프트웨어공학, 보안 |

**팀 구성 (컴퓨터공학과 4인 학생팀 기준):**

| 역할 | 담당 | 필요 역량 |
|------|------|----------|
| 팀장/백엔드 | API, 다국적 결제, 물류 연동 | NestJS, Stripe, Flexport API |
| 블록체인/IoT | Hyperledger, MQTT, 콜드체인 모니터링 | Go/Node.js, MQTT, LoRaWAN |
| AI/CV 엔지니어 | YOLOv8, 수요예측, 위성 분석 | Python, PyTorch, Prophet |
| 프론트엔드/인프라 | PWA, 모바일, 멀티리전 배포 | React Native, AWS, Docker |

**기술적 실현 가능성:**
- Hyperledger Fabric은 Linux Foundation 오픈소스로 무료 사용 가능하며, 농산물 이력추적 PoC는 1개월 내 구현 가능
- YOLOv8은 Ultralytics 오픈소스로 농산물 품질 분류 모델을 소규모 데이터(1,000장)로도 학습 가능
- World Bank(2024)에 따르면 개도국 소농 5.7억 가구 중 80%가 국제 시장 접근 불가로 [3], 플랫폼의 사회적 임팩트가 명확
- LoRaWAN은 개도국 저전력 광역 통신에 적합하며, $10 미만의 센서로 콜드체인 모니터링이 가능

## 10. 감성 마무리 — "이것은 남의 일이 아닙니다"

### 한 잔의 커피가 만드는 변화

당신이 오늘 아침 마신 커피 한 잔. 그 안에는 보이지 않는 이야기가 있습니다.

에티오피아 시다마 지역 해발 1,900미터. 아베베(55세)는 새벽 5시에 일어나 커피 체리를 땁니다. 40년간 매일 같은 일을 했습니다. 그의 손은 커피 열매로 물들어 있고, 허리는 이미 굽었습니다. 하루 수입 3달러. 그가 평생 만져보지 못한 것은 자신이 키운 커피가 담긴 컵입니다.

베트남 다크락의 란(34세)은 두 아이를 학교에 보내고 커피 농장으로 향합니다. 올해 가격이 폭락했습니다. 시어머니의 수술비를 내려면 내년 수확까지 버텨야 합니다. "내 커피가 한국에서 얼마에 팔리는지, 오늘 처음 알았다."

케냐 나이바샤의 줄리안(42세)은 아보카도 40톤을 수확합니다. 유럽에서는 개당 2유로에 팔리지만, 그에게 돌아오는 것은 13%뿐입니다. 큰아들의 대학 등록금 2,000달러가 없어 진학을 포기시켜야 할지도 모릅니다.

**이것은 남의 일이 아닙니다.**

우리가 매일 먹는 커피, 아보카도, 바나나, 후추, 카카오. 이 모든 것의 시작에는 소농이 있습니다. 전 세계 식량의 70%를 생산하지만, 국제 시장에서 배제된 5.7억 가구. 그들이 만든 가치의 75%를 중간 유통이 가져갑니다. 이 구조는 수백 년간 바뀌지 않았습니다.

**하지만 기술이 있습니다.**

AI가 아베베에게 "지금이 최적의 출하 시기"라고 알려줄 수 있습니다. 블록체인이 란의 커피가 서울의 카페에 도착하기까지의 여정을 모두 기록할 수 있습니다. IoT 센서가 줄리안의 아보카도가 4도씨를 유지하며 도쿄에 도착하는 것을 보장할 수 있습니다.

그리고 무엇보다, 중간 유통 7단계를 1단계로 줄여 **농부에게 공정한 가격을 돌려줄 수 있습니다.**
