### 창업 아이템 개요

**마음이음**은 대학생을 위한 **로컬 LLM 기반 AI 또래 멘탈케어 플랫폼**이다. 본 서비스는 대학생 정신건강 위기라는 사회적 문제에 대응하여, 전문 상담에 접근하기 어려운 대학생들에게 24시간 AI 감정분석 챗봇, 훈련된 또래 상담자 매칭, 실시간 감정 대시보드를 통합 제공한다.

본 플랫폼의 핵심 기술적 차별점은 **로컬 LLM(Local Large Language Model)** 아키텍처에 있다. OpenAI GPT, Claude 등 클라우드 기반 API를 사용하지 않고, Llama 3.1 8B 또는 Mistral 7B 모델을 자체 서버에서 직접 운영한다. 정신건강 데이터는 개인의 가장 민감한 정보에 해당하며, 이러한 데이터가 외부 API 서버로 전송되는 것은 개인정보보호법 위반 소지가 있을 뿐 아니라, 사용자 신뢰 확보에 치명적 장벽이 된다. 로컬 LLM을 채택함으로써 모든 대화 데이터가 자체 보안 서버 내에서만 처리되어 완전한 데이터 주권을 보장하고, 동시에 API 호출 비용(토큰당 과금)을 제거하여 지속가능한 비용 구조를 확보한다.

## 1. 문제 인식 (Problem) — 창업 아이템의 필요성

### 1.1 대한민국 청년 정신건강 위기의 심각성

대한민국은 현재 전례 없는 청년 정신건강 위기에 직면해 있다. 건강보험심사평가원의 통계에 따르면, **20대 우울증 진료 인원은 2019년 약 9만 8천 명에서 2024년 약 22만 3천 명으로 5년간 127% 증가**하였다. 이는 전 연령대 중 가장 가파른 증가율로, 같은 기간 전체 연령 평균 증가율(68%)을 크게 상회한다. 불안장애 역시 20대에서 동일 기간 98% 증가하여, 청년층의 정신건강 악화가 단순한 일시적 현상이 아닌 구조적 문제임을 보여준다.

이러한 위기의 배경에는 복합적인 사회구조적 요인이 존재한다. 첫째, 극심한 취업 경쟁과 불확실한 미래에 대한 불안이 대학생의 만성적 스트레스를 유발한다. 2025년 대학 졸업자 취업률은 67.4%에 머무르고 있으며, 체감 실업률은 이보다 훨씬 높다. 둘째, 코로나19 팬데믹 이후 사회적 고립과 관계 단절의 후유증이 지속되고 있다. 대학생활의 핵심인 대면 교류, 동아리 활동, 캠퍼스 생활이 2020-2022년간 크게 위축된 여파로, 현재 대학생들의 사회적 기술과 정서적 회복력이 이전 세대에 비해 현저히 저하되었다. 셋째, SNS와 디지털 환경에서의 끊임없는 사회적 비교가 자존감 저하와 우울감을 증폭시키고 있다.

특히 주목해야 할 점은 **정신건강 문제의 '조기 발견-조기 개입' 실패**이다. 대학생 우울증의 약 60%는 경미한 수준에서 시작되어 적절한 개입 없이 중등도 이상으로 악화된다. 초기 단계에서 적절한 지지와 모니터링이 이루어졌다면 전문 치료 없이도 회복이 가능한 사례가 상당수이나, 현재 시스템에서는 이러한 조기 개입이 구조적으로 불가능한 상태이다.

### 1.2 대학 상담 인프라의 구조적 한계

한국대학상담학회의 2024년 실태조사에 따르면, 전국 대학 상담센터의 **상담 대기 기간은 평균 4-6주**에 달한다. 일부 대규모 대학에서는 학기 중 대기 기간이 8주를 초과하는 경우도 보고되고 있다. 이는 대학 상담센터의 만성적인 인력 부족에 기인한다. 한국 대학의 평균 전문상담인력 대 학생 비율은 약 1:3,500으로, 미국대학상담학회(IACS)의 권장 기준인 1:1,000-1,500에 크게 미달한다.

더욱 심각한 문제는 **접근 자체의 장벽**이다. 한국청소년상담복지개발원의 조사에 의하면, **정신건강 문제를 경험하는 20대 청년의 약 70%가 전문 상담을 받지 못하고 있으며**, 그 주된 이유는 다음과 같다:

- **비용 부담 (38%)**: 민간 심리상담 1회 비용은 평균 8만-15만 원으로, 경제적 여유가 없는 대학생에게 지속적인 상담은 사실상 불가능
- **사회적 낙인 (27%)**: "정신과에 가면 기록이 남는다", "약한 사람으로 보일까 봐" 등의 인식이 여전히 강하게 존재하며, 특히 취업을 앞둔 대학생들의 우려가 큼
- **시간적 제약 (19%)**: 상담센터 운영시간(주중 9-18시)이 수업/아르바이트 일정과 충돌
- **정보 부족 (11%)**: 어디서, 어떻게 도움을 받을 수 있는지 모름
- **기타 (5%)**: 상담에 대한 부정적 선경험, 언어/문화적 장벽 등

이러한 구조적 한계는 대학 상담센터의 예산과 인력을 단순히 증가시키는 것만으로는 해결이 어렵다. 24시간 접근 가능하고, 익명성이 보장되며, 비용 부담이 없는 새로운 형태의 정신건강 지원 시스템이 필요한 이유이다.

### 1.3 디지털 멘탈케어 시장의 성장성

글로벌 디지털 멘탈헬스 시장은 폭발적인 성장세를 보이고 있다. Market Research Future(2024)의 보고서에 따르면, **글로벌 멘탈헬스 시장 규모는 2025년 약 4,350억 달러에서 2035년 약 1조 3,200억 달러로 성장이 전망**되며, 이 중 디지털 부문의 성장이 핵심 동력이다.

| 시장 구분 | 2024-2025년 | 2035년 (전망) | CAGR |
|-----------|-------------|---------------|------|
| 한국 멘탈헬스 전체 시장 | ~2,600억 원 (2025) | ~8,460억 원 | 15.51% |
| 한국 디지털 멘탈헬스 시장 | ~7,280억 원 (2024) | ~4.7조 원 | 18.54% |
| 글로벌 멘탈헬스 시장 | ~4,350억 달러 (2025) | ~1.3조 달러 | 11.6% |
| 글로벌 AI 멘탈헬스 시장 | ~21억 달러 (2025) | ~170억 달러 | 23.7% |

**한국 디지털 멘탈헬스 시장은 2024년 약 7,280억 원에서 2035년 약 4.7조 원으로, 연평균 18.54%의 높은 성장률이 전망**된다. 특히 코로나19 이후 비대면 상담에 대한 수요가 급증하면서, 2020-2024년간 비대면 심리상담 이용건수는 연평균 42% 증가하였다. 정부 역시 2023년부터 '마음건강 종합대책'을 통해 디지털 멘탈헬스 산업 육성을 국정과제로 추진하고 있어, 정책적 지원 환경도 조성되고 있다.

### 1.4 기존 서비스의 한계와 Gap Analysis

현재 국내외 멘탈케어 시장에는 다양한 서비스가 존재하나, **한국 대학생의 특수한 니즈를 충족시키는 서비스는 부재**하다.

| 기존 서비스 | 주요 한계점 |
|-------------|-------------|
| **Calm / Headspace** (글로벌) | 명상/마음챙김 중심으로 심층적 감정 케어 부재, 영어 기반, 한국 문화 맥락 부재, 높은 구독료 (월 $14.99), 클라우드 API 의존 |
| **BetterHelp** (글로벌) | 전문상담사 매칭 중심으로 비용이 월 $65-100, 한국 서비스 미제공, 2023년 FTC 개인정보 유출 제재 사례 |
| **트로스트** (국내) | 전문상담사 중심으로 비용 부담 (회당 4-8만원), AI 기능 제한적, 대학생 특화 콘텐츠 부족 |
| **마인드카페** (국내) | 커뮤니티 중심으로 전문적 감정 분석 부재, 익명성 보장에 한계, 또래 매칭 알고리즘 없음 |
| **대학 상담센터** | 4-6주 대기, 주중 주간만 운영, 낙인 우려, 졸업 후 지원 중단 |

이러한 기존 서비스들의 **공통적 한계점**은 다음과 같다:

1. **클라우드 API 의존으로 인한 개인정보 위험**: 대부분의 AI 기반 서비스가 OpenAI, Google 등의 클라우드 API를 사용하여, 사용자의 민감한 정신건강 대화 데이터가 외부 서버로 전송됨. BetterHelp의 경우 2023년 미국 FTC로부터 사용자 건강 데이터 무단 공유로 780만 달러 제재를 받은 바 있음
2. **한국 대학생 맥락의 부재**: 학점, 취업, 등록금, 병역, 학과 관계 등 한국 대학생 고유의 스트레스 요인을 이해하고 반영하는 서비스 부재
3. **또래 지지체계의 부재**: 전문상담사-내담자의 수직적 관계만 존재하고, 유사한 경험을 공유하는 또래 간 수평적 지지체계가 부재
4. **높은 비용 장벽**: 지속적 이용을 위한 비용이 대학생 경제 수준에 부적합
5. **사후적 대응 중심**: 이미 심각한 상태의 사용자만 대상으로 하며, 예방적/조기 개입 기능이 부족

**마음이음**은 이러한 Gap을 정확히 해결한다. 로컬 LLM으로 완전한 데이터 주권을 보장하고, 한국 대학생 맥락에 파인튜닝된 AI 감정분석으로 24시간 조기 개입을 가능하게 하며, 훈련된 또래 상담자 매칭으로 접근성과 비용 문제를 동시에 해결한다. 이는 기존 시장의 어떤 서비스도 제공하지 못하는 고유한 가치 제안(Value Proposition)이다.

## 2. 실현 가능성 (Solution) — 창업 아이템의 개발 계획

### 2.1 핵심 기술 아키텍처: 로컬 LLM 기반 AI 멘탈케어 엔진

#### 2.1.1 왜 로컬 LLM인가 — 클라우드 API 대신 자체 모델을 운영하는 이유

마음이음의 가장 핵심적인 기술적 의사결정은 **클라우드 기반 LLM API(OpenAI GPT, Anthropic Claude, Google Gemini 등)를 사용하지 않고, 오픈소스 LLM을 자체 서버에서 직접 운영(로컬 LLM)하는 것**이다. 이 결정에는 다음과 같은 결정적 이유가 있다.

**첫째, 정신건강 데이터의 극도의 민감성과 데이터 주권 확보이다.**

정신건강 관련 대화 데이터는 개인정보 중에서도 가장 민감한 범주에 속한다. 사용자가 AI 챗봇에게 자살 충동, 가정 문제, 성적 고민, 대인관계 갈등 등을 털어놓을 때, 이 데이터가 해외 클라우드 서버(미국 소재 OpenAI 서버 등)로 전송된다면 다음과 같은 위험이 발생한다:

- **개인정보보호법 제17조(개인정보의 제공) 및 제39조의12(국외 이전 특례)** 위반 소지: 민감 정보의 국외 이전 시 별도의 동의 및 보호조치 필요
- **정신건강복지법 제71조(비밀 누설의 금지)** 저촉 가능성: 정신건강 관련 정보의 부적절한 취급
- **EU GDPR Article 9 (특수 범주 개인정보 처리)**에서 건강 데이터를 가장 높은 수준의 보호 대상으로 규정하고 있으며, 글로벌 서비스 확장 시 준수 필수
- BetterHelp 사례(2023년 FTC 제재)에서 보듯, 클라우드 기반 멘탈헬스 서비스의 데이터 유출/무단 공유 위험은 현실적 위협

로컬 LLM을 채택하면 **모든 사용자 대화 데이터가 자체 보안 서버(국내 IDC 또는 자체 서버룸) 내에서만 처리되고, 외부로 일체 전송되지 않는다.** 이는 "데이터가 서버를 떠나지 않습니다"라는 가장 강력한 프라이버시 보장을 사용자에게 제공하며, 개인정보보호법 및 GDPR 완벽 준수를 가능하게 한다.

**둘째, API 비용 종속에서의 탈피와 지속가능한 비용 구조이다.**

클라우드 LLM API는 토큰(단어) 단위로 과금된다. GPT-4o 기준 입력 100만 토큰당 $2.5, 출력 100만 토큰당 $10이다. 멘탈케어 챗봇의 특성상 대화가 길고 깊어지는 경향이 있어, 사용자 1인당 월 평균 대화량을 약 50회, 회당 평균 2,000 토큰(입출력 합산)으로 가정하면:

| 항목 | 클라우드 API (GPT-4o) | 로컬 LLM (Llama 3.1 8B) |
|------|----------------------|-------------------------|
| 사용자 1인 월 토큰 소비 | 100,000 토큰 | 100,000 토큰 |
| 사용자 1인 월 API 비용 | ~$0.75 (~1,000원) | $0 (고정비만) |
| MAU 10만 명 기준 월 비용 | ~$75,000 (~1억 원) | 서버 운영비 ~500만 원 |
| MAU 50만 명 기준 월 비용 | ~$375,000 (~5억 원) | 서버 운영비 ~2,000만 원 |
| 연간 비용 (MAU 50만) | ~60억 원 | ~2.4억 원 |

사용자 규모가 커질수록 **로컬 LLM의 비용 이점은 기하급수적으로 확대**된다. 스타트업의 초기 자본이 제한된 상황에서, API 비용에 종속되면 사용자 증가가 곧 비용 폭증으로 이어져 지속가능한 성장이 불가능하다. 로컬 LLM은 초기 GPU 서버 투자비(RTX 4090 기준 약 300-500만 원)만으로 사실상 무제한 추론이 가능하다.

**셋째, API 의존성 제거와 서비스 자주권 확보이다.**

OpenAI 등 외부 API에 의존하면, 해당 업체의 가격 정책 변경, 서비스 약관 변경, API 서비스 중단, 지역 차단 등에 의해 서비스 전체가 위험에 처할 수 있다. 실제로 OpenAI는 2023-2024년간 수차례 가격 정책을 변경하였으며, 특정 사용 사례에 대한 이용 약관을 제한한 바 있다. 로컬 LLM은 이러한 외부 의존성을 완전히 제거하여, 서비스의 자주권과 안정성을 보장한다.

**넷째, 한국어 정신건강 데이터에 대한 자유로운 파인튜닝이 가능하다.**

로컬 LLM은 한국어 상담 대화 데이터, 한국 대학생 정서 표현 패턴, PHQ-9/GAD-7 한국어 버전 응답 데이터 등으로 자유롭게 파인튜닝할 수 있다. 클라우드 API는 파인튜닝 옵션이 제한적이거나 추가 비용이 발생하며, 파인튜닝 데이터 자체가 외부 서버에 업로드되어야 하는 추가적인 프라이버시 문제가 있다.

#### 2.1.2 AI 감정분석 챗봇 — "이음이"

마음이음의 AI 챗봇 "이음이"는 다음과 같은 기술 구조로 동작한다:

**기반 모델**: Llama 3.1 8B Instruct (4-bit GPTQ 양자화) 또는 Mistral 7B Instruct v0.3
- 8B 파라미터 모델은 RTX 4090 24GB VRAM에서 4-bit 양자화 시 약 5-6GB VRAM만 사용하여 동시 다중 사용자 처리 가능
- 응답 생성 속도: 약 40-60 tokens/sec (RTX 4090 기준), 사용자 체감 지연 1-2초 이내

**파인튜닝 데이터셋**:
- 한국어 상담 대화 코퍼스 (한국상담심리학회 제공 비식별화 데이터, 약 5만 회 대화)
- 한국 대학생 정서 표현 사전 (약 3,000개 표현 패턴)
- PHQ-9, GAD-7 한국어 표준화 버전 문항 및 응답 데이터
- 한국 대학생 스트레스 요인 분류 체계 (학업, 취업, 대인관계, 경제, 가족, 진로 등 12개 범주)

**파인튜닝 방법**: QLoRA (Quantized Low-Rank Adaptation)
- 전체 모델 재학습 없이 소규모 어댑터만 학습하여, RTX 4090 1대로 약 8-12시간 내 파인튜닝 완료
- 학습 데이터 추가 시 증분 학습(incremental learning) 가능

**안전 장치**:
- 자살/자해 위험 감지 시 즉각 전문 상담 연결 트리거 (위기 개입 프로토콜)
- 모델 환각(hallucination) 방지를 위한 가드레일(guardrail) 시스템 적용
- 의학적 진단이나 처방을 절대 하지 않도록 시스템 프롬프트에 명시
- 모든 대화 세션에 "이음이는 AI 챗봇이며, 전문 상담을 대체하지 않습니다" 고지

**핵심 기능**:
1. **감정 인식 및 분류**: 사용자 발화에서 감정을 실시간 분석 (기쁨, 슬픔, 분노, 불안, 외로움, 무기력 등 18개 감정 범주)
2. **공감적 대화**: 로저스의 인간중심 상담 이론에 기반한 적극적 경청, 반영, 공감 반응 생성
3. **인지행동치료(CBT) 기반 개입**: 부정적 자동사고 인식, 인지 재구조화 기법 적용
4. **감정 추적 및 패턴 분석**: 장기간 대화 데이터에서 감정 변화 패턴을 분석하여 위험 징후 조기 감지
5. **맞춤형 심리교육 콘텐츠 추천**: 사용자의 현재 감정 상태와 관심사에 따른 콘텐츠 큐레이션

#### 2.1.3 또래 상담자 매칭 시스템

AI 챗봇만으로는 인간적 연결의 욕구를 완전히 충족시킬 수 없다. 마음이음은 **훈련된 또래 상담자(Peer Counselor)**를 AI와 결합하여 시너지를 창출한다.

- **또래 상담자 선발**: 심리학/상담학 전공 대학(원)생 중 40시간 교육 이수자
- **매칭 알고리즘**: 사용자의 고민 유형, 감정 상태, 선호 성별, 학교, 전공 등을 기반으로 최적의 또래 상담자를 추천
- **AI 보조 시스템**: 또래 상담 진행 중 AI가 실시간으로 상담자에게 대화 팁, 감정 분석 결과, 주의사항 등을 제공 (상담자 전용 화면)
- **슈퍼비전 체계**: 전문 상담심리사가 또래 상담 내용을 (비식별화하여) 슈퍼비전하는 구조

#### 2.1.4 감정 대시보드

사용자에게 자신의 감정 상태를 시각적으로 보여주는 개인 대시보드를 제공한다:

- **일간/주간/월간 감정 추이 그래프**: AI 대화 및 자가 체크인 데이터 기반
- **PHQ-9 / GAD-7 정기 자가진단**: 격주 자동 알림으로 우울/불안 수준 추적
- **감정 일기**: AI가 대화 내용을 바탕으로 자동 요약하여 감정 일기 초안 생성
- **위험 알림**: 감정 추이에서 급격한 악화가 감지되면 사용자에게 전문 상담 권유 알림
- **성장 기록**: 긍정적 변화, 대처 전략 활용 횟수, 자기 돌봄 습관 형성 추이 등

### 2.3 임상 도구 통합: PHQ-9 및 GAD-7

마음이음은 임상적으로 검증된 자가진단 도구를 플랫폼에 통합하여, AI 분석의 신뢰성과 과학적 근거를 확보한다.

**PHQ-9 (Patient Health Questionnaire-9)**:
- 우울증 선별을 위한 9문항 자가보고식 척도
- 한국어 표준화 연구(안제용 외, 2013)에서 민감도 81.1%, 특이도 89.9% 확인
- 절단점 10점 기준으로 주요우울장애 선별
- 마음이음에서 격주 자동 제시, 결과를 감정 대시보드에 시계열로 표시

**GAD-7 (Generalized Anxiety Disorder-7)**:
- 범불안장애 선별을 위한 7문항 자가보고식 척도
- 한국어 표준화 연구(서신영 외, 2015)에서 내적 일관성 Cronbach's α = 0.915
- 절단점 8점 기준으로 범불안장애 선별
- PHQ-9와 병행 실시하여 우울-불안 공병 패턴 분석

**AI 통합 방식**:
- PHQ-9/GAD-7 점수와 AI 대화 감정분석 결과를 교차 분석하여 정확도 향상
- 자가진단 점수가 임계값을 초과하면 자동으로 위기 개입 프로토콜 발동
- 전문 상담 연계 시 비식별화된 추이 데이터를 상담사에게 사전 제공 (사용자 동의 하에)

## 3. 성장전략 (Scale-up) — 사업화 추진 전략

### 3.1 시장 진입 전략: 대학 캠퍼스 집중 공략

마음이음의 시장 진입은 **"좁고 깊게"** 시작한다. 초기 타겟을 수도권 5개 대학으로 한정하여, 해당 대학 내에서 압도적인 인지도와 사용률을 확보한 후 전국으로 확대하는 전략이다.

**시범 운영 대학 선정 기준**:
1. 학생 수 1만 명 이상의 종합대학
2. 상담센터와의 협력 의향이 확인된 대학
3. 대학혁신지원사업, 대학생활문화원 등을 통한 예산 확보가 가능한 대학
4. 수도권 소재 (초기 대면 지원 및 또래 상담자 확보 용이)

**시범 운영 계획**:
- **1차 (M6-M9)**: 2개 대학 200명 대상 클로즈드 베타 → 사용성 검증, 모델 성능 개선
- **2차 (M9-M12)**: 5개 대학 2,000명 대상 오픈 베타 → 사업 모델 검증, 수익화 테스트
- **3차 (M12-M18)**: 15개 대학 확대 → 스케일업 및 투자 유치
- **4차 (M18-M24)**: 전국 50개 대학 + 기업 EAP 진출

**캠퍼스 마케팅 전술**:
- 신학기 OT, 총학생회 협업, 동아리 박람회 부스 운영
- "마음 건강 주간" 이벤트 대학과 공동 기획
- 또래 상담자 모집을 통한 바이럴 효과 (상담학 전공생의 실습 기회 제공)
- 캠퍼스 내 상담센터와 공식 연계 (AI 사전 스크리닝 → 대기 없이 상담센터 우선 배정)

### 3.7 ESG 및 사회적 가치

마음이음은 디지털 멘탈헬스 스타트업으로서 ESG(환경, 사회, 지배구조) 가치를 핵심 경영 원칙으로 수립한다.

#### 환경 (Environment)
- **로컬 LLM의 환경적 이점**: 클라우드 API 대비 에너지 효율적인 추론 (필요할 때만 GPU 가동, 유휴 시 전력 절감). 대형 클라우드 데이터센터의 냉각 에너지 소비 없이 소규모 온프레미스 서버로 운영
- **페이퍼리스 상담 기록**: 기존 종이 기반 상담 기록지의 디지털 전환으로 자원 절약
- **탄소 배출 추적 및 상쇄**: 서버 운영으로 인한 탄소 배출을 측정하고, 성장 단계에서 탄소 상쇄 프로그램 참여

#### 사회 (Social)
- **정신건강 불평등 해소**: 경제적 이유로 상담을 받지 못하는 대학생에게 무료/저비용 서비스 제공
- **또래 상담자 역량 강화**: 상담학 전공 대학생에게 실질적 상담 경험과 수입 기회 제공 (또래 상담자 활동비 지급)
- **정신건강 인식 개선**: 캠퍼스 내 정신건강 캠페인, 낙인 해소 활동 지속적 추진
- **취약계층 지원**: 국가장학금 수급자, 차상위 계층 학생에게 프리미엄 기능 무료 제공
- **정신건강 데이터의 공익적 활용**: 비식별화된 대학생 정신건강 트렌드 데이터를 학술 연구 및 정책 수립에 기여

#### 지배구조 (Governance)
- **윤리 위원회 설치**: 임상심리 전문가, 법률 전문가, 사용자 대표로 구성된 윤리 위원회가 AI 모델의 안전성과 윤리적 운영을 감독
- **투명한 AI 운영**: AI 챗봇의 한계를 사용자에게 명확히 고지, 정기적인 모델 편향성 감사 실시
- **데이터 거버넌스**: 개인정보보호책임자(CPO) 선임, 데이터 처리 방침 투명 공개, 사용자의 데이터 삭제 요청권 완전 보장
- **이해관계자 소통**: 분기별 사용자 자문회의, 연 1회 투명성 보고서 발간
