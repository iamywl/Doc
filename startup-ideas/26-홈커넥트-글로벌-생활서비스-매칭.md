### 창업 아이템 개요

**홈커넥트**는 생활서비스(청소, 수리, 인테리어, 이사, 배관, 전기, 에어컨, 해충방제 등)가 필요한 가정·사무실과 검증된 전문가를 AI로 매칭하는 **글로벌 홈서비스 플랫폼**이다. 당근마켓이 동네 물건을 연결하듯, 홈커넥트는 집에서 발생하는 모든 문제와 그 문제를 해결할 수 있는 최적의 전문가를 연결한다. 1인 가구 3.5억+ 시대, 외주 생활서비스 수요는 폭발적으로 증가하고 있지만, 기존 시장은 **가격 불투명(42%), 서비스 품질 편차(38%), 안전 우려(35%)**라는 삼중 문제를 안고 있다. 본 플랫폼은 LLM 기반 자연어 수요 파싱으로 고객의 요청("욕실 타일이 들떠서 물이 새요")을 정확한 서비스 카테고리(타일 시공 + 방수 처리)로 변환하고, AI 기반 전문가 검증(신원인증 + 자격증 + 작업 품질 AI 분석)으로 신뢰를 보장하며, 실시간 AI 견적과 Before/After 사진 품질 검증으로 가격·품질 투명성을 구현한다.

### 1.2 글로벌 생활서비스 시장의 구조적 문제

전 세계적으로 1인 가구 증가, 맞벌이 가정 확산, 도시화 가속이라는 메가트렌드에 의해 **전문 생활서비스 외주** 수요가 폭발적으로 증가하고 있다. 그러나 생활서비스 시장은 여전히 오프라인 중심의 비체계적 구조를 유지하며, 디지털 전환이 가장 더딘 산업 중 하나이다.

핵심 통계:

- **글로벌 1인 가구**: 3.5억+ (UN DESA, 2024), 2030년 4.8억 전망
- **한국 1인 가구**: 981만 가구 (전체의 35.5%, 통계청 2024), 2030년 1,100만 전망
- **생활서비스 외주 의향**: 1인 가구 82%, 맞벌이 가정 76%, 시니어 가구 68% (한국소비자원, 2024)
- **불만족 경험률**: 35% — 가격 불투명 42%, 실력 편차 38%, 안전 우려 35%, 시간 미준수 28%
- **생활서비스 가격 비교 불가**: 소비자의 67%가 "적정 가격을 모른다"고 응답
- **전문가 신원 미확인**: 서비스 이용자의 58%가 "전문가의 신원·경력을 확인한 적 없다"
- **분쟁 경험**: 서비스 이용자의 22%가 "비용·품질 관련 분쟁을 경험", 이 중 48%가 해결 불가

한국 생활서비스 시장의 구조를 보면:

- **영세 자영업자 중심**: 청소·수리·인테리어 사업자의 87%가 5인 미만 영세 사업체 (소상공인시장진흥공단, 2024)
- **무자격 시공**: 인테리어·수리 분야 무자격 시공 비율 35%+ (국토교통부, 2024)
- **보험 미가입**: 생활서비스 사업자의 65%가 배상책임보험 미가입
- **현금 결제 비중**: 여전히 45%가 현금·계좌이체 결제 (세금계산서 미발행)
- **온라인 전환율**: 8% 미만 — 소비자 니즈 대비 디지털 인프라 크게 부족

### 1.3 사회적 비용 분석

생활서비스 시장의 구조적 문제는 단순한 개인 불편을 넘어, 사회 전체에 막대한 비용을 발생시킨다.

| 사회적 비용 항목 | 연간 추정 규모 | 산출 근거 | 영향 대상 |
|------------------|----------------|-----------|-----------|
| **가격 부풀리기 피해** | 4,200억원 | 소비자원 불만 18,500건 × 평균 피해 230만원 | 독거노인, 1인 여성, 정보취약계층 |
| **무자격 시공 재시공 비용** | 2,700억원 | 인테리어 피해 8,200건 × 평균 320만원 | 신혼부부, 자가소유자 |
| **안전사고 의료·배상 비용** | 1,100억원 | 작업 중 사고 연 3,200건 × 평균 340만원 | 전문가·고객 모두 |
| **분쟁 해결 사회적 비용** | 680억원 | 법적 분쟁 4,100건 × 소송·조정 평균 165만원 | 사법 시스템, 소비자·전문가 |
| **소비자 탐색·비교 시간 비용** | 3,500억원 | 서비스 이용자 800만명 × 연 4.4시간 × 시급 1만원 | 맞벌이, 1인 가구 |
| **전문가 유휴·영업 비용** | 5,800억원 | 영세사업자 35만명 × 유휴시간 30% × 연수입 | 영세 자영업자 |
| **세금 누수 (현금결제)** | 1,800억원 | 현금결제 45% × 시장규모 35조 × 부가세율 | 국가 재정 |
| **합계** | **약 1조 9,780억원** | | 사회 전체 |

> 홈커넥트가 시장의 50%를 디지털화할 경우, 연간 약 **9,900억원**의 사회적 비용 절감이 가능하다.

### 1.4 사회적 문제의 심각성과 공감대 형성

#### 페르소나 1: 가격 불투명의 피해자 — 독거 어르신 이순복(72세)

```
┌─────────────────────────────────────────────────────────────┐
│  페르소나 1: 이순복 (72세, 서울 마포구, 독거)                │
├─────────────────────────────────────────────────────────────┤
│  상황: 화장실 배관 막힘                                      │
│  ├─ 인터넷 검색 → "24시 배관 전문" 전화                     │
│  ├─ 전화 상담: "출장비 3만원"                                │
│  ├─ 현장 도착: "배관 노후, 전체 교체 필요" → 85만원 청구     │
│  ├─ 비교 수단 없음 → 그대로 결제                             │
│  └─ 적정 가격: 25~35만원 (2.5배 과다 청구)                   │
├─────────────────────────────────────────────────────────────┤
│  핵심 문제: 정보 비대칭 + 정보취약계층 타겟팅                 │
│  영향 범위: 독거노인 190만 가구                               │
│  한국소비자원: 가격 부풀리기 접수 연 18,500건+ (1위 34%)      │
│  고령자·1인 여성 피해율: 평균 대비 2.3배                      │
└─────────────────────────────────────────────────────────────┘
```

서울 마포구에 사는 이순복(72세) 할머니는 화장실 배관이 막혀 인터넷에서 찾은 "24시 배관 전문"에 전화했다. 전화로 "출장비 3만원"이라 했지만, 현장에서 "배관 노후로 전체 교체 필요"라며 **85만원을 청구**했다. 혼자 사는 할머니는 비교할 방법이 없어 그대로 결제했다. 나중에 이웃에게 물어보니 같은 작업 적정 가격은 **25-35만원**이었다. **이것이 한국 190만 독거노인이 생활서비스에서 겪는 현실이다.** 한국소비자원(2024)에 따르면, 생활서비스 관련 소비자 불만 접수 건수는 연 18,500건+이며, 이 중 **"가격 부풀리기"가 34%로 1위**를 차지한다. 특히 고령자·1인 여성 가구의 피해 비율이 평균보다 2.3배 높다.

#### 페르소나 2: 품질 편차의 악순환 — 맞벌이 부부 정대호(39세)

```
┌─────────────────────────────────────────────────────────────┐
│  페르소나 2: 정대호 (39세, 경기 판교, 맞벌이 부부)           │
├─────────────────────────────────────────────────────────────┤
│  상황: 30평 아파트 리모델링 (예산 3,000만원)                  │
│  ├─ 숨고 견적 3개: 1,200만~1,900만원 (편차 700만원)          │
│  ├─ 적정가 판단 불가 → 최저가 선택                            │
│  ├─ 시공 6개월 후: 벽지 박리 + 타일 들뜸 + 배관 누수         │
│  ├─ A/S 요청 → 업체 연락처 변경, 연락 두절                   │
│  └─ 재시공 추가 비용: 450만원                                 │
├─────────────────────────────────────────────────────────────┤
│  핵심 문제: 품질 검증 불가 + 사후 보호 부재                   │
│  "먹튀" 피해: 연 8,200건+ / 평균 피해 320만원                 │
└─────────────────────────────────────────────────────────────┘
```

경기 판교의 맞벌이 부부 정대호(39세)는 이사를 앞두고 리모델링 업체를 찾았다. "숨고"에서 견적을 3개 받았지만, 가격 차이가 700만원(1,200만원~1,900만원)이나 났고, 어떤 견적이 적정한지 판단할 기준이 없었다. 가장 저렴한 업체를 선택했더니, 시공 후 6개월 만에 벽지 박리, 타일 들뜸, 배관 누수가 발생했다. A/S를 요청했지만 업체가 **전화번호를 변경하고 연락이 두절**되었다. 다시 시공하는 데 추가 450만원이 들었다. **인테리어·수리 분야의 "먹튀" 피해는 연 8,200건+**(한국소비자원, 2024)이며, 평균 피해 금액은 320만원이다.

```
│  └─ 작업 후에도 "그 사람이 내 주소를 안다"는 지속적 불안       │

│  핵심 문제: 신원 미검증 + 안전 보장 부재                      │
│  여성 1인 가구: 384만 (64%가 안전 불안 경험)                  │
│  한국여성정책연구원 (2024)                                    │
└─────────────────────────────────────────────────────────────┘
```

서울 강남구 원룸에 사는 직장인 김지연(28세)은 에어컨 수리를 위해 기사를 불렀다. 집 안에 모르는 남성을 들이는 것이 불안했지만, 에어컨 없이 한여름을 보낼 수 없었다. 기사가 방문했는데 **신분증·자격증 확인이 불가능**했고, 작업 중 개인적인 질문을 하여 불쾌했다. 작업 후 "추가 부품비"로 15만원을 더 청구했다. **한국 1인 가구 981만 중 여성 1인 가구는 384만**(통계청), 이들의 64%가 "생활서비스 이용 시 안전에 대한 불안"을 경험한다 (한국여성정책연구원, 2024).

이러한 사례들이 보여주는 공통 구조는 **정보 비대칭**이다. 소비자는 (1) 적정 가격을 모르고, (2) 전문가의 실력을 판단할 수 없으며, (3) 분쟁 시 보호받을 수 없다. 반대로 양심적인 전문가는 (1) 고객을 찾기 어렵고, (2) 가격 경쟁에 밀리며, (3) 실력을 증명할 체계가 없다. **양측 모두 손해를 보는 레몬 시장(Market for Lemons)**이다.

### 1.5 기존 솔루션의 한계

**1) 한국 플랫폼**
- **숨고**: 누적 230억원+ 투자, 50만+ 전문가 등록. 견적 입찰 방식 → 가격 비교는 가능하나 품질 판단 불가. AI 매칭 없음 (전문가가 입찰), 품질 검증 시스템 부재, 보증보험 없음
- **크몽 (생활서비스)**: 프리랜서 마켓, 생활서비스 전문이 아님. 대면 서비스 관리 체계 미흡
- **미소**: 가사 청소 특화, 100억원+ 투자. 청소 외 카테고리 제한적, 수리·인테리어 미지원

**2) 해외 플랫폼**
- **TaskRabbit** (미국): IKEA에 인수 (2017), 가구 조립·이사·소일거리 매칭. 미국 한정, 전문 시공(인테리어·배관·전기) 미지원
- **Thumbtack** (미국): 기업가치 $3.3B (2021), 1,100+ 카테고리, 300,000+ 전문가. 견적 입찰 방식, AI 매칭 기본 수준, 품질 AI 검증 없음
- **Urban Company** (인도): 기업가치 $2.8B, 70개 도시, 50,000+ 전문가, 월 400만+ 서비스. 자체 교육·인증 모델이 강력하나, 전문가 자유도 낮음(고정 가격). 인도·UAE 한정
- **Angi/HomeAdvisor** (미국): NASDAQ $2.5B+, 미국 최대 홈서비스 플랫폼. 리드 판매 모델(전문가에게 고객 정보 판매) → 전문가 불만 높음, 소비자 만족도 3.2/5.0
- **Care.com** (미국): IAC $500M 인수, 돌봄 서비스 특화. 생활서비스(청소·수리·인테리어) 미지원

**3) 공통 한계점**
- LLM 기반 자연어 수요 파싱 부재 (소비자가 정확한 카테고리를 선택해야 함)
- AI 기반 Before/After 품질 검증 부재 (리뷰에만 의존)
- AI 표준가(적정 가격) 제시 시스템 부재 (가격 불투명 문제 미해결)
- 보증보험·에스크로 통합 부재 (분쟁 시 소비자·전문가 보호 미흡)
- 글로벌 확장 미미 (각 플랫폼이 특정 국가에 한정)

#### TAM (Total Addressable Market) — 전체 시장

**글로벌 온라인 홈서비스 시장: $156B (2030년)**

글로벌 홈서비스 시장($600B)에서 온라인 플랫폼 경유 비중이 2024년 9.7%에서 2030년 13%로 성장하며, 이 온라인 전환 시장이 TAM이다. 특히 아시아-태평양 지역은 스마트폰 보급률과 1인 가구 증가에 힘입어 연 22.1%의 최고 성장률을 보이고 있다.

#### SAM (Serviceable Available Market) — 유효 시장

**한국 + 동남아·일본 온라인 홈서비스: $32B (2030년)**

홈커넥트가 1차적으로 공략하는 시장: (1) 한국 온라인 생활서비스(8조원), (2) 일본 온라인 홈서비스($12B), (3) 동남아 주요국(베트남·태국·인도네시아) 온라인 홈서비스($10B). 이 세 시장은 (1) 1인 가구 급증, (2) 스마트폰 보급 90%+, (3) 기존 서비스 디지털 전환 초기 단계라는 공통 특성을 가진다.

#### SOM (Serviceable Obtainable Market) — 획득 가능 시장

**한국 온라인 생활서비스 시장의 10-15% 점유 = 3년차 서비스 600만건, 매출 1,000억원**

한국 온라인 생활서비스 시장(8조원)의 10-15%를 3년 내 점유하는 것이 목표. 이는 월 50만건의 서비스(청소 25만건 + 수리 10만건 + 인테리어 5만건 + 이사 3만건 + 기타 7만건)에 해당하며, 전문가 100,000명, 고객 1,000,000명 규모이다.

### 1.8 사용자 구매동인(Purchase Motivation) 분석

생활서비스는 **비자발적 수요(broken pipe, 고장)와 자발적 수요(인테리어, 청소 대행)**가 혼재하는 시장이다. 비자발적 수요는 긴급성이, 자발적 수요는 편의성과 품질이 핵심 동인이다.

#### 기능적 동인 (Functional Motivation)

| 동인 | 구체적 내용 | 지불 의향 영향 |
|------|-----------|--------------|
| **긴급 해결** | 배관 파열, 전기 합선, 보일러 고장 → 2시간 내 전문가 출동 | 긴급 할증 20-30% 수용 |
| **적정 가격 확인** | AI 표준가로 "이 작업의 적정 가격은 15-20만원입니다" → 바가지 방지 | 가격 신뢰 시 즉시 결제 전환율 85% |
| **품질 보장** | AI 품질 검증 + 보증보험 → 시공 불량 시 무료 재시공 보장 | 프리미엄 10-15% 추가 지불 의향 |
| **시간 절약** | 전문가 탐색·비교·예약 → 앱에서 3분 내 완료 | 연간 30시간+ 시간 절약 |

#### 감정적 동인 (Emotional Motivation)

| 동인 | 구체적 내용 | 심리적 가치 |
|------|-----------|-----------|
| **안심** | 신원·범죄이력 100% 검증 전문가만 방문 → 1인 가구·여성·시니어 안전 보장 | "모르는 사람을 집에 들이는" 불안 해소 |
| **투명성** | 작업 전 견적, 작업 중 실시간 소통, Before/After 사진 → "숨겨진 비용" 불안 제거 | "내가 통제하고 있다"는 주도감 |
| **깨끗한 집** | 전문가 청소 후의 쾌적함 → 정신적 웰빙·자존감 향상 | "나를 위한 투자"라는 만족 |
| **완성감** | 인테리어·리모델링 완성 → 꿈꾸던 공간 실현 | SNS 공유, 사회적 인정 욕구 충족 |

#### 사회적 동인 (Social Motivation)

| 동인 | 구체적 내용 | 사회적 가치 |
|------|-----------|-----------|
| **전문가 정당 대우** | 직접 매칭으로 중개 수수료 최소화 → 전문가 수취율 80-85% | "기술자를 존중하는 소비" |
| **지역 상생** | 동네 전문가 우선 매칭 → 지역 경제 활성화 | "우리 동네 사장님을 돕는" 선택 |
| **환경** | 수리·리퍼로 폐기 감소, 친환경 자재·세제 옵션 | ESG 가치 소비 니즈 |

#### 페르소나별 구매 여정

**페르소나 A: 1인 여성 직장인 김지연(28세) — 정기 가사 청소**
1. **인식**: 주말마다 청소에 3시간 소비, 워라밸 개선 필요
2. **탐색**: 미소·숨고 비교 → 미소는 청소만, 숨고는 품질 편차 크다는 리뷰
3. **전환**: 홈커넥트에서 "원룸 정기 청소, 격주, 토요일 오전" 입력 → AI가 "20평 원룸 청소 표준가 7-9만원" 제시, 4.8점 이상 청소 전문가 3명 추천. 신원·범죄이력 검증 완료 배지 확인
4. **결제**: 격주 정기 구독(월 15만원), 에스크로 결제, 첫 회 불만족 시 100% 환불
5. **유지**: Before/After 사진 AI 품질 체크 → 매회 만족도 4.8 → 6개월 유지, 추가로 에어컨 청소·이사 등 크로스셀

**페르소나 B: 신혼부부 정대호(39세) — 인테리어 리모델링**
1. **인식**: 첫 집 구매 후 30평 아파트 리모델링 결정, 예산 3,000만원
2. **탐색**: 오늘의집·숨고 견적 비교 → 가격 차이 700만원, 적정가 판단 불가, 시공 품질 사전 확인 불가
3. **전환**: 홈커넥트에서 "30평 아파트, 거실+주방+욕실 리모델링, 예산 3,000만원" 입력 → AI가 공간별 작업 세부 항목 자동 파싱, 표준가 범위 2,500-3,500만원 제시, 포트폴리오·리뷰 기반 업체 5곳 추천
4. **결제**: 마일스톤 기반 에스크로 (철거 20% → 배관·전기 30% → 타일·도배 30% → 마감 20%), 보증보험 가입
5. **유지**: 시공 중 실시간 진행 사진·동영상, AI Before/After 품질 체크, 완공 후 1년 A/S 보증 → 지인 추천 바이럴

**페르소나 C: 빌딩 관리사무소 박관리(55세) — B2B 정기 관리**
1. **인식**: 15층 오피스빌딩 관리, 청소·소방·배관·전기·승강기 등 월 10-15건의 서비스 필요
2. **탐색**: 기존 업체 계약 → 긴급 대응 느림(평균 48시간), 비용 협상력 부족
3. **전환**: 홈커넥트 B2B에서 "15층 오피스빌딩, 월 정기 청소 + 설비 유지보수 + 긴급 출동" 등록 → AI가 서비스 유형별 최적 전문가 매칭, 월 정기 계약 패키지 제안
4. **결제**: 월 280만원 정기 계약, 긴급 출동 2시간 이내 보장, SLA 기반 계약
5. **유지**: 월간 시설 관리 리포트, 예방 정비 알림, 비용 최적화 분석 → 연간 계약 갱신

### 2.2 핵심 기능

#### 1) LLM 기반 자연어 수요 파싱 & AI 매칭

- 고객이 자연어로 문제를 설명하면 LLM이 **서비스 카테고리(50개+), 세부 작업 항목, 긴급도(즉시/예약), 예상 규모(면적/수량), 예상 비용 범위**를 자동 파싱
- 예시: "욕실 타일이 들떠서 물이 새요" → {카테고리: 타일 시공 + 방수 처리, 규모: 욕실 1개, 긴급도: 높음, 표준가: 30-50만원}
- 예시: "32평 아파트 전체 도배 하고 싶어요" → {카테고리: 도배, 규모: 32평, 긴급도: 보통, 표준가: 150-250만원}
- 매칭 알고리즘: **전문 분야 + 지역(반경 5km) + 가용 시간 + 평점 + 유사 작업 경력 + 가격대** 기반 최적 전문가 3-5명 추천
- **수요 예측 AI**: 과거 데이터 + 날씨 + 시즌을 학습하여 지역별 서비스 수요 예측 → 전문가에게 사전 알림 (예: "다음 주 한파 예보, 보일러 수리 수요 3배 증가 예상")
- 매칭 정확도 목표: 만족도 4.5+/5.0 (기존 숨고 3.8/5.0 대비)

#### 2) AI 신원·자격 인증 시스템

- **신분증 OCR + 얼굴 인증 (FaceNet)**: 등록 시 신분증 촬영 + 셀카 대조 → 본인 확인
- **범죄이력 조회**: 경찰청 범죄경력회보서 API 연동 (성범죄·폭력 전과 확인)
- **자격증 자동 검증**: 건축기능사, 전기기능사, 배관기능사, 위생관리사 등 국가기술자격 → 한국산업인력공단 API 자동 대조
- **배상책임보험 확인**: 보험 가입 여부·한도 자동 확인, 미가입 시 플랫폼 제휴 보험 가입 유도
- **포트폴리오 인증**: 과거 시공 사진·영상 업로드, AI가 실제 시공 여부 검증 (메타데이터 분석)
- 검증 완료 전문가에게 "인증 배지" 부여 → 고객 신뢰 + 매칭 우선순위 상승

#### 3) AI 표준가 & 실시간 견적 시스템

- **AI 표준가(Fair Price)**: 과거 거래 데이터(지역, 카테고리, 규모, 시기)를 학습하여 작업별 적정 가격 범위 자동 산출
- "이 작업의 적정 가격은 **15-20만원**입니다" → 고객에게 사전 제시 → 바가지 방지
- **실시간 견적 비교**: 매칭된 3-5명 전문가의 견적을 표준가 대비 시각화 ("이 전문가 견적은 표준가의 95%, 적정 범위 내")
- **투명 가격 구조**: 인건비, 자재비, 출장비를 항목별로 분리 표시
- **시즌·긴급 가격 조정**: 한파·폭염 등 수요 급증 시 동적 가격 안내 (할증이 아닌 "수요-공급 기반 적정가" 표시)

#### 4) ViT 기반 Before/After 품질 검증

- **작업 전 사진 촬영 (Before)**: 전문가가 작업 전 현장 사진 의무 촬영 → 앱에 업로드
- **작업 후 사진 촬영 (After)**: 작업 완료 후 같은 앵글에서 사진 촬영
- **ViT(Vision Transformer) AI 품질 분석**: Before/After 사진을 비교하여 작업 완성도 자동 평가
  - 청소: 먼지·얼룩 제거율, 광택도 변화
  - 타일 시공: 줄눈 균일성, 타일 정렬도
  - 도배: 이음매 정밀도, 기포·주름 유무
  - 배관: 연결부 밀봉 상태 (육안 검사용)
- 품질 점수 80점 이상 시 자동 결제 완료, 미달 시 고객 확인 후 재시공 연결
- 품질 점수 이력 → 전문가 프로필에 누적 → 고객 선택의 신뢰 근거

#### 5) 에스크로 결제 & 보증보험

- **에스크로 결제**: 고객 결제 → 플랫폼 보관 → 작업 완료 + 품질 확인 후 전문가에게 정산
- **대형 시공 마일스톤 결제**: 인테리어·리모델링 등 대규모 작업 시 단계별 분할 결제
- **보증보험 자동 가입**: 작업 중 발생하는 재산 피해(파손, 누수 등) + 인체 상해에 대한 보험 (보험료 플랫폼 부담)
- **A/S 보증**: 시공 작업 완료 후 3개월(기본) ~ 1년(인테리어) A/S 보증
- **24시간 환불 보호**: 작업 미완료·품질 미달 시 24시간 내 전액 환불

#### 6) 실시간 예약·추적·소통

- **즉시 예약 / 스케줄 예약**: 긴급 출동(2시간 내) 또는 원하는 날짜·시간대 선택
- **전문가 실시간 위치 추적**: 고객 앱에서 전문가의 현재 위치·도착 예정 시간 확인 (배달앱 UX)
- **앱 내 채팅·전화**: 개인 연락처 노출 없이 앱 내에서 안전하게 소통
- **작업 진행 타임라인**: 출발 → 도착 → 작업 시작 → 중간 사진 → 작업 완료 → 평가 전 과정 기록
- **그룹 서비스**: 아파트·오피스 단체 공동 요청 → 전문가 배치 최적화 → 비용 분담

#### 7) 다국어 & 글로벌 서비스

- **6개 언어 지원**: 한국어, 영어, 일본어, 베트남어, 태국어, 인도네시아어
- **외국인 거주자 모국어 서비스**: 한국 내 체류 외국인 250만명+(법무부)에게 모국어 UI + 전문가 소통 번역
- **글로벌 전문가 인증 상호인정**: 각국 기술 자격증 등급 매핑 테이블 구축

## 5. 컴퓨터공학과 학생의 기술적 강점 (Why CS Students)

### 왜 컴퓨터공학과 대학생이 이 사업을 해야 하는가?

홈커넥트는 **컴퓨터비전(CV), NLP, 위치 기반 서비스(LBS), 결제 시스템**이라는 컴퓨터공학의 다양한 분야가 융합된 프로젝트이다. 특히 ViT 기반 시공 품질 검증은 학술적으로도 의미 있는 CV 응용 사례이며, LLM 수요 파싱은 NLP의 실전 적용이다.

| 핵심 기술 영역 | 컴공과 학생의 역량 | 학습 과목 연관성 |
|---------------|-------------------|----------------|
| **LLM 수요 파싱** | 자연어 → 서비스 카테고리 분류 | 자연어처리, 기계학습 |
| **ViT 품질 검증** | 이미지 분류·변화 감지 | 컴퓨터비전, 딥러닝 |
| **PostGIS 위치 서비스** | 공간 쿼리, 근거리 매칭 | 데이터베이스, 알고리즘 |
| **에스크로 결제** | 트랜잭션, 상태 머신 설계 | 소프트웨어공학, 데이터베이스 |
| **OCR/FaceNet** | 신분증 인식, 얼굴 인증 | 컴퓨터비전, 패턴인식 |
| **수요 예측** | Prophet 시계열, ML 예측 | 데이터마이닝, 통계 |

**팀 구성 (컴퓨터공학과 4인 학생팀 기준):**

| 역할 | 담당 | 필요 역량 |
|------|------|----------|
| 팀장/백엔드 | API, 결제, 에스크로, 보험 연동 | NestJS, PostgreSQL, Stripe |
| AI/CV 엔지니어 | ViT 품질분석, OCR, FaceNet, LLM | PyTorch, HuggingFace, OpenCV |
| 프론트엔드/모바일 | 앱 UI, 지도 통합, GPS 추적 | React Native, Google Maps SDK |
| 데이터/인프라 | PostGIS, 수요예측, 모니터링 | AWS, TimescaleDB, Grafana |

**기술적 실현 가능성:**
- ViT(Vision Transformer)는 Google의 오픈소스 모델로, Before/After 이미지 비교를 위한 Fine-tuning이 학생 GPU(RTX 4090)로 가능
- PostGIS는 PostgreSQL 확장으로 별도 인프라 없이 위치 기반 매칭 구현 가능
- 한국소비자원(2024)에 따르면 생활서비스 불만 접수 연 18,500건+으로, AI 기반 품질 검증의 사회적 필요성이 입증됨
