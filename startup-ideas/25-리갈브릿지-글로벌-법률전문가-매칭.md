## □ 창업 아이템 개요

**리갈브릿지**는 법률 서비스가 필요한 개인·기업과 전 세계 변호사를 AI로 매칭하는 **크로스보더 법률 서비스 플랫폼**이다. 해외 진출, 국제 계약, 이민, 특허 등 크로스보더 법률 수요가 폭발하고 있으나 해외 변호사를 찾고 검증하기 극도로 어렵다.

| 요약 | 내용 |
|------|------|
| **문제** | 중소기업 88% "법률비용이 해외진출 장벽" (ICC). 변호사 시간당 $200-1,000. 적합한 해외 변호사 찾기 4-8주 |
| **솔루션** | LLM 법률이슈 분류 + 전문가 매칭, 법률 번역 AI, E2E 암호화 문서관리, 에스크로 |
| **성장** | 한국 기업→해외 변호사 → 양방향. 수수료 10-15%. 3년 내 연매출 150억원 |

## 1. 문제 인식 (Problem)

- **중소기업 88%**: "법률 비용이 해외 진출 최대 장벽" (ICC, 2024)
- **국제 상사 분쟁**: 연 85,000건+
- **글로벌 특허 출원**: 연 350만건+ (WIPO)
- **한국 해외 진출 기업**: 8만개사+ (KOTRA)
- **국제 법률 자문 비용**: 건당 1,500-5,000만원

## 2. 실현 가능성 (Solution)

### 핵심 기능

1. **AI 법률 이슈 분류 & 매칭**: 자연어 → 분야·관할권·긴급도 분류 → 3-5명 추천 → 15분 무료 상담
2. **법률 번역 AI**: 6개 언어 법률 전문 번역, 화상 상담 실시간 통역
3. **보안 문서 관리**: AES-256 E2E 암호화, Attorney-Client Privilege 보호, 전자서명
4. **화상/서면 자문**: 30-60분 화상 + 서면 의견서 3-7일
5. **비용 투명성**: 시간당 요율 공개, 고정 요금 옵션, AI 비용 견적, 에스크로

### 기술 스택

Next.js + React Native / Go + Node.js / LegalBERT + GPT-4o / AES-256, AWS KMS, SOC 2 / WebRTC + 통역 파이프라인 / Stripe + 에스크로 / DocuSign API

## 3. 성장전략 (Scale-up)

| 수익원 | 비중 |
|--------|------|
| 매칭 수수료 (10-15%) | 45% |
| 기업 구독 (월 49만원~) | 30% |
| AI 문서 자동화 (건당 5-30만원) | 15% |
| 프리미엄 변호사 등록 (월 19만원) | 10% |

- **Phase 1**: 한국 기업 → 미국·일본·동남아 변호사 500명 (특허·법인·계약)
- **Phase 2**: 양방향 + 개인 (이민·부동산·상속) + 유럽·중동
- **Phase 3**: 50개국+ 양방향, 법률 AI SaaS, 연 36,000건, 연매출 300억원

### ESG
- **사회**: 법률 접근성 민주화, 중소기업·개인에게 합리적 글로벌 법률 서비스
- **환경**: 원격 법률 상담으로 해외 출장 탄소배출 절감
- **지배구조**: 법률 비용 투명성, 품질 평가 표준화
