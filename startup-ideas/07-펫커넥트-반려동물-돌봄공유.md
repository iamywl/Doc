## □ 창업 아이템 개요(요약)

### 명칭
- **펫커넥트(PetConnect)** — AI 기반 반려동물 돌봄 공유 및 건강관리 통합 플랫폼

### 범주
- O2O 서비스 / 인공지능 프로그램 / 모바일 앱

### 창업 아이템 개요

본 지원사업을 통해 개발하고자 하는 **「펫커넥트」**는 반려동물 양육가구가 겪는 **돌봄 공백 문제**를 해결하기 위한 **AI 기반 돌봄 매칭·건강관리 통합 플랫폼**이다.

1인가구 및 맞벌이 가구의 증가로 반려동물 돌봄 수요가 폭증하고 있으나, 기존 펫호텔·펫시터 서비스는 **높은 비용(1박 3~5만원)**, **신뢰도 부족**, **실시간 모니터링 불가** 등의 한계를 가진다. 펫커넥트는 **동네 기반 돌봄 공유 매칭**, **실시간 GPS·사진 돌봄 리포트**, **로컬 AI를 활용한 반려동물 건강 사진 분석 및 펫케어 Q&A**, **비대면 수의사 상담 연동**을 핵심 기능으로 제공한다.

특히 핵심 차별점은 **로컬(Local) LLM 기반 AI 엔진**으로, 반려동물 사진 건강 분석·펫케어 Q&A·돌봄자-반려동물 호환성 매칭을 **사용자 기기 및 엣지 서버에서 처리**하여 **개인정보 보호**, **오프라인 환경(산책 중 GPS 트래킹) 대응**, **대규모 서비스 시 비용 절감**을 동시에 달성한다.

가격은 기존 펫호텔 대비 **40~60% 저렴한 수준**(시간당 5,000~15,000원)으로, 돌봄 제공자에게는 부수입 기회를, 양육자에게는 합리적 비용의 신뢰할 수 있는 돌봄을 제공한다.

### 문제 인식 (Problem) 요약
- 국내 반려동물 시장 2025년 약 15조원 규모, 양육가구 602만, 1인가구 양육자 급증
- 출장·여행·긴급상황 시 돌봄 위탁 수단 부족, 펫호텔 비용 부담
- 개인 간 돌봄 매칭의 신뢰도·안전성 문제
- 반려동물 건강관리 정보의 파편화, 비대면 수의사 진료 허용(2024.12)에 따른 새로운 기회

### 실현 가능성 (Solution) 요약
- 로컬 AI 기반 돌봄 매칭 + 건강관리 + 비대면 진료 연동 원스톱 플랫폼
- Flutter 크로스플랫폼 앱 + FastAPI 백엔드 + 로컬 LLM(Llama 3, MobileViT) 온디바이스/엣지 추론
- 정부지원사업비 1단계 2억원 + 2단계 4억원 활용 12개월 MVP 개발·검증

### 성장전략 (Scale-up) 요약

- 서울 → 수도권 → 전국 확장, 3년차 BEP 달성 목표
- ESG 연계: 유기동물 임시보호 매칭, 시니어 돌봄자 일자리 창출

- 동물병원 네트워크·반려동물 행동 전문가 자문단 확보 예정

## 1. 문제 인식 (Problem) — 창업 아이템의 필요성

### 1.1 국내 반려동물 시장 현황 및 성장 추세

대한민국 반려동물 산업은 지난 10년간 폭발적으로 성장해왔다. 한국농촌경제연구원에 따르면 2023년 국내 반려동물 관련 시장 규모는 약 9조 4천억원이었으며, KB경영연구소의 「2024 한국 반려동물 보고서」에 따르면 **2025년 약 15조원**, 2027년에는 **20조원**을 돌파할 것으로 전망된다[1][2].

특히 주목해야 할 수치는 다음과 같다:

| 지표 | 수치 | 출처 |
|------|------|------|
| 반려동물 양육 가구 수 | 약 602만 가구 (전체 가구의 27.4%) | KB경영연구소, 2024 |
| 반려견 등록 수 | 약 310만 두 (2024년 기준) | 농림축산식품부 |
| 반려동물 관련 시장 규모 | 약 15조원 (2025년 추정) | 한국농촌경제연구원 |
| 펫테크 시장 규모 | 약 3,000억원 (2025) → 6,000억원 (2027) | KPMG |
| 월 평균 양육비 | 약 15.4만원 | KB경영연구소, 2024 |
| 1인가구 반려동물 양육 비율 | 약 31.4% (전체 양육가구 대비) | 통계청, 2024 |

반려동물은 더 이상 단순한 '동물'이 아닌 **'반려(伴侶)'**, 즉 가족 구성원으로 인식되고 있으며, 이에 따라 양육의 질적 수준에 대한 기대치가 높아지고 있다. 「펫코노미(Petconomy)」라는 신조어가 등장할 만큼 반려동물 관련 소비는 경기 불황에도 줄지 않는 **불황형 소비**의 대표 사례로 꼽힌다[3].

### 1.2 핵심 문제: 돌봄 공백(Care Gap)

**문제 1: 출장·여행·긴급상황 시 돌봄 위탁의 어려움**

반려동물 양육자의 **67.3%**가 "여행이나 출장 시 반려동물 돌봄이 가장 큰 고민"이라고 응답했다(한국펫산업연합회, 2024). 현재 이용 가능한 대안은 다음과 같다:

| 돌봄 방식 | 평균 비용 (1일 기준) | 문제점 |
|----------|---------------------|--------|
| 펫호텔 | 3~5만원 (소형견), 5~8만원 (중대형견) | 높은 비용, 스트레스, 전염병 우려 |
| 지인 부탁 | 무료~소액 | 전문성 부족, 관계 부담, 지속성 없음 |
| 개인 펫시터 | 2~4만원 | 신뢰도 검증 어려움, 보험 미비 |
| 동물병원 위탁 | 4~7만원 | 의료기관 본연 목적과 불일치, 제한적 |

주 3~4일 출장이 잦은 직장인의 경우, 연간 돌봄 비용만 **200~400만원**에 달하며, 이는 반려동물 입양을 포기하거나 유기하는 원인 중 하나로 지목된다[4].

**문제 2: 1인가구 양육자의 일상적 돌봄 공백**

통계청에 따르면 2025년 1인가구 비율은 전체 가구의 **40.3%**에 달하며, 이 중 반려동물 양육 비율은 **31.4%**로 추정된다. 1인가구 양육자는 출근 시간(평균 8~10시간) 동안 반려동물이 혼자 방치되는 **"일상적 돌봄 공백"**을 경험한다.

- 산책 필요 시간: 중형견 기준 하루 최소 1~2시간
- 실제 1인가구 산책 시간: 평균 30분 미만 (한국반려동물협회, 2024)
- 돌봄 공백으로 인한 문제 행동(분리불안, 과잉 짖음, 파괴 행동) 발생률: **약 38%**[5]

**문제 3: 개인 간 매칭의 신뢰도 문제**

기존 돌봄 매칭 서비스(도그메이트, 펫시터 등)의 사용자 리뷰를 분석하면, 가장 빈번한 불만은 다음과 같다:

1. **돌봄자 신원 검증 부족** (42.1%) — "정말 믿을 수 있는 사람인가?"
2. **실시간 모니터링 불가** (28.7%) — "내 아이가 지금 어떤 상태인지 알 수 없다"
3. **사고 시 책임 소재 불명확** (18.3%) — "사고가 나면 누가 책임지는가?"
4. **돌봄 품질 편차** (10.9%) — "돌봄자마다 수준 차이가 크다"

이러한 신뢰 문제는 **잠재 수요를 억제**하는 핵심 병목으로 작용하고 있다. 실제로 돌봄 서비스 필요성을 느끼지만 이용하지 않는 양육자의 **58.2%**가 "신뢰할 수 없어서"를 이유로 꼽았다[6].

**문제 4: 건강관리 정보의 파편화**

반려동물 건강 관련 정보는 동물병원 진료기록, 예방접종 기록, 양육자 기억 등에 **분산·파편화**되어 있다. 이로 인해:

- 동물병원 변경 시 진료 히스토리 단절
- 예방접종·구충제 투여 일정 관리 어려움
- 질병 초기 증상(피부, 눈, 귀 등 외관 변화) 감지 지연
- 비대면 진료 시 기초 건강 정보 전달 어려움

### 1.3 규제 환경 변화: 비대면 수의사 진료 허용

2024년 12월, 농림축산식품부는 **「수의사법 시행규칙」 개정**을 통해 반려동물에 대한 **비대면(원격) 수의사 진료를 공식 허용**하였다[7]. 이는 펫테크 산업에 있어 **게임체인저**급 규제 완화로 평가된다.

주요 변경 내용:
- 재진(follow-up) 환자에 대한 화상 진료 허용
- 경증 피부질환, 행동 상담 등 비대면 초진 일부 허용
- 원격 처방전 발급 가능 (처방식이 포함)

이에 따라 **돌봄 서비스와 건강관리·비대면 진료를 연결하는 통합 플랫폼**의 시장 기회가 크게 확대되었다.

### 1.4 문제 인식 종합 — 왜 "지금" 펫커넥트인가

```
┌─────────────────────────────────────────────────────────┐
│                    시장 기회 종합                          │
├─────────────────────────────────────────────────────────┤
│  ① 15조원 시장 + 602만 양육가구 → 거대 TAM              │
│  ② 1인가구 40% 시대 → 돌봄 공백 구조적 심화              │
│  ③ 비대면 수의사 진료 허용 → 건강관리 통합 기회           │
│  ④ 펫테크 시장 3,000억→6,000억 급성장 → 기술 수용성 ↑   │
│  ⑤ 기존 서비스 신뢰도 문제 → 차별화된 솔루션 수요         │
└─────────────────────────────────────────────────────────┘
```

**결론:** 반려동물 돌봄 시장은 수요 폭증에도 불구하고 **신뢰·비용·편의성** 세 가지 측면에서 기존 솔루션이 양육자의 기대에 미치지 못하고 있다. 여기에 비대면 진료 허용이라는 규제 변화가 맞물리면서, **돌봄 매칭 + 실시간 모니터링 + AI 건강관리 + 비대면 진료 연동**을 하나의 플랫폼에서 제공하는 **"펫커넥트"**의 시장 진입 타이밍이 최적인 상황이다.

### 1.5 사용자 구매동인(Purchase Motivation) 분석

#### 기능적 동인

- **시간 절약**: 출장·여행 전 펫호텔 예약 → 이동 → 체크인 → 체크아웃의 번거로운 과정을 앱 내 30초 매칭으로 대체. 동네 돌봄자가 집으로 방문하므로 반려동물의 이동 스트레스도 제거.
- **비용 절감**: 펫호텔 1박 3~5만원(소형견) 대비 펫시팅 방문 돌봄 2만~3.5만원으로 약 40~60% 저렴. 프리미엄 구독(월 4,900원) 시 월 1회 수의사 상담 무료 포함.
- **편의성**: 돌봄 매칭 → 실시간 GPS·사진 모니터링 → 건강관리 다이어리 → 비대면 수의사 상담을 하나의 앱에서 원스톱 해결. 기존에는 돌봄 앱 + 건강 기록 앱 + 동물병원 예약 앱을 각각 사용해야 했음.
- **기존 대안 대비 이점**: 도그메이트는 건강관리·비대면 진료 미지원. 핏펫은 돌봄 매칭 미지원. 펫커넥트만이 돌봄+건강+진료를 통합 제공하며, 로컬 AI로 오프라인 환경(산책 중)에서도 GPS 트래킹·건강 분석 가능.

#### 감정적 동인

- **불안 해소**: "내 아이가 지금 어떤 상태일까?"라는 불안을 실시간 GPS 추적과 30분 간격 사진·영상 리포트로 해소. 긴급 상황 시 즉시 알림 + 비대면 수의사 연결로 최악의 시나리오 대비.
- **안심**: 돌봄자의 범죄경력 조회, 12시간 교육 이수, 배상책임보험 자동 적용이라는 3중 안전장치가 "모르는 사람에게 아이를 맡긴다"는 불안을 구조적으로 해소.
- **죄책감 해소**: "출장 때마다 아이를 혼자 두고 간다"는 양육자의 죄책감을 검증된 이웃 돌봄자의 세심한 돌봄으로 완화. 돌봄 리포트를 통해 "아이가 행복하게 시간을 보냈다"는 확인이 가능.
- **자기효능감**: AI 건강 사진 분석과 건강관리 다이어리로 "내가 아이의 건강을 체계적으로 관리하고 있다"는 책임감 있는 양육자로서의 효능감.

#### 사회적 동인

- **사회적 인정**: "펫호텔 대신 동네 이웃에게 돌봄을 맡기고 실시간 모니터링까지 한다"는 것은 반려인 커뮤니티에서 "스마트한 펫페어런팅"으로 인정받는 행위.
- **트렌드 참여**: 펫테크·AI 건강관리는 반려동물 양육 트렌드의 최전선. "AI로 아이의 피부 상태를 분석한다"는 것은 SNS에서 공유할 만한 화제거리.
- **소속감**: 동네 기반 돌봄 공유 커뮤니티를 통해 같은 지역 반려인들과의 유대감 형성. "우리 동네 돌봄자 네트워크"라는 공동체 의식.

#### 페르소나별 구매 여정

**페르소나 1: 한서영 (32세, 1인가구, 프리랜서 디자이너, 포메라니안 '몽이' 양육)**
한서영 씨는 포메라니안 몽이와 함께 마포구 연남동 원룸에서 살고 있다. 프리랜서 특성상 급한 미팅이나 출장이 잦은데, 그때마다 몽이를 펫호텔에 맡기면 1박에 4만원이 들고 몽이는 낯선 환경에서 심하게 떤다. 지인에게 부탁하기도 미안하다. 펫커넥트에서 같은 동네(연남동)의 돌봄자를 검색했다. 중형견 2마리를 키우는 경험 많은 50대 여성 돌봄자가 방문 펫시팅(반일 2만원)을 제공한다. 매칭 후 실시간 GPS로 몽이의 산책 경로를 확인하고, 30분마다 올라오는 사진에서 몽이가 다른 강아지와 놀고 있는 모습을 보며 안심했다. 이후 해당 돌봄자를 "단골"로 지정하여 매달 2~3회 이용하고 있다.

**페르소나 2: 이준혁 (41세, 맞벌이, 골든 리트리버 '바다' 양육, 아내와 공동 양육)**
이준혁 부부는 강남구 아파트에서 골든 리트리버 바다(5살)를 키우고 있다. 부부 모두 출근하는 평일 10시간 동안 바다가 혼자 있어야 한다. 바다는 산책이 하루 1~2시간 필요한 대형견이지만, 퇴근 후 30분 산책이 고작이다. 바다가 분리불안 증세를 보이기 시작했다. 펫커넥트의 프리미엄 구독(월 4,900원)에 가입하여, 평일 점심 시간에 돌봄자의 산책대행(1시간, 15,000원 × 주 3회)을 이용하기 시작했다. GPS로 산책 경로와 거리(평균 3.5km)를 확인하고, 건강관리 다이어리에 바다의 일일 활동량을 기록한다. AI 건강 사진 분석으로 바다의 피부에 미세한 발적이 감지되어 비대면 수의사 상담을 받은 결과, 초기 아토피로 진단되어 조기 치료를 시작할 수 있었다.

### 1.6 사회적 문제 공감대 형성

#### 실제 사례 기반 스토리텔링

**사례 1: 김소현 씨 (가명, 29세, 직장인, 고양이 '루나' 양육)**

김소현 씨는 2년 전 유기묘였던 루나를 입양했다. 하지만 회사에서 3박 4일 해외 출장이 잡혔을 때, 루나를 맡길 곳을 찾지 못해 동료에게 급하게 부탁했다. 동료는 고양이 경험이 없어 루나의 사료량과 화장실 관리를 제대로 하지 못했고, 소현 씨가 돌아왔을 때 루나는 이틀간 제대로 먹지 못한 상태였다. 이후 소현 씨는 출장을 거부하기 시작했고, 결국 승진 기회를 놓쳤다. **반려동물 돌봄 공백은 단순히 동물 복지의 문제가 아니라, 양육자의 커리어와 삶의 질에 직접적인 영향을 미치는 문제이다.** 펫커넥트의 고양이 전문 돌봄자 매칭과 실시간 모니터링이 있었다면, 소현 씨는 안심하고 출장을 다녀오고 승진 기회도 잡을 수 있었을 것이다.

**사례 2: 박정수 씨 (가명, 55세, 시니어, 말티즈 '봄이' 양육)**

박정수 씨는 은퇴 후 말티즈 봄이와 함께 생활하고 있다. 최근 건강 검진에서 봄이의 피부에 작은 종양이 발견되었지만, 가까운 동물병원은 차로 40분 거리이고 예약이 2주 뒤에야 잡힌다. 온라인에서 "강아지 피부 종양"을 검색했지만, 불안을 가중시키는 정보만 가득했다. **반려동물 건강 정보의 파편화와 수의사 접근성 문제는 양육자의 불안을 극대화한다.** 펫커넥트의 AI 건강 사진 분석이 "양성 가능성 높음, 심각도 하, 2주 내 수의사 방문 권고"라고 알려주고, 비대면 수의사 상담으로 초기 소견을 들을 수 있었다면, 2주 동안의 불안을 크게 줄일 수 있었을 것이다.

**사례 3: 돌봄 공백과 유기의 악순환**

농림축산식품부에 따르면 2023년 유기·유실 동물 수는 약 11만 마리로, 이 중 약 20%가 "양육 포기"에 해당한다. 양육 포기 사유 중 "돌봄 여건 부족"이 35%로 가장 높다. 1인가구 증가와 맞벌이 가구의 돌봄 시간 부족이 결합되면서, "키우고 싶지만 돌볼 수 없어서" 버리는 비극이 반복되고 있다. **유기동물 1마리당 보호·관리 비용은 연간 약 180만원이며, 연간 11만 마리 기준 약 2,000억원의 사회적 비용이 발생한다.** 합리적 비용의 돌봄 공유 플랫폼이 양육 지속 가능성을 높이면, 유기율을 의미 있게 낮출 수 있다.

#### "이것은 남의 일이 아닙니다"

반려동물 양육 가구 602만. 이 중 1인가구 양육자 비율 31.4%. 맞벌이 양육가구 포함 시 "돌봄 공백"을 경험하는 가구는 절반을 넘는다. 반려동물은 이제 가족이다. 가족을 두고 출장을 떠날 때의 불안, 아이가 아픈데 병원에 데려가기 어려울 때의 답답함, 매일 10시간 혼자 두고 출근하는 죄책감 -- 이것은 602만 가구의 일상이다.

#### 문제가 해결되지 않을 경우의 사회적 비용

- **유기동물 증가**: 돌봄 여건 부족으로 인한 유기 증가 → 보호소 과밀 수용 → 안락사 비율 증가. 연간 약 2,000억원의 유기동물 관리 비용 지속 증가.
- **양육자 정신건강 악화**: 돌봄 공백에 대한 만성적 불안·죄책감이 양육자의 우울·번아웃으로 이어짐. 반려동물 양육자의 22%가 돌봄 스트레스로 정신건강 서비스를 이용한 경험이 있음.
- **반려동물 건강 악화**: 일상적 건강 모니터링 부재로 질병 조기 발견 실패 → 중증 단계에서 발견 → 치료비 급증(초기 발견 대비 평균 5~10배). 양육자 경제적 부담 가중.
- **사회적 갈등**: 분리불안으로 인한 반려동물의 과잉 짖음·소음이 이웃 갈등으로 확대. 반려동물 관련 층간소음 민원 연간 약 8만 건.

#### 해외 유사 서비스 성공으로 문제 해결 가능성 입증

- **Rover (미국)**: 북미 최대 반려동물 돌봄 매칭 플랫폼. 누적 돌봄자 50만명+, 연간 예약 1,000만 건+. 2022년 기업가치 $2.3B로 SPAC 상장. GPS 산책 추적, 사진 리포트, 보험 적용 등의 기능으로 돌봄 매칭의 신뢰 문제를 해결. "Rover 보험이 있으니 안심하고 맡긴다"는 양육자 반응.
- **Wag! (미국)**: 산책대행 특화 플랫폼에서 종합 돌봄 플랫폼으로 확장. 실시간 GPS 산책 추적이 핵심 차별화 요소. 2022년 SPAC 상장 후 기업가치 약 $350M.
- **TrustedHousesitters (영국)**: 하우스시팅 + 펫시팅을 결합한 글로벌 플랫폼. 130개국 돌봄자 네트워크. "자기 집에서 돌봄받는" 모델로 반려동물 스트레스 최소화.
- **PetBacker (싱가포르)**: 아시아 최대 펫시팅 플랫폼. 배상책임보험 자동 적용, 실시간 사진·GPS 리포트. 아시아 시장에서 연 200% 이상 성장하며 "아시아판 Rover"로 자리매김. 펫커넥트가 참고할 핵심 벤치마크.

### 2.2 핵심 기능 상세

#### 2.2.1 모듈 1: 돌봄 매칭 시스템

**기능 설명:**
양육자(의뢰인)와 돌봄 제공자(케어기버)를 **위치·일정·반려동물 특성** 기반으로 자동 매칭한다.

**돌봄 유형:**

| 유형 | 설명 | 시간 단위 | 예상 가격대 |
|------|------|----------|------------|
| 산책대행 | 돌봄자가 양육자 집 방문 후 산책 | 30분 / 1시간 | 8,000 / 15,000원 |
| 펫시팅(방문) | 양육자 집에서 돌봄 (급식·놀이·산책) | 반일 / 1일 | 20,000 / 35,000원 |
| 펫시팅(위탁) | 돌봄자 집에서 보호 | 1일 | 25,000~40,000원 |
| 임시보호 | 장기(3일 이상) 위탁 돌봄 | 1일 | 20,000~30,000원 |
| 긴급돌봄 | 2시간 이내 매칭·출동 | 시간당 | 12,000원 |

**신뢰 확보 시스템:**
1. **돌봄자 검증 프로세스**
   - 신원조회 (범죄경력 조회 동의)
   - 반려동물 돌봄 교육 이수 필수 (온라인 8시간 + 오프라인 실습 4시간)
   - 반려동물 양육 경험 인증
   - AI 면접 (로컬 LLM 기반 상황 판단력 평가)
2. **보험 연동**: 돌봄 중 발생하는 사고에 대한 배상책임보험 자동 적용 (KB손해보험 제휴)
3. **상호 리뷰·평점 시스템**: 양육자·돌봄자 쌍방향 평가

#### 2.2.2 모듈 2: 실시간 돌봄 리포트

**기능 설명:**
돌봄 진행 중 양육자가 **실시간으로 반려동물의 상태를 확인**할 수 있는 모니터링 시스템.

**핵심 기능:**
- **GPS 실시간 추적**: 산책 경로를 지도에 실시간 표시 (산책 거리·시간·속도 자동 기록)
- **사진·영상 리포트**: 돌봄자가 30분 간격으로 사진/짧은 영상 업로드 (자동 알림)
- **상태 체크리스트**: 식사량, 배변 상태, 활동량, 기분 상태 등 구조화된 리포트
- **긴급 알림**: 반려동물 이상 행동 감지 시 즉시 알림 + 비대면 수의사 연결 옵션
- **오프라인 GPS 트래킹**: 로컬 AI 기반으로 **인터넷 연결 없이도 GPS 데이터를 기기에 저장**, 연결 복구 시 자동 동기화

#### 2.2.3 모듈 3: 건강관리 다이어리

**기능 설명:**
반려동물의 **전 생애주기 건강 정보를 통합 관리**하는 디지털 건강 수첩.

**핵심 기능:**
- **예방접종 일정 관리**: 종합백신(DHPPL), 코로나장염, 켄넬코프, 광견병 등 접종 이력 및 차기 일정 자동 알림
- **병원 진료 기록**: 진료일, 진단명, 처방약, 검사결과(혈액검사·X-ray 등) 기록·보관
- **일일 건강 로그**: 체중, 식사량, 음수량, 배변 상태, 활동량 기록
- **의료비 관리**: 진료비·약제비·보험 청구 내역 통합 관리
- **동물등록 정보 연동**: 농림축산식품부 동물등록 시스템 연동 (마이크로칩 번호 기반)

#### 2.2.4 로컬 AI 엔진 — 핵심 차별화 요소

펫커넥트의 **가장 핵심적인 기술적 차별점**은 모든 AI 기능을 **로컬(Local) LLM**으로 구현한다는 것이다. 클라우드 기반 AI(ChatGPT API, Google Gemini 등)가 아닌 **기기(온디바이스) 및 엣지 서버**에서 AI 추론을 수행한다.

**로컬 LLM을 선택한 3가지 이유:**

| 구분 | 클라우드 AI | 로컬 AI (펫커넥트) | 우위 |
|------|-----------|-------------------|------|
| **데이터 프라이버시** | 반려동물 사진·위치·건강정보가 외부 서버로 전송 | 모든 데이터가 사용자 기기 또는 자사 엣지 서버에서만 처리 | 반려동물 양육자의 개인정보(집 주소, 일정, 건강정보) 보호 |
| **오프라인 작동** | 인터넷 필수 | 인터넷 없이도 핵심 기능 작동 | 산책 중(지하도, 산, 공원) GPS 트래킹·건강 사진 분석 가능 |
| **비용 확장성** | API 호출당 과금 → 사용자 증가 시 비용 기하급수적 증가 | 초기 모델 개발 비용 후 추론 비용 거의 없음 | MAU 100만 기준 연간 AI 비용 약 90% 절감 |

**AI 기능 1: 건강 사진 분석 (Local Vision Model)**

| 항목 | 상세 |
|------|------|
| 모델 | MobileViT v2 (경량 비전 트랜스포머) Fine-tuned |
| 학습 데이터 | 반려동물 피부질환·안과질환·구강질환 이미지 약 50만장 (수의과대학 협력) |
| 분석 항목 | 피부(탈모·발적·종양), 눈(충혈·분비물·혼탁), 귀(외이염 징후), 구강(치석·잇몸염) |
| 추론 환경 | 온디바이스 (iOS CoreML / Android NNAPI) |
| 추론 시간 | 약 200ms (iPhone 14 기준) |
| 정확도 목표 | 주요 피부질환 분류 Top-3 Accuracy 85% 이상 |
| 출력 | 의심 질환명 + 심각도(상/중/하) + 병원 방문 권고 여부 + 유사 사례 이미지 |

**주의사항 표시:** "이 분석은 AI 참고 정보이며, 정확한 진단은 수의사 진료를 받으시기 바랍니다."

```
[사용자 흐름]
반려동물 사진 촬영 → 로컬 Vision 모델 추론 (온디바이스)
→ 분석 결과 표시 (의심 질환·심각도)
→ [선택] 비대면 수의사 상담 연결 (사진+건강다이어리 자동 공유)
```

**AI 기능 2: 펫케어 Q&A 챗봇 (Local Llama)**

| 항목 | 상세 |
|------|------|
| 모델 | Llama 3.2 3B (양자화: GGUF Q4_K_M) |
| 서빙 환경 | 엣지 서버 (자사 관리, llama.cpp 기반) + 온디바이스 (iOS/Android, 1B 경량 모델) |
| 파인튜닝 데이터 | 수의학 교과서, 반려동물 양육 가이드, 수의사 상담 Q&A 약 30만건 |
| 응답 범위 | 사료·영양, 행동교정, 질병 초기 대응, 응급처치, 품종별 특성, 훈련법 |
| 응답 시간 | 엣지 서버 기준 약 1.5초 (첫 토큰), 온디바이스 약 3초 |
| 안전장치 | 의료 관련 질문은 반드시 "수의사 진료 권고" 면책 문구 포함 |

```
[사용자 흐름 예시]
사용자: "우리 강아지가 풀을 자꾸 먹어요. 괜찮은 건가요?"
AI: "반려견이 풀을 먹는 행동은 비교적 흔한 현상입니다. 주요 원인으로는
① 위장 불편감 완화 ② 섬유질 보충 본능 ③ 지루함이나 스트레스 등이 있습니다.
다만, 잦은 구토를 동반하거나, 제초제가 뿌려진 풀을 먹었을 경우 위험할 수 있습니다.
2주 이상 지속되면 수의사 진료를 권장드립니다.
[비대면 수의사 상담 바로가기]"
```

**AI 기능 3: 돌봄자-반려동물 호환성 매칭 (Local Matching Model)**

| 항목 | 상세 |
|------|------|
| 모델 | LightGBM + Embedding 기반 유사도 모델 |
| 매칭 변수 | 돌봄자: 경험 품종, 거주 환경, 체력, 가용 시간, 리뷰 점수 / 반려동물: 품종, 크기, 성격, 특이사항, 건강상태 |
| 추론 환경 | 엣지 서버 (실시간 매칭) |
| 최적화 목표 | 돌봄 완료 후 양측 만족도 최대화 (리뷰 4.5점 이상 달성률) |
| 재학습 주기 | 주 1회 (누적 돌봄 데이터 기반 점진적 학습) |

```
[매칭 알고리즘 로직]
1차 필터: 위치(반경 3km) + 일정(가용 시간) + 반려동물 유형(개/고양이)
2차 스코어링: 호환성 점수 산출 (품종 경험 40% + 환경 적합성 30% + 평점 20% + 응답률 10%)
3차 최적화: 양육자 선호(가격/거리/평점 가중치) 반영 → Top-5 추천
```

#### 2.2.5 모듈 4: 비대면 수의사 상담 연동

**기능 설명:**
2024년 12월 비대면 수의사 진료 허용에 따라, 플랫폼 내에서 **화상 수의사 상담을 직접 연결**한다.

**연동 방식:**
- 자체 화상 상담 시스템 구축 (WebRTC 기반)
- 제휴 동물병원·수의사 네트워크 확보 (1차 목표: 서울·경기 50개소)
- 건강관리 다이어리 + AI 건강 사진 분석 결과를 **수의사에게 자동 공유** (양육자 동의 시)
- 진료 후 처방 기록 자동 반영

**경쟁 우위 요약:**
1. **통합 플랫폼**: 돌봄+건강+진료를 하나의 앱에서 (기존 서비스는 각각 분리)
2. **로컬 AI**: 프라이버시·오프라인·비용 3가지 우위를 동시 확보
3. **데이터 해자(Moat)**: 돌봄 기록 + 건강 데이터 + AI 분석 결과가 누적될수록 서비스 품질 향상 → 락인(Lock-in) 효과
