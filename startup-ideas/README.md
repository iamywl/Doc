## 각 문서 공통 구조 (예비창업패키지 양식 준수)

```
□ 일반현황          — 창업아이템명, 산출물, 팀 구성 현황
□ 창업 아이템 개요    — 명칭, 범주, 개요, 문제인식, 실현가능성, 성장전략, 팀구성 요약
1. 문제 인식 (Problem)  — 시장 현황, 문제점, 통계, 개발 필요성
2. 실현 가능성 (Solution) — 핵심 기능, 로컬 LLM 구동 방식, 기술 스택, 개발 일정, 예산
3. 성장전략 (Scale-up)   — 경쟁사 분석, 비즈니스 모델, 투자유치, ESG 사회적 가치
4. 팀 구성 (Team)       — 대표자 역량, 팀원, 협력기관
참고문헌              — 각 문서별 15~35개 참고문헌 수록
```

## 128GB 로컬 LLM 인프라 요약

모든 아이디어에 공통 적용되는 핵심 기술 인프라입니다. 상세 내용은 [공통기술 문서](00-로컬LLM-인프라-공통기술.md) 참조.

### 서버 사양
| 항목 | 사양 |
|------|------|
| CPU | AMD EPYC 7313P 16코어 / Intel Xeon w5-2465X |
| RAM | **DDR5 ECC 128GB** (32GB × 4) |
| GPU | NVIDIA RTX 4090 24GB VRAM |
| Storage | NVMe SSD 2TB |
| OS | Ubuntu 22.04 LTS |

### 128GB RAM이 필요한 이유
```
┌─────────────────────────────────────────────┐
│            128GB RAM 메모리 배분              │
├─────────────────────────────────────────────┤
│ Llama 3.1 70B (Q4_K_M) offload   ~40GB     │
│ Whisper large-v3 버퍼              ~2GB     │
│ YOLOv8 + CLIP 버퍼                 ~2GB     │
│ VITS TTS 버퍼                      ~1GB     │
│ Embedding 모델 (BGE-M3)           ~2GB     │
│ PostgreSQL + Redis                 ~8GB     │
│ Node.js 서버                       ~4GB     │
│ OS + 시스템                        ~8GB     │
│ KV-cache + 배치처리 버퍼          ~61GB     │
├─────────────────────────────────────────────┤
│ 합계                             128GB     │
└─────────────────────────────────────────────┘
```

### 구동 방식
```
[사용자] → [React Native 앱 / Next.js 웹]
              ↓
         [Node.js API 서버]
              ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
[Ollama]  [vLLM]  [llama.cpp]    ← 128GB RAM + RTX 4090
 LLM서빙   배치추론   CPU+GPU혼합
    ↓         ↓         ↓
[PostgreSQL + pgvector + Redis]
```

### 3년 비용 비교
| 항목 | 로컬 128GB 서버 | OpenAI GPT-4o API |
|------|----------------|-------------------|
| 초기 비용 | ~1,000만원 | 0원 |
| 월 운영비 | ~5만원 (전기) | ~500~2,000만원 |
| **3년 총비용** | **~1,180만원** | **1.8~7.2억원** |
| 데이터 보안 | 서버 내부 처리 | 외부 전송 필수 |
