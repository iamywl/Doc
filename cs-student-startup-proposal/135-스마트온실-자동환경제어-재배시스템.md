### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 센서 | SHT40, SCD41, PAR, EC/pH | 환경·토양·양액 데이터 수집 |
| 제어기 | 자체 개발 엣지 컨트롤러 (STM32) | 현장 자동 제어, 안전 로직 |
| 통신 | LoRaWAN + LTE Cat-M1 | 센서→엣지→클라우드 전송 |
| AI 엔진 | PyTorch + Prophet | 생육 예측, 환경 최적화 |
| 강화학습 | Stable Baselines3 (PPO) | 에너지 최적화 제어 정책 |
| 백엔드 | Django REST Framework | API 서버, 농가 대시보드 |
| 모바일 | Flutter | 농가용 원격 관리 앱 |
| 데이터베이스 | TimescaleDB + PostgreSQL | 시계열 환경 데이터 + 관계형 |
| 클라우드 | NCloud (네이버 클라우드) | 공공 클라우드 인프라 |
| 양액 제어 | 자체 펌프·밸브 + PID 제어 | 양액 EC·pH 자동 조절 |

## 9. 사회적 가치

### 9-1. 농가 소득 증대

AI 자동 환경제어로 수확량을 평균 27.9% 증가시키고, 에너지 비용을 20~30% 절감하여 농가 순소득을 크게 개선합니다.

### 9-2. 청년 농업인 유치

첨단 AI·IoT 기술 기반 농업으로 농업의 이미지를 혁신하여 청년 농업인의 진입을 촉진합니다. 현재 4.8%에 불과한 39세 이하 농업인 비율 개선에 기여합니다.

### 9-3. 식량 안보 기여

기후변화에 대응하는 AI 환경제어로 안정적인 식량 생산을 보장하고, 수확량 변동폭을 최소화합니다.

### 9-4. 탄소 중립 농업

에너지 사용 최적화와 정밀 양액 관리로 탄소 배출을 줄이고, 지속 가능한 농업 생산 방식에 기여합니다.

### 9-5. 농촌 디지털 전환

IT 기술의 농업 현장 적용을 통해 농촌의 디지털 전환을 가속하고, 데이터 기반 정밀 농업 생태계를 구축합니다.
