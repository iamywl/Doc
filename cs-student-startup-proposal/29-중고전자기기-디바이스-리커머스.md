## 1. 문제 인식 (Problem)

### 1.1 중고 전자기기 거래의 신뢰 문제

국내 중고 거래 시장은 빠르게 성장하고 있으나, 전자기기 분야에서는 품질 불확실성, 가격 불투명성, 개인정보 유출 위험이 주요 장벽으로 작용하고 있다. 한국소비자원에 따르면, 중고 전자기기 거래 관련 소비자 피해 상담은 2024년 기준 연간 12,000건을 초과하였다 [1].

특히 전자기기는 외관만으로는 내부 상태를 판단하기 어려워, 구매 후 숨겨진 결함이 발견되는 사례가 빈번하다. 배터리 성능 저하, 내부 수리 이력, 침수 흔적 등은 육안으로 확인이 불가능하며, 판매자의 양심에 의존하는 현재 시스템은 구조적으로 정보 비대칭 문제를 안고 있다.

또한 중고 전자기기를 통한 개인정보 유출은 심각한 사회 문제가 되고 있다. 초기화만으로는 데이터가 완전히 삭제되지 않으며, 전문 복구 도구를 사용하면 이전 사용자의 사진, 메시지, 금융 정보 등을 복원할 수 있다. 이러한 우려로 43%의 소비자가 중고 전자기기 거래 자체를 포기하고 있다.

#### 주요 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 중고 전자기기 거래 사기 피해 경험 | **25%** | 한국소비자원, 2024 [1] |
| 개인정보 유출 우려로 거래 포기 | **43%** | 개인정보보호위원회, 2024 [2] |
| 중고 스마트폰 적정 시세 모르는 비율 | **67%** | 중고나라, 2025 [5] |
| 국내 연간 전자폐기물 발생량 | **약 100만 톤** | 환경부, 2024 [6] |
| 중고 전자기기 재사용률 | **18%** | 한국환경산업기술원, 2024 [7] |
| 기기 상태 허위 기재로 인한 분쟁 | **38%** | 당근마켓 분쟁 통계, 2024 [8] |
| 중고 거래 후 하자 발견 비율 | **31%** | 한국소비자원, 2024 [9] |
| 기업 IT 자산 처분 시 데이터 유출 우려 | **76%** | 한국인터넷진흥원, 2024 [10] |

#### 사용자 목소리

> "중고 아이폰을 샀는데, 배터리 건강도가 80%라고 했는데 실제로는 72%였다. 확인할 방법이 없어서 속은 거다." — 중고 거래 구매자 A씨

> "오래된 노트북을 팔고 싶은데, 하드디스크에 있는 개인 파일이 걱정돼서 그냥 서랍에 넣어두고 있다." — 중고 거래 망설임 B씨

> "직원 200명이 쓰던 노트북을 교체했는데, 데이터 삭제 인증을 받을 수 있는 곳이 마땅치 않아 전부 파쇄 처리했다. 환경적으로도 낭비다." — 중견기업 IT 관리자 C씨

### 1.2 기존 플랫폼의 한계

- **당근마켓/중고나라**: 범용 중고 거래로 전자기기 전문 기능 부재. 기기 상태 검증, 데이터 삭제 인증 등 전자기기 특화 서비스 없음
- **번개장터**: 가격 산정 기준 없음, 기기 상태 검증 불가. 판매자 신고에 의존하는 수동적 품질 관리
- **셀잇/민트잇**: 스마트폰 한정, 노트북·태블릿·웨어러블 등 다양한 전자기기 미지원
- **직거래**: 사기 위험, 만남의 불편, 기기 검증 어려움, 사후 보상 없음

### 1.3 전자기기 거래 특수 페인포인트

| 페인포인트 | 판매자 | 구매자 | 기업(B2B) |
|-----------|--------|--------|-----------|
| 가격 산정 | 적정 가격 모름 (67%) | 바가지 우려 | 대량 처분 시 시세 파악 어려움 |
| 기기 상태 | 정직하게 기재해도 신뢰 못 받음 | 숨겨진 결함 우려 (31%) | 일일이 검수 불가능 |
| 개인정보 | 완전 삭제 방법 모름 | 이전 소유자 데이터 우려 | 규정 준수 필요 (76%) |
| 거래 안전 | 입금 후 연락 두절 우려 | 결제 후 상품 불일치 | 계약 불이행 리스크 |
| 환경 | 쓸 만한 기기 폐기 죄책감 | - | ESG 경영 요구 충족 필요 |

### 2.2 핵심 기능

1. **AI 시세 산정 엔진**: 실시간 시장 데이터(네이버 중고, 당근, 번개장터 등), 기기 상태, 출시 시기, 감가상각 모델을 종합하여 공정 시세 자동 산출. XGBoost 기반 시계열 모델로 시세 변동 예측까지 제공
2. **기기 자동 진단 앱**: 배터리 건강도, 화면 상태(터치 데드존 검사), 센서 동작(가속도계, 자이로, GPS), 카메라/마이크/스피커 테스트, 내부 부품 이상을 자동 검사하여 S/A/B/C 등급 부여
3. **데이터 완전 삭제 인증**: DoD 5220.22-M 기준 데이터 삭제(3회 덮어쓰기) 후 블록체인 기반 삭제 인증서 발급. 기업 고객의 경우 GDPR/개인정보보호법 컴플라이언스 보고서 제공
4. **에스크로 안전 결제**: 구매자 확인 후 대금 지급, 3일간 하자 신고 기간 운영. 분쟁 발생 시 기기 진단 데이터 기반의 객관적 중재
5. **등급 인증 시스템**: S/A/B/C 등급 분류로 기기 상태 표준화. 각 등급별 보증 기간 차등 적용

### 2.3 기기 진단 항목 상세

| **화면** | 터치 정확도, 데드존, 번인 | 전체 화면 터치 그리드 테스트 | 매우 높음 |
| **배터리** | 건강도, 충전 사이클, 예상 수명 | 배터리 API + 충방전 패턴 분석 | 높음 |
| **카메라** | 전/후면 작동, 초점, 노이즈 | 자동 촬영 + AI 이미지 품질 분석 | 중간 |
| **오디오** | 스피커, 마이크, 이어폰잭 | 주파수 응답 테스트 | 중간 |
| **센서** | 가속도, 자이로, 근접, 기압 | 센서 API 호출 + 정상 범위 검증 | 낮음 |
| **연결** | Wi-Fi, Bluetooth, NFC, GPS | 연결 테스트 + 신호 강도 측정 | 중간 |
| **성능** | CPU 벤치마크, 메모리, 저장장치 | 경량 벤치마크 실행 | 중간 |
