|------|------|
| 팀명 | 트루스가드(TruthGuard) |

## 3. 문제 인식

### 3.1 허위정보 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 한국인 가짜뉴스 경험률 | 84.2% | 한국언론진흥재단 |
| 가짜뉴스 사회적 비용 (연간) | 약 30조원 (추정) | 한국경제연구원 |
| SNS 허위정보 확산 속도 | 진실 대비 6배 빠름 | MIT Media Lab |
| 딥페이크 콘텐츠 증가율 (연간) | +400% | Deeptrace |
| 선거 기간 허위정보 건수 (2024) | 약 18,000건 | 중앙선거관리위원회 |
| 팩트체크 인력 (언론사 전체) | 약 50명 미만 | 서울대 팩트체크센터 |

| 건강 허위정보 피해 사례 (연간) | 약 3,200건 | 식약처 |
| 뉴스 신뢰도 (한국) | OECD 최하위권 (28%) | 로이터 저널리즘 연구소 |

### 3.2 이해관계자 인터뷰

> "팩트체크 기자가 한 건을 검증하는 데 하루 종일 걸립니다. AI가 1차 검증을 해주면 생산성이 10배 이상 올라갈 것입니다."
> — 주요 일간지 팩트체크팀 기자

> "선거 기간에 허위정보가 폭증하는데 인력으로는 대응이 불가능합니다. 자동화 도구가 절실합니다."
> — 중앙선거관리위원회 사이버선거범죄 대응팀

> "딥페이크 영상이 퍼지면 기업 주가가 급락합니다. 실시간으로 진위를 판별할 수 있는 시스템이 필요합니다."
> — 상장기업 IR팀장

> "건강 관련 가짜뉴스 때문에 검증되지 않은 치료법을 시도하는 분들이 많습니다. 생명에 관한 문제입니다."
> — 대학병원 가정의학과 교수

## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 팩트체크 엔진 | LLM 기반 주장 분석, 팩트DB 대조, 출처 검증을 30초 내 수행 |
| 딥페이크 탐지 | 영상/음성/이미지 딥페이크 여부 AI 판별 (정확도 96%) |
| 소셜미디어 모니터링 | 트위터/유튜브/커뮤니티 허위정보 실시간 감시 및 확산 추적 |
| 출처 추적 (Source Tracing) | 콘텐츠의 원본 출처 자동 추적, 변형 과정 시각화 |
| 신뢰도 점수 | 콘텐츠에 신뢰도 점수(0~100) 부여, 판단 근거 제시 |
| 팩트체크 리포트 자동생성 | AI가 검증 결과를 기자가 편집 가능한 리포트로 자동 작성 |
| 브라우저 확장 | 웹 브라우징 중 뉴스/게시물 신뢰도를 실시간 표시 |
| 기업 브랜드 모니터링 | 기업/인물 관련 허위정보 자동 감지 및 알림 |

## 5. 시장 분석

### 5.1 시장 규모 (TAM/SAM/SOM)

| 시장 구분 | 규모 | 산출 근거 |
|-----------|------|-----------|
