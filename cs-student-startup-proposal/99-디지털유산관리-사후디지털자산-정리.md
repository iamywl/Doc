## 4. 솔루션

### 4.1 핵심 기능

**① 디지털 자산 인벤토리**
- 온라인 계정(SNS, 이메일, 금융 등), 암호화폐 지갑, 클라우드 저장소 등 목록화
- 자산 유형별 분류 및 가치 추정
- 주기적 업데이트 알림 및 가이드

**② 안전한 유언 저장소**
- 영지식증명(ZKP) 기반 암호화 저장 - 운영자도 내용 열람 불가
- 유언 메시지, 비밀번호, 지시사항을 수신자별 지정
- 영상/음성/텍스트 유언 메시지 저장

**③ 사후 자동 전달 프로토콜**
- 사망 감지: 지정 기간 미접속 + 지정인 확인 + 공적 서류 검증
- 다단계 검증 후 지정된 수신자에게 자동 전달
- 스마트 컨트랙트 기반 암호화폐 자산 자동 이전

**④ 계정 관리 대행**
- 고인의 지시에 따른 계정 삭제 또는 추모 전환 대행
- 각 플랫폼별 사후 계정 처리 가이드 제공
- 법률사무소 연계 상속 절차 지원

### 4.3 기술 스택

| 구분 | 기술 | 선정 사유 |
|------|------|-----------|
| 암호화 | AES-256, ZKP (zk-SNARKs) | 최고 수준 데이터 보안, 운영자 열람 불가 |
| 블록체인 | Ethereum L2 (Polygon) | 낮은 가스비, 스마트 컨트랙트 안정성 |
| 키 관리 | AWS KMS, HSM | 하드웨어 기반 암호키 관리 |
| 백엔드 | Go (Gin) | 높은 보안성, 동시성 처리 |
| 프론트엔드 | Next.js 14 | SSR 기반 SEO, 빠른 로딩 |
| 모바일 | Flutter | iOS/Android 동시 개발 |
| 데이터베이스 | PostgreSQL (암호화) | FIPS 인증 암호화 저장 |
| 인증 | OAuth 2.0, FIDO2 | 다단계 인증, 생체 인증 지원 |
| 인프라 | AWS (VPC, Private Subnet) | 격리된 보안 환경 |
| 모니터링 | ELK Stack, PagerDuty | 보안 이벤트 실시간 감시 |
