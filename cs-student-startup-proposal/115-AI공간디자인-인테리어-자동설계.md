## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 공간 스캔 | 스마트폰 사진/LiDAR로 방 크기, 구조, 기존 가구를 자동 인식하여 3D 모델 생성 |
| AI 인테리어 디자인 | 스타일/예산/취향 입력하면 AI가 여러 인테리어 디자인 안 자동 생성 |
| 가구 자동 배치 | 공간 크기와 동선을 고려한 최적 가구 배치 AI 추천 |
| AR 미리보기 | 제안된 디자인을 AR로 실제 공간에 가상 배치하여 미리 체험 |
| 포토리얼 렌더링 | AI 생성 디자인을 포토리얼 3D 렌더링으로 시각화 (10초 이내) |
| 견적/구매 연동 | 제안된 가구/건자재의 즉시 견적 산출, 원클릭 구매 연동 |
| 디자이너 협업 도구 | 인테리어 디자이너가 AI를 활용하여 작업 속도 10배 향상 |
| 스타일 레퍼런스 AI | 사진/이미지 업로드하면 동일 스타일 인테리어를 우리 집에 적용 |

## 5. 시장 분석

### 5.1 시장 규모 (TAM/SAM/SOM)

| 시장 구분 | 규모 | 산출 근거 |
|-----------|------|-----------|
| TAM (전체 시장) | 45조원 | 국내 인테리어 시장 전체 |
| SAM (유효 시장) | 5,500억원 | 인테리어 디자인 서비스 + 온라인 가구 시장의 디자인 도구 |
| SOM (목표 시장) | 180억원 | 초기 3년 내 AI 인테리어 디자인 플랫폼 점유 목표 |

### 5.2 경쟁사 분석

| 경쟁사 | 유형 | 강점 | 약점 | 차별점 |
|--------|------|------|------|--------|
| 오늘의집 | 인테리어 플랫폼 | 대규모 사용자, 커머스 | AI 디자인 기능 미약 | 콘텐츠+쇼핑 vs AI 디자인 |
| 집꾸미기 | 인테리어 매칭 | 업체 네트워크 | 자동 설계 없음 | 매칭 vs 자동 설계 |
| Planner 5D (해외) | 3D 인테리어 | 3D 도구, 글로벌 | 한국 가구 DB 없음, AI 약함 | 수동 3D vs AI 자동 생성 |
| IKEA Place | AR 가구배치 | 자사 가구 AR | IKEA 가구만 | 단일 브랜드 vs 멀티 브랜드 |
