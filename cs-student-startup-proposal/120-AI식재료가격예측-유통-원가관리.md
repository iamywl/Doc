## 3. 문제 인식

### 3.1 식재료 가격 변동 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 외식산업 시장 규모 | 약 168조 원 | 한국외식산업연구원 (2025) |
| 외식업체 폐업률 (3년 이내) | 약 72.8% | 국세청 (2025) |
| 폐업 원인 중 '원가 관리 실패' 비율 | 34.2% | 소상공인시장진흥공단 (2025) |
| 주요 채소류 연간 가격 변동폭 | 최대 300% (배추, 대파 등) | 한국농수산식품유통공사 (2025) |
| 급식업체 식재료비 비중 | 매출 대비 42~55% | 한국급식산업연구원 (2025) |
| 식재료 가격 정보 획득 경로 | 78%가 '경험과 감' | 외식경영학회 (2025) |
| 도매시장 가격 정보 실시간 제공률 | 23.5% | 서울시농수산식품공사 (2025) |
| 식재료 폐기율 (가격 변동 관련) | 약 8.7% | 환경부 음식물 쓰레기 통계 (2025) |

### 3.2 이해관계자 인터뷰

> "대파 가격이 3배 뛴 적이 있는데, 미리 알았으면 메뉴 구성을 바꿨을 겁니다. 한 달 적자가 800만 원이었어요."
> — 서울 강남 한식당 대표 박OO (48세)

> "400명 급식을 하는데, 식재료비가 계약 단가를 넘어가면 손해를 저희가 다 감수합니다. 가격 예측이 절실해요."
> — 경기도 학교 급식업체 영양사 김OO (36세)

> "100개 매장 프랜차이즈인데, 식재료 원가 1%만 줄여도 연 10억 이상 절감됩니다."
> — 치킨 프랜차이즈 본부 구매팀장 이OO (42세)

> "농산물 시세가 너무 요동쳐서, 언제 사야 하는지 매번 고민입니다. 데이터 기반 판단이 필요해요."
> — 식자재 마트 운영자 정OO (51세)

### 4.3 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 시계열 예측 | TFT (Temporal Fusion Transformer) | 다변량 시계열 가격 예측 핵심 모델 |
| 앙상블 모델 | N-BEATS, Prophet, LightGBM | 예측 정확도 향상을 위한 앙상블 |
| 데이터 파이프라인 | Apache Airflow, Spark | 데이터 수집·전처리 자동화 |
| 백엔드 | FastAPI, Python | API 서버, 예측 서비스 |
| 데이터베이스 | PostgreSQL, ClickHouse | 시계열 가격 데이터, 분석 쿼리 |
| 캐시/큐 | Redis, Celery | 실시간 가격 캐시, 비동기 예측 작업 |
| 프론트엔드 | React, D3.js, Recharts | 대시보드, 가격 차트 시각화 |
| 모바일 | Flutter | iOS/Android 알림 앱 |
| 인프라 | AWS (ECS, S3, SageMaker) | 모델 학습/배포, 데이터 저장 |
| 모니터링 | MLflow, Grafana | 모델 성능 추적, 시스템 모니터링 |

## 5. 시장 분석

### 5.1 시장 규모 (TAM/SAM/SOM)

| 구분 | 규모 | 산출 근거 |
|------|------|-----------|
| TAM (전체 시장) | 약 5조 원 | 국내 식자재 유통·구매 관련 IT 시장 |
| SAM (유효 시장) | 약 8,000억 원 | 외식/급식업 원가 관리 솔루션 시장 |
| SOM (목표 시장) | 약 200억 원 | AI 식재료 가격 예측 SaaS (3년 내) |

### 5.2 경쟁사 분석

| 경쟁사 | 강점 | 약점 | 프라이스팜 차별점 |
|--------|------|------|-------------------|
