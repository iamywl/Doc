### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| OBD 하드웨어 | ELM327 호환 BLE 어댑터 | 차량 ECU 데이터 수집 |
| 이상 탐지 | Isolation Forest + Autoencoder | OBD 데이터 이상 패턴 감지 |
| 비전 AI | YOLOv8 + Segment Anything | 외관 결함·도장 분석 |
| 가격 예측 | XGBoost + LightGBM 앙상블 | 적정 가격 범위 산출 |
| 백엔드 | FastAPI + PostgreSQL | API 서버, 진단 데이터 관리 |
| 모바일 | Flutter | 진단 앱 (iOS/Android) |
| 프론트엔드 | React | 웹 대시보드 (딜러·B2B) |
| 데이터 파이프라인 | Apache Airflow | 시세·이력 데이터 수집 자동화 |
| 인프라 | AWS EC2 + S3 + SageMaker | 클라우드 AI 추론 |
| 보안 | AES-256 + SSL | 차량 데이터 암호화 |
