## 1. 문제 인식

### 식품 알레르기 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 성인 식품 알레르기 유병률 | 5.6% | 대한알레르기학회 (2025) |
| 소아 식품 알레르기 유병률 | 8.9% | 질병관리청 (2024) |
| 아나필락시스 응급실 내원 | 15,000건/년 | 건강보험심사평가원 (2025) |
| 식품 성분표 오독 경험 | 43.2% | 식품안전정보원 (2024) |
| 알레르기 관련 연 의료비 | 인당 120만원 | 국민건강보험공단 (2025) |
| 식품 알레르기 진료비 총액 | 연 3,000억원 | 건강보험심사평가원 (2025) |
| 알레르기 표시 위반 적발 건수 | 연 2,800건 | 식품의약품안전처 (2025) |
| 외식 시 알레르기 사고 비율 | 전체의 62% | 한국소비자원 (2024) |

식품 알레르기 환자가 증가하는 가운데, 복잡한 성분표 확인의 어려움, 교차 오염 정보 부족, 외식 시 안전 확인 불가 등이 일상의 주요 문제다. 특히 영유아 자녀를 둔 부모의 성분 확인 스트레스가 매우 높다.

### 사용자 페인포인트 심층 분석

**실제 알레르기 환자/보호자 인터뷰 (2025년 10월, 70명 대상)**

> "아이가 우유 알레르기인데, '카세인나트륨'이 우유 유래 성분인지 몰라서 먹였다가 두드러기가 났어요." - 34세, 3세 아이 엄마

> "성분표에 글씨가 너무 작고, '대두'가 다른 이름으로 적혀 있으면 놓치기 쉬워요. 콩 알레르기인데 '레시틴(대두)'을 못 봤어요." - 28세, 대두 알레르기 환자

> "외식할 때 매번 '이 메뉴에 새우 들어갔나요?' 물어보는 게 너무 불편해요. 직원도 정확히 모르는 경우가 많고요." - 25세, 갑각류 알레르기 대학생

> "비건 식단을 하는데, 생각보다 동물성 성분이 숨어있는 제품이 많아요. 하나하나 확인하는 데 시간이 너무 걸려요." - 31세, 비건 직장인

**핵심 페인포인트 5가지:**

1. **성분표 해독 어려움**: 화학명, 이명(異名)으로 표기된 알레르기 유발 성분을 일반인이 식별하기 어려움 (예: 카제인 = 우유, 글루텐 = 밀)
2. **교차 오염 정보 부재**: "같은 제조 라인에서 땅콩을 처리합니다" 등의 교차 오염 경고가 불충분
3. **외식 불안**: 프랜차이즈도 알레르기 정보 제공이 의무화되었으나 현장 준수율 45% 미만
4. **교차 반응 무지**: 새우 알레르기 환자가 게, 랍스터도 위험할 수 있다는 교차 반응 정보 부족
5. **대체 식품 탐색 피로**: 안전한 대체 식품을 찾기 위해 매장에서 수십 개 제품을 비교해야 하는 시간 소모

### 식품 알레르기의 의학적 심각성

식품 알레르기는 가벼운 피부 증상부터 생명을 위협하는 아나필락시스까지 다양한 중증도를 보인다:

- **경증 (Grade 1)**: 피부 발진, 두드러기, 가려움 → 전체 반응의 60%
- **중등증 (Grade 2)**: 구토, 복통, 안면 부종, 호흡 곤란 → 전체 반응의 30%
- **중증 (Grade 3, 아나필락시스)**: 혈압 저하, 기도 폐쇄, 의식 저하 → 전체 반응의 10%, 치료 지연 시 사망 위험
- **한국의 주요 알레르기 유발 식품**: 계란(34%), 우유(22%), 새우(12%), 밀(9%), 땅콩(8%), 대두(6%), 과일류(5%), 기타(4%)

### 핵심 기능 상세

#### 2-1. 바코드/OCR 성분 스캔

- **바코드 스캔**: 1D/2D 바코드 인식으로 식품 DB에서 즉시 성분 정보 조회 (응답 시간 0.3초)
- **OCR 성분 스캔**: 성분표 사진 촬영 → PaddleOCR 한국어 특화 모델로 텍스트 추출 → 정확도 96%
- **성분명 NER**: BERT 기반 Named Entity Recognition으로 성분명, 첨가물, 알레르기 유발 물질 자동 식별
- **이명 매핑**: "카세인나트륨"→"우유 유래", "레시틴(대두)"→"대두 유래" 등 1,200개 이상의 이명-원재료 매핑
- **수입 제품 지원**: 영어, 일본어, 중국어 성분표 OCR 지원 (향후 확장)

#### 2-2. 알레르기 경고 시스템

- **3단계 경고**: 안전(녹색) / 주의(황색, 교차 반응 가능성) / 위험(적색, 직접 알레르기 유발)
- **교차 반응 분석**: 알레르기 간 교차 반응 정보를 Knowledge Graph로 관리 (예: 새우↔게 교차율 75%, 라텍스↔바나나 교차율 35%)
- **"May Contain" 분석**: "이 제품은 땅콩을 사용한 시설에서 제조되었습니다" 등의 교차 오염 경고 탐지
- **가족 프로필**: 최대 10명의 가족 알레르기 프로필 관리, 한 번 스캔으로 전 가족의 안전 여부 확인
- **위험도 점수**: 성분별 알레르기 유발 위험도를 0-100 점수로 정량화

#### 2-3. 대체 식품 추천

- **안전 대체품 추천**: 알레르기 유발 식품의 안전한 대체 제품을 카테고리/가격/영양소 기준으로 추천
- **영양소 비교**: 원래 제품과 대체 제품의 영양소(칼로리, 단백질, 탄수화물 등) 비교 차트 제공
- **가격 비교**: 주요 온라인 쇼핑몰의 가격 비교 및 최저가 링크 제공
- **사용자 리뷰**: 같은 알레르기를 가진 사용자들의 대체 식품 리뷰 및 평점 공유
- **하이브리드 추천 알고리즘**: Collaborative Filtering + Content-Based Filtering으로 개인화 추천

#### 2-4. 외식 안전 가이드

- **프랜차이즈 메뉴 DB**: 주요 프랜차이즈 50곳의 메뉴별 알레르기 성분 정보 제공
- **매장 안전 등급**: 알레르기 대응 수준(별도 조리, 성분 표시, 직원 교육 등)에 따른 매장 등급 평가
- **위치 기반 검색**: 현재 위치 기반 알레르기 안전 식당 검색, 지도 표시
- **번역 카드**: 해외 여행 시 사용할 수 있는 다국어 알레르기 안내 카드 (12개 언어)

### 알레르기 분석 알고리즘 상세

AllerScan의 핵심 기술인 알레르기 분석 파이프라인은 다음과 같다:

1. **입력 처리**: 바코드 → 식품 DB 조회 / OCR → PaddleOCR 텍스트 추출 → 전처리 (노이즈 제거, 정규화)
2. **성분명 NER**: BERT 기반 NER 모델이 텍스트에서 성분명/첨가물/알레르기 항원을 추출 (F1-score 0.94)
3. **이명 해소**: 1,200개 이상의 이명-원재료 매핑 DB를 통해 화학명/상품명을 표준 알레르기 항원으로 변환
4. **교차 반응 분석**: Neo4j Knowledge Graph에서 알레르기 항원 간 교차 반응 경로를 탐색, 위험도 계산
5. **사용자 프로필 매칭**: 사용자/가족의 알레르기 프로필과 성분 목록을 매칭하여 3단계 경고 생성
6. **추천 생성**: 위험 제품의 대체 식품을 CF+CBF 하이브리드 알고리즘으로 추천, 영양소/가격 비교 데이터 첨부

### 시장 성장 동인

1. **알레르기 환자 지속 증가**: 환경 변화, 식생활 서구화로 식품 알레르기 유병률 매년 5.8% 증가
2. **식품 알레르기 표시제 강화**: 2025년 개정 식품위생법으로 알레르기 표시 의무 품목 22개로 확대
3. **건강 소비 트렌드**: 클린라벨, 비건, 글루텐프리 등 성분에 민감한 소비자 층 확대
4. **부모 세대의 높은 관심**: 영유아 식품 알레르기에 대한 부모의 불안감 증가, 앱 기반 관리 수요 상승
5. **글로벌 확장 가능성**: 식품 알레르기는 글로벌 이슈, 한국 시장 검증 후 일본/동남아 진출 용이
