## 1. 문제 인식

### 법률 문서 작성의 현실

| 지표 | 수치 | 출처 |
|------|------|------|
| 법률 비용 부담으로 계약서 미검토 비율 | 72% | 중소기업중앙회 (2025) |
| 불공정 계약 피해 소상공인 수 (연간) | 약 34만건 | 공정거래위원회 (2025) |
| 변호사 계약서 검토 평균 비용 | 50~200만원 | 대한변호사협회 (2025) |
| 프리랜서 계약서 미작성 비율 | 61% | 한국프리랜서연합 (2024) |
| 스타트업 법률 분쟁 경험률 | 38% | 스타트업얼라이언스 (2025) |
| 소상공인 계약 관련 분쟁 증가율 (전년 대비) | +23% | 법원행정처 (2025) |
| 온라인 계약서 검색량 증가율 | +67% | 네이버 데이터랩 (2025) |

소상공인과 프리랜서의 대다수가 높은 법률 비용 때문에 전문적인 계약서 검토 없이 사업을 진행하고 있다. 이로 인해 불공정 조항에 노출되거나 법적 분쟁이 발생하는 사례가 매년 증가하고 있으며, 특히 임대차 계약, 용역 계약, 근로 계약에서 피해가 집중된다.

### 사용자 페인포인트 분석

**소상공인 김사장 (카페 운영, 3년차)**
> "임대차 계약을 갱신하는데 임대인이 보낸 계약서에 불리한 조항이 있는지 확인할 방법이 없어요. 변호사 상담은 50만원 이상이라 부담이 너무 크고, 인터넷에서 찾은 템플릿은 제 상황에 맞는지 확신이 안 됩니다."

**프리랜서 박디자이너 (UI/UX, 5년차)**
> "클라이언트가 보내준 용역 계약서에 서명했는데, 나중에 알고 보니 저작권이 전부 클라이언트에게 귀속되는 조항이 있었어요. 작업물을 포트폴리오에도 못 쓰게 됐습니다."

**스타트업 대표 이개발 (IT 서비스, 1년차)**
> "투자 계약서, 주주간 계약서, 직원 근로계약서, NDA 등 작성해야 할 법률 문서가 너무 많은데, 매번 변호사에게 맡기면 비용이 감당이 안 됩니다."

### 핵심 기능

- **AI 계약서 템플릿 생성**: 계약 유형 선택 후 질문 응답만으로 맞춤형 계약서 자동 생성. 사용자가 "카페 임대차 계약서를 작성하고 싶습니다"라고 입력하면, AI가 업종, 면적, 보증금 등 필요한 정보를 단계별로 질문하고 법률적으로 완전한 계약서를 생성한다.
- **조항 분석 엔진**: NLP 기반 각 조항의 법적 의미와 리스크 수준을 분석하여 시각화. KoELECTRA 모델을 법률 데이터로 파인튜닝하여 한국어 법률 문서 이해도를 극대화했으며, 각 조항에 대해 1~5단계 리스크 점수를 부여하고 일반인이 이해할 수 있는 해설을 제공한다.
- **불공정 조항 감지**: 공정거래위원회 불공정 약관 기준에 따른 자동 감지 및 대안 제시. 약관규제법 제6조~제14조에 해당하는 불공정 조항 패턴 120가지를 학습하여 자동으로 감지하고, 공정한 대안 조항을 제안한다.
- **전자 서명 통합**: PKI 기반 전자 서명으로 계약 체결부터 보관까지 원스톱 처리. 전자서명법에 따른 공인전자서명과 간편인증(카카오, PASS 등)을 지원하며, 블록체인 기반 타임스탬프로 계약 시점을 증명한다.

### AI 알고리즘 상세 설명

**1. 계약서 생성 파이프라인 (LLM + RAG)**

계약서 생성 엔진은 Retrieval-Augmented Generation(RAG) 아키텍처를 기반으로 한다. 사용자의 입력 정보를 바탕으로 벡터 데이터베이스에서 관련 판례, 법령, 기존 템플릿을 검색하고, 이를 컨텍스트로 GPT-4 파인튜닝 모델에 전달하여 법률적으로 정확한 계약서를 생성한다.

- 벡터 DB: Pinecone을 활용하여 50만건의 판례와 전체 법령을 임베딩 저장
- 검색 정확도: Top-5 관련 문서 검색 정확도 94.2% 달성 (자체 벤치마크)
- 생성 품질: 변호사 3인 블라인드 평가 결과 평균 4.2/5.0점 (일반 LLM 대비 +1.8점)

**2. 불공정 조항 탐지 알고리즘**

약관규제법에 정의된 불공정 약관 유형을 기반으로 120가지 탐지 패턴을 구축했다. KoELECTRA 모델을 법률 말뭉치(판례 50만건, 약관 10만건)로 추가 학습하여 한국어 법률 문서 이해 능력을 크게 강화했다.

- 탐지 정확도: F1-Score 0.91 (불공정 조항 탐지 기준)
- False Positive Rate: 8% 이하 (불필요한 경고 최소화)
- 처리 속도: 10페이지 계약서 기준 평균 3.2초
