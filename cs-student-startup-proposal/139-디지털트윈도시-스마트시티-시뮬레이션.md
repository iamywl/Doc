### 4.1 플랫폼 개요

시티미러는 도시의 건물, 도로, 인프라를 1:1 디지털 복제하고, 실시간 IoT 데이터를 반영하여 교통·에너지·재난·환경 시나리오를 AI 시뮬레이션하는 웹 기반 디지털 트윈 플랫폼입니다.

| 시나리오 비교 | 복수 정책 시나리오를 동시 비교하여 최적 대안 도출 |

### 4.4 기술 스택

| 계층 | 기술 |
|------|------|
| 3D 렌더링 | WebGPU + Three.js + Cesium.js (3D Tiles) |
| 도시 모델링 | CityGML 3.0 + IFC (BIM) + LAS (포인트클라우드) |
| AI/ML | PyTorch (교통 예측), SUMO (교통 시뮬레이션), OpenFOAM (유체역학) |

| 데이터베이스 | PostGIS (공간 DB) + TimescaleDB + Redis |
| 스트리밍 | Apache Kafka + Apache Flink (실시간 처리) |
| 인프라 | AWS EKS + GPU 인스턴스 (렌더링/시뮬레이션) |
| IoT 연동 | MQTT + OGC SensorThings API |
| 시각화 | D3.js + Deck.gl (2D 대시보드) |
| DevOps | Terraform + ArgoCD + GitHub Actions |
