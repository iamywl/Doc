## 1. 문제 인식

### 대학생 재무 관리 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 월 생활비 파악 못하는 비율 | 54.2% | 한국장학재단 2025 |
| 가계부 앱 1개월 내 이탈률 | 72.4% | 한국금융교육연구소 2025 |
| 월 과소비 경험 비율 | 67.8% | 대학내일 2025 |
| 카드/간편결제 비중 | 89.3% | 한국은행 2025 |
| 재무 관리 교육 경험 비율 | 12.7% | 금융감독원 2025 |
| 자동 분류 기능 요구 | 83.5% | 자체 설문 (n=1,500) |
| 또래 비교 관심도 | 76.2% | 자체 설문 (n=1,500) |
| 월말 잔액 부족 경험 | 61.5% | 한국장학재단 2025 |

대학생들은 간편결제와 카드 사용이 일상화되면서 현금 흐름에 대한 감각을 잃어가고 있다. 기존 가계부 앱은 수동 입력 부담이 크고, 대학생 특유의 소비 카테고리(학식, 교재, 동아리비 등)를 지원하지 않아 이탈률이 높다. 또한 자신의 소비가 합리적인지 판단할 기준이 없어, 또래 비교 데이터에 대한 수요가 크다.

### 사용자 Pain Point 심층 분석

**가계부 이탈 원인**
- "일일이 입력하는 게 귀찮아서 3일 만에 포기했어요." (21세, 대학생)
- "카테고리가 너무 복잡하고, 학식이나 MT비 같은 항목이 없어요." (22세, 대학생)
- "기록만 하고 분석 결과가 없으니 동기부여가 안 돼요." (23세, 대학생)

**소비 관리 어려움**
- "배달 앱 결제가 많은데, 한 달에 얼마나 쓰는지 정확히 몰라요." (20세, 대학생)
- "친구들은 월 얼마 쓰는지 궁금한데 물어보기 애매해요." (24세, 대학생)
- "학기 중이랑 방학 때 소비 패턴이 완전 달라서 예산 짜기가 어려워요." (22세, 대학생)

### 기존 가계부 앱의 한계점

| 한계 영역 | 기존 앱 문제점 | MoneyMate 해결 방안 |
|-----------|---------------|-------------------|
| 입력 방식 | 수동 입력 중심, 번거로움 | 오픈뱅킹/카드사 자동 연동 |
| 카테고리 | 일반 성인 기준 분류 | 대학생 특화 30+ 카테고리 |
| 분석 기능 | 단순 합계/그래프 | AI 기반 패턴 분석, 예측 |
| 동기 부여 | 기록만 가능 | 또래 비교, 게이미피케이션 |
| 교육 | 금융 교육 부재 | 소비 습관 코칭, 금융 리터러시 |
| 맞춤화 | 범용적 기능 | 학기/방학 모드, 학교별 물가 반영 |

### 핵심 기능

1. **AI 자동 지출 분류**: 오픈뱅킹/카드사 연동으로 거래 내역 자동 수집, NLP 기반 가맹점명 분석으로 30+ 대학생 특화 카테고리 자동 분류. KoNLPy와 자체 학습 모델을 결합하여 "이디야커피 신촌점" → "카페/음료", "CU 연대점" → "편의점/간식" 등 95% 이상 정확도로 분류한다.

2. **소비 패턴 분석**: 주간/월간 소비 트렌드, 요일별 지출 패턴, 충동 소비 감지 알림. 사용자의 과거 3개월 데이터를 기반으로 소비 패턴을 학습하고, 평소 대비 150% 이상 지출 시 실시간 충동 소비 경고를 발송한다. 학기 중/방학/시험 기간 등 대학생 고유의 시간적 패턴도 반영한다.

3. **또래 비교 인사이트**: 같은 지역/학교/생활비 수준의 익명 소비 데이터 비교, 카테고리별 순위. "같은 학교 학생들 중 식비 지출 상위 30%입니다" 등의 맞춤형 인사이트를 제공하며, 모든 비교 데이터는 완전 익명화 처리된다.

4. **절약 목표 게이미피케이션**: 절약 목표 설정, 달성 시 배지/포인트 지급, 월간 절약 챌린지. "이번 주 커피 지출 2만원 이내", "배달 앱 주 2회 이내" 등 구체적 목표를 설정하고, 달성 시 경험치와 배지를 획득하는 시스템이다. 월간 랭킹과 챌린지로 지속적 동기 부여를 제공한다.

5. **스마트 예산 알림**: 카테고리별 예산 설정, 80%/100% 도달 시 실시간 알림. 과거 소비 패턴을 기반으로 AI가 적절한 예산을 추천하며, 예산 초과 위험 시 대안 소비를 제안한다 (예: "이번 달 카페 예산이 90% 소진되었습니다. 학교 무료 커피머신을 이용해보세요").

6. **금융 리터러시 코칭**: 소비 데이터 기반 맞춤형 금융 교육 콘텐츠 제공. 복리 계산기, 투자 기초 가이드, 신용점수 관리 팁 등 대학생에게 필요한 금융 지식을 소비 데이터와 연계하여 전달한다.

### 경쟁 우위 상세 분석

**뱅크샐러드 대비**: 뱅크샐러드는 전 연령층 대상 종합 자산관리 앱으로, 대학생 특화 기능이 부재하다. MoneyMate는 학식/동아리비/교재 등 대학생만의 카테고리와 또래 비교 기능으로 차별화한다.

**토스 대비**: 토스는 결제/송금 중심의 슈퍼앱으로, 가계부 기능은 부가 기능에 불과하다. MoneyMate는 소비 분석과 행동 변화에 집중하는 전문 PFM 앱이다.

**편한가계부 대비**: 수동 입력 방식의 전통적 가계부 앱으로, AI 분석이나 소셜 비교 기능이 없다. MoneyMate는 자동 분류와 인사이트 기반 코칭으로 이탈률을 크게 낮춘다.
