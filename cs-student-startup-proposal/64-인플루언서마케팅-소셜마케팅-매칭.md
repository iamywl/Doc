## 1. 문제 인식

### 인플루언서 마케팅 시장 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 인플루언서 마케팅 시장 규모 | 1.8조원 | 한국인터넷마케팅협회 (2025) |
| 인플루언서 마케팅 활용 기업 비율 | 78% | 대한상공회의소 (2025) |
| ROI 측정 불가 기업 비율 | 67% | 한국인터넷마케팅협회 (2025) |
| 가짜 팔로워 피해 경험 | 52% | 자사 설문조사 (2025) |
| 적합한 인플루언서 찾기 어려움 | 71% | 마케팅연구소 (2025) |
| 캠페인 관리에 소요되는 평균 시간 | 주 15시간 | 자사 설문조사 (2025) |
| 인플루언서당 평균 검토 시간 | 2.5시간 | 마케팅연구소 (2025) |

인플루언서 마케팅은 가장 효과적인 디지털 마케팅 채널이지만, 적합한 인플루언서를 찾기 어렵고, 가짜 팔로워에 의한 허위 지표, ROI 측정 불가 등의 문제가 심각하다. 특히 마케팅 전담 인력이 부족한 중소 브랜드는 인플루언서 선정부터 계약, 성과 측정까지 전 과정에서 어려움을 겪는다.

### 사용자 페인포인트 분석

> "인플루언서를 찾고, 연락하고, 계약하고, 콘텐츠 확인하는 데 2주가 걸려요. 이걸 매달 반복해야 해요." - D2C 커머스 대표 이OO

> "캠페인 끝나고 효과가 있었는지 아닌지 알 수가 없어요. 좋아요 수만 보고 판단해야 해요." - 식품 브랜드 마케터 박OO

### 주요 기능 상세

- **AI 인플루언서 추천**: 브랜드 카테고리, 타겟 고객, 예산에 맞는 최적 인플루언서를 AI가 추천 (매칭 정확도 92%). BERT 기반 콘텐츠 카테고리 분류와 GNN(Graph Neural Network) 기반 소셜 영향력 분석을 결합하여 단순 팔로워 수가 아닌 실질적 영향력과 브랜드 적합도를 평가한다. 예상 도달률, 참여율, 전환율을 사전에 예측하여 ROI 기대치를 제시한다.

- **팔로워 진위 검증**: 가짜 팔로워, 봇 계정, 인위적 참여율을 탐지하여 실제 영향력 점수 산출. Isolation Forest와 LSTM 기반 이상 탐지 모델이 팔로워 성장 패턴, 참여 패턴, 댓글 품질을 분석하여 가짜 팔로워 비율을 0.1% 단위로 산출한다. "진위 검증 배지" 시스템으로 검증된 인플루언서를 식별한다.

- **캠페인 통합 관리**: 제안 발송, 계약, 콘텐츠 검수, 일정, 결제까지 원스톱으로 관리. 전자 계약서 자동 생성, 에스크로 결제(콘텐츠 승인 후 지급), 캘린더 기반 일정 관리, 실시간 메시지 기능을 제공한다.

- **ROI 자동 분석**: 캠페인별 도달률, 참여율, 전환율, ROAS를 실시간 추적 및 자동 리포트 생성. UTM 파라미터 자동 생성, 전환 추적 픽셀 연동, A/B 테스트 비교 분석을 지원한다.

- **숏폼 콘텐츠 분석**: 릴스, 숏츠, 틱톡 등 숏폼 콘텐츠의 성과를 전용 분석 대시보드로 추적한다. 시청 완료율, 공유율 등 숏폼 특화 지표를 제공한다.

- **경쟁사 벤치마킹**: 동종 업계 경쟁사의 인플루언서 마케팅 현황을 분석하여 벤치마크 리포트를 제공한다. 어떤 인플루언서와 협업했는지, 어떤 성과를 냈는지를 추적한다.

### AI 매칭 알고리즘 상세

| 단계 | 모델 | 기능 | 성능 |
|------|------|------|------|
| 1단계: 카테고리 분류 | KoBERT | 콘텐츠/프로필 텍스트 기반 카테고리 분류 | F1 Score 94.3% |
| 2단계: 영향력 분석 | GraphSAGE (GNN) | 소셜 그래프 기반 실질 영향력 산출 | 기존 대비 35% 정확 |

| 5단계: ROI 예측 | Gradient Boosting | 과거 캠페인 데이터 기반 ROI 예측 | MAPE 18.2% |

### 시장 성장 동력

숏폼 콘텐츠(릴스, 숏츠, 틱톡) 확산과 마이크로/나노 인플루언서 마케팅 트렌드로 시장이 세분화되고 있으며, 이에 따라 데이터 기반 매칭과 성과 측정 솔루션에 대한 수요가 급증하고 있다.

| 성장 동력 | 상세 내용 | 영향도 |
|----------|----------|--------|
| 숏폼 콘텐츠 폭발 | 릴스/숏츠/틱톡 콘텐츠 소비 연 45% 증가 | 매우 높음 |
| 마이크로 인플루언서 부상 | 팔로워 1천~5만 규모, 높은 참여율(5~8%), 합리적 비용 | 높음 |
| D2C 커머스 성장 | 자체 브랜드 온라인 판매 기업 증가, 인플루언서 의존도 높음 | 높음 |
| 광고 차단(Ad Block) 확산 | 기존 배너 광고 효과 감소, 인플루언서 네이티브 광고 선호 | 중간 |
| 라이브커머스 성장 | 인플루언서 기반 실시간 판매 채널 확대 | 높음 |
| AI 마테크 수요 | 데이터 기반 마케팅 의사결정 도구 수요 증가 | 높음 |
