## 1. 문제 인식

### 부동산 가치평가의 정보 비대칭

| 지표 | 수치 | 출처 |
|------|------|------|
| 연간 부동산 거래 건수 | 120만건 (2025년) | 국토교통부 |
| 감정평가 비용 | 건당 50~200만원 | 한국감정원 |
| 감정평가 소요 기간 | 7~14일 | 한국감정평가사협회 |
| 부동산 투자 실패율 | 32% (기대 수익 미달성) | 한국부동산분석학회 |
| 정보 비대칭으로 인한 과다 지불 | 평균 7~12% | 부동산114 |
| 부동산 관련 민원 건수 | 연 28만건 | 국토교통부 |
| 1인 가구 중 부동산 정보 접근 어려움 | 68% | 한국부동산원 |
| 공인중개사 가격 판단 오류율 | 18% | 한국공인중개사협회 |
| 부동산 데이터 분석 서비스 이용 의향 | 82% | KB부동산 |
| 재개발/재건축 정보 부재로 투자 실패 | 연 1.8조원 추정 피해 | 한국경제연구원 |

부동산은 대부분의 한국인에게 가장 큰 자산이지만, 정확한 가치를 평가하기 어렵다. 전문 감정평가는 비용과 시간이 많이 소요되고, 개인 투자자는 호가와 실거래가 간 괴리, 주변 개발 호재 영향, 적정 임대 수익률 등을 종합 판단할 역량이 부족하다.

### 사용자 페인 포인트 분석

**부동산 매수 희망자:**
> *"이 아파트가 적정 가격인지, 더 오를 여지가 있는지 판단하기 어렵습니다. 네이버 부동산, 호갱노노 다 봐도 확신이 안 서요."*

> *"재개발 예정이라는데 정말인지, 언제쯤 될지 확인할 방법이 없습니다."*

**임대 투자자:**
> *"월세 수익률이 은행 이자보다 나은지 정확히 계산하고 싶습니다. 공실률, 관리비, 세금까지 고려한 순수익률을 알고 싶어요."*

**공인중개사:**
> *"고객에게 적정 가격을 제시할 때 객관적 근거가 부족합니다. AI 분석 리포트를 보여줄 수 있으면 신뢰도가 높아질 것 같습니다."*

**금융기관 담보 평가 담당:**
> *"담보 대출 심사 시 빠른 가치 평가가 필요한데, 감정평가 의뢰하면 2주가 걸립니다."*

## 2. 솔루션

### 핵심 기능

| 기능 | 설명 | 기술 요소 |
|------|------|-----------|
| AI 실시간 가치평가 | 수백 가지 변수 분석으로 적정 가격 산출 (5분 내) | GNN + XGBoost 앙상블 |
| 가격 변동 예측 | 1~3년 가격 변동 트렌드 및 시나리오 분석 | LSTM 시계열 + 경제지표 연동 |

### 기술 스택

| 구분 | 기술 | 선정 이유 |
|------|------|-----------|
| AI 모델 | PyTorch Geometric (GNN) + XGBoost | 부동산 공간 관계 학습, 높은 예측 정확도 |
| 시계열 예측 | LSTM + Prophet | 가격 변동 트렌드 및 계절성 분석 |
| 공간 분석 | H3 (Uber), GeoPandas, PostGIS | 위치 기반 입지 점수화 |
| 데이터 파이프라인 | Apache Airflow + Spark | 대규모 부동산 데이터 ETL |
| 백엔드 | FastAPI + PostgreSQL | 고성능 API, 공간 쿼리 (PostGIS) |
| 프론트엔드 | Next.js + Mapbox GL | 지도 기반 부동산 시각화 |
| 모바일 | React Native | 크로스플랫폼 앱 |
| 인프라 | AWS (EC2, RDS, S3, Lambda) | 확장 가능한 클라우드 인프라 |
| 리포트 생성 | Jinja2 + WeasyPrint | PDF 자동 생성 |
