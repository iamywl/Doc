## 1. 문제 인식

### 1.1 사이버보안 인력 부족 위기

#### 글로벌 사이버보안 인력 부족 현황

전 세계적으로 사이버보안 인력 부족 문제가 심화되고 있다. ISC²의 2024 Cybersecurity Workforce Study에 따르면, 글로벌 사이버보안 인력 부족 규모는 약 **400만 명**에 달하며, 이는 전년 대비 12.6% 증가한 수치이다 [1]. 한국 역시 예외가 아니어서, 한국인터넷진흥원(KISA)은 2024년 기준 국내 정보보호 전문인력 부족 규모를 약 **3만 명**으로 추산하였다 [2].

| 지표 | 수치 | 출처 |
|:---|:---|:---|
| 글로벌 사이버보안 인력 부족 | **약 400만 명** | ISC² (2024) [1] |
| 한국 정보보호 전문인력 부족 | **약 3만 명** | KISA (2024) [2] |
| 사이버 공격 피해액 (글로벌, 2025 예상) | **$10.5조 (약 1경 3,650조원)** | Cybersecurity Ventures (2024) [3] |
| 한국 사이버침해 사고 신고 건수 (2024) | **1,899건 (전년 대비 +35%)** | KISA (2024) [4] |
| AI 기반 사이버공격 증가율 (2025 전망) | **+300%** | AhnLab (2026 위협 전망) [5] |
| 사이버보안 직무 평균 연봉 (한국) | **5,800만원** | 사람인 (2024) [6] |
| 사이버보안 직무 공석 충원 소요 기간 | **평균 6개월 이상** | (ISC)² (2024) [1] |

### 1.2 기존 보안 교육의 한계

#### 이론 중심 교육의 문제

현행 대학 보안 교육의 가장 큰 문제는 **이론과 실습의 괴리**이다. 한국정보보호학회의 2024년 조사에 따르면, 정보보안 관련 학과 졸업생 중 **68%**가 "대학에서 배운 내용이 실무에 직접 도움이 되지 않았다"고 응답하였다 [8].

| 교육 방식 | 현행 비중 | 이상적 비중 | 격차 |
|:---|:---:|:---:|:---:|
| 이론 강의 | 70% | 30% | -40%p |
| 실습 프로젝트 | 15% | 40% | +25%p |
| CTF/워게임 | 5% | 20% | +15%p |
| 산업 연계 프로젝트 | 10% | 10% | 0%p |

> 출처: 한국정보보호학회 (2024) [8], 교수진·졸업생 설문 기반

#### 영어 장벽

기존 글로벌 보안 교육 플랫폼(HackTheBox, TryHackMe, PortSwigger Academy)은 모두 **영어 기반**으로 운영되며, 한국어 지원이 전무하다. 이로 인해 다음과 같은 문제가 발생한다:

| 문제 영역 | 구체적 현황 | 영향 |
|:---|:---|:---|
| 언어 장벽 | 보안 개념 + 영어 동시 학습 부담 | 학습 포기율 높음 |
| 초보자 진입장벽 | 배경지식 없이 영어 문제 해독 어려움 | 보안 전공 기피 현상 |
| 한국 특화 콘텐츠 부재 | 국내 법규(개인정보보호법 등) 학습 불가 | 실무 적용 어려움 |
| 기업 교육 활용 한계 | 영어 기반 플랫폼 직원 교육 저항 | 기업 보안 역량 저하 |

#### 게이미피케이션의 학습 효과

게이미피케이션(Gamification)은 게임의 요소(포인트, 배지, 리더보드, 레벨업 등)를 비게임 맥락에 적용하는 방법론이다. TalentLMS의 2024년 연구에 따르면, 게이미피케이션을 적용한 교육은 다음과 같은 효과를 보인다 [12]:

| 효과 지표 | 개선율 | 출처 |
|:---|:---:|:---|
| 학습 **참여도** 향상 | **+48%** | TalentLMS (2024) [12] |
| 지식 **기억 유지율** 증가 | **+40%** | TalentLMS (2024) [12] |
| 과정 **완료율** 향상 | **+34%** | Deloitte (2023) [13] |
| 학습 **동기부여** 향상 | **+83%** | TalentLMS (2024) [12] |
| 학습 시 **지루함 감소** | **-61%** | Gartner (2023) [14] |

이러한 데이터는 CyberDojo의 핵심 전략인 **"게임화 보안 교육"**이 학습 효과를 극대화할 수 있는 과학적 근거를 제공한다.

| **캐시/세션** | Redis 7 | 랭킹 정렬, 세션 관리, Pub/Sub |
| **파일 저장** | MinIO (S3 호환) | 챌린지 파일, 사용자 업로드 관리 |
| **로그/검색** | Elasticsearch | 풀이 로그 분석, 콘텐츠 검색 |
| **모니터링** | Prometheus + Grafana | 서버 메트릭, 사용자 활동 대시보드 |
| **CI/CD** | GitHub Actions + ArgoCD | 자동 빌드·배포, GitOps |
