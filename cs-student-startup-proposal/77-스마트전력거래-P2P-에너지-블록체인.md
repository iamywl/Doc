## 1. 문제 인식

### 에너지 프로슈머 시대의 거래 인프라 부재

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 태양광 발전 가구 수 | 170만 가구 (2025년) | 산업통상자원부 |
| 가정용 태양광 잉여전력 비율 | 평균 35% | 한국에너지공단 |
| 한전 역송 매입 단가 | kWh당 60원 (시장가 대비 50% 이하) | 한국전력공사 |
| 소비자 전기 요금 인상률 | 연평균 8.5% (최근 3년) | 통계청 |
| 전력 중개 거래 시장 규모 | 1.2조원 (2026년 전망) | 한국에너지경제연구원 |
| 에너지 프로슈머 확대 목표 | 2030년 500만 가구 | 에너지전환 로드맵 |
| 마이크로그리드 실증 지구 | 전국 32개소 | 산업통상자원부 |
| 블록체인 에너지 거래 파일럿 | 8개 프로젝트 | 한국전력공사 |
| 재생에너지 발전 비중 | 12.8% (2025년) | 한국전력거래소 |
| 전력 피크 관리 비용 | 연 2.3조원 | 전력거래소 |

분산 에너지 자원(DER)의 확대와 함께 소규모 태양광 발전 가구가 급증하고 있으나, 이들이 생산한 잉여전력을 효율적으로 거래할 수 있는 인프라가 부재하다. 현재는 한전에 낮은 단가로 역송하는 것이 유일한 방법으로, 프로슈머의 투자 수익성이 낮아 재생에너지 확대에 걸림돌이 되고 있다.

### 사용자 페인 포인트 분석

**태양광 발전 가구 (프로슈머):**
> *"3kW 태양광을 설치했는데, 낮에 생산한 전기를 한전에 팔면 kWh당 60원밖에 안 됩니다. 이웃에게 직접 팔 수 있으면 좋겠어요."*

> *"태양광 투자비 회수에 15년이 걸린다고 하니 주변에 추천하기 어렵습니다."*

**전력 소비 가구:**
> *"전기요금이 매년 오르는데, 옆집에서 남는 전기를 시장가보다 싸게 살 수 있다면 좋겠습니다."*

> *"아파트 단지 내에서 전력을 공유하는 시스템이 있으면 관리비를 줄일 수 있을 텐데요."*

**지자체 에너지 담당자:**
> *"마이크로그리드를 운영하고 있지만, 주민 간 전력거래 시스템이 없어 효율이 낮습니다."*

## 2. 솔루션

### 핵심 기능

| 기능 | 설명 | 기술 요소 |
|------|------|-----------|
| P2P 전력 거래 매칭 | 프로슈머-소비자 간 실시간 자동 매칭 및 가격 합의 | 스마트 계약 기반 자동 체결, 실시간 주문서 매칭 |

### 기술 스택

| 구분 | 기술 | 선정 이유 |
|------|------|-----------|
| 블록체인 | Hyperledger Fabric + Layer-2 Rollup | 허가형 블록체인으로 에너지 규제 환경 적합, 높은 처리 성능 |
| 스마트 계약 | Solidity / Chaincode (Go) | 자동화된 전력 거래 체결 및 정산 |
| 백엔드 | Go (Gin Framework) | 고성능 동시성 처리, 블록체인 노드 연동 |
| AI/ML | Python (PyTorch) | 전력 수요 예측 LSTM/Transformer 모델 |
| IoT | MQTT, InfluxDB | 스마트미터 실시간 데이터 수집 및 시계열 저장 |
| 메시징 | Apache Kafka | 대용량 실시간 전력 데이터 스트리밍 |
| 프론트엔드 | React + D3.js | 거래 대시보드 및 데이터 시각화 |
| 모바일 | Flutter | 크로스플랫폼 프로슈머/소비자 앱 |
| 인프라 | AWS EKS + Terraform | 컨테이너 오케스트레이션 및 IaC |

### 정성적 기대 효과

- **에너지 민주화**: 전력 거래의 독점 구조를 탈피하고 소비자 주도의 에너지 시장 구현
- **재생에너지 확대**: 프로슈머 수익성 향상으로 태양광 등 재생에너지 설치 촉진
- **지역 경제 활성화**: 지역 내 에너지 자급자족 체계 구축으로 에너지 비용 지역 순환
- **탄소중립 기여**: 재생에너지 활용 극대화 및 탄소 크레딧 거래를 통한 탄소 중립 목표 달성 기여
- **스마트시티 연계**: 마이크로그리드 기반 스마트시티 에너지 인프라의 핵심 구성 요소로 자리매김
- **에너지 빈곤 해소**: 저소득 가구에 저렴한 재생에너지 전력 공급 경로 마련
