### 핵심 기능 상세

- **활동 감지**: PIR 센서로 실내 동선 추적 (카메라 없이 프라이버시 보호), 일상 활동 패턴 학습. 방별 PIR 센서 배치로 거실-침실-주방-화장실 이동 패턴을 시간대별로 기록하며, 2주간의 학습 기간을 거쳐 개인별 정상 활동 프로파일을 생성한다.
- **낙상 감지**: 착용형 가속도/자이로 센서로 낙상 자동 감지, 30초 내 응급 연락. 6축 IMU(3축 가속도 + 3축 자이로) 데이터를 100Hz로 샘플링하여 TensorFlow Lite 모델이 엣지에서 실시간 추론. 낙상 감지 시 30초 카운트다운 후 자동 119 신고 및 보호자 알림.
- **이상 패턴 알림**: 평소와 다른 활동 패턴 감지 시 보호자/돌봄 매니저에게 자동 알림. LSTM 기반 시계열 모델이 평소 활동 패턴과의 편차를 실시간 모니터링하여 장시간 비활동(화장실 3시간 이상 등), 야간 잦은 기상, 외출 후 미귀가 등 이상 징후를 단계별로 알림.
- **보호자 대시보드**: 실시간 활동 현황, 건강 리포트, 응급 상황 이력 관리. 웹/앱 양방향 지원으로 일간/주간/월간 활동 리포트 제공, 복수 보호자 등록 및 권한 관리 기능.

### AI 낙상 감지 알고리즘 상세

SafeElder의 낙상 감지는 3단계 파이프라인으로 동작한다:

**1단계: 센서 데이터 수집 (Edge)**
- 6축 IMU (MPU-6050): 3축 가속도 + 3축 자이로스코프
- 샘플링 레이트: 100Hz (초당 100회)
- 데이터 윈도우: 2초 단위 슬라이딩 윈도우 (200 샘플)

**2단계: 특징 추출 및 추론 (Edge AI)**
- SMV (Signal Magnitude Vector) 계산: sqrt(ax^2 + ay^2 + az^2)
- 임계값 1차 필터링: SMV > 3.0g 시 낙상 후보 이벤트 트리거
- TensorFlow Lite 모델 추론: 1D-CNN + LSTM 하이브리드 모델
- 입력: 2초 윈도우 (200 x 6 텐서), 출력: 낙상 확률 (0~1)

**3단계: 상태 전이 판단 (Cloud)**
- 낙상 확률 > 0.85: 낙상 이벤트 확정
- 낙상 후 30초간 비활동 확인 → 응급 단계 진입
- 30초 카운트다운 중 본인 해제 가능 (기기 버튼 또는 앱)
- 해제 없으면 자동 119 신고 + 보호자 알림 + GPS 위치 전송

### 팀 역량 상세

**윤성호 (대표/PM)**: 대학 재학 중 IoT 스마트홈 스타트업에서 2년간 PM으로 근무하며 하드웨어-소프트웨어 통합 프로젝트 3건 수행. 사회적기업 컨설팅 인턴 경험을 바탕으로 실버테크 분야의 사회적 가치와 비즈니스 모델을 결합한 사업 모델 설계. 2025 소셜벤처 경진대회 대상 수상.

**김현수 (백엔드 개발)**: Go 언어 3년 경력, 이전 직장에서 실시간 센서 데이터 수집 파이프라인 구축 경험. 초당 10만 건의 MQTT 메시지 처리 아키텍처 설계 및 운영. AWS IoT Core 기반 디바이스 관리 시스템 개발.

**이재민 (IoT 개발)**: 임베디드 시스템 석사 과정에서 저전력 BLE 메쉬 네트워크 연구. ESP32 기반 센서 노드 설계 및 PCB 레이아웃 경험. Zephyr RTOS 커뮤니티 컨트리뷰터로 활동 중.

**박소영 (AI 엔지니어)**: 대학원에서 시계열 이상 탐지 연구, TensorFlow Lite 모델 최적화 전문. 엣지 디바이스(Coral TPU, ESP32-S3)에서의 모델 경량화 및 양자화 경험. 낙상 감지 관련 논문 2편 발표.
