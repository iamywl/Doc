## 1. 문제 인식

### 고객 피드백 관리의 현실

| 지표 | 수치 | 출처 |
|------|------|------|
| 체계적 피드백 수집 미실시 중소기업 비율 | 83% | 중소벤처기업부 (2025) |
| 부정 리뷰 미대응 비율 | 67% | 한국소비자원 (2025) |
| 고객 이탈 원인 중 서비스 불만족 | 52% | McKinsey Korea (2025) |
| 피드백 기반 개선 후 재구매율 증가 | +35% | Harvard Business Review (2024) |
| 멀티채널 피드백 통합 관리 기업 비율 | 12% | 가트너 코리아 (2025) |
| 고객 리뷰 확인 후 구매 결정 비율 | 89% | 대한상공회의소 (2025) |
| 부정 리뷰 1건당 잠재 매출 손실 | 약 120만원 | 한국소비자원 (2025) |

고객 피드백은 네이버 리뷰, 카카오 톡톡, 배달의민족 리뷰, 인스타그램 DM, 전화 CS 등 다양한 채널에 분산되어 있다. 중소기업은 이를 통합적으로 수집하고 분석할 인력과 도구가 부족하여 중요한 고객 인사이트를 놓치고 있다.

### 사용자 페인포인트 분석

**F&B 사업자 김사장 (카페 3개 매장 운영)**
> "네이버 리뷰, 배민 리뷰, 인스타그램 DM까지 매일 확인하려면 2시간이 넘게 걸려요. 그래도 놓치는 리뷰가 많아서 고객이 불만을 갖고 있는지도 모르고 지나가는 경우가 많습니다."

**이커머스 운영자 박대표 (의류 쇼핑몰)**
> "고객 불만이 반복되는 패턴이 있는 건 알지만, 수백 개의 리뷰에서 공통점을 찾아내기가 너무 어렵습니다. 엑셀로 정리해봤지만 한계가 있어요."

**서비스업 이매니저 (피트니스 센터 체인)**
> "회원들이 퇴회할 때 이유를 제대로 파악하지 못해요. 나중에 SNS에서 불만 글을 발견하면 이미 늦은 상태입니다."

### 핵심 기능

- **멀티채널 피드백 수집**: 네이버, 카카오, 배민, SNS, CS 전화 등 10개+ 채널 통합 수집. API 연동이 가능한 채널은 실시간 수집, 불가능한 채널은 스마트 크롤링으로 15분 간격 수집. 음성 CS의 경우 STT(Speech-to-Text) 변환 후 텍스트 분석을 수행한다.

- **AI 감성 분석**: KoBERT 기반 한국어 특화 감성 분석 (긍정/부정/중립 + 감정 세분화). 한국어 리뷰 100만건으로 파인튜닝하여 93.5%의 분류 정확도를 달성했다. 단순 긍정/부정을 넘어 기쁨, 분노, 실망, 감동, 불안, 놀라움, 혐오, 기대 등 8가지 감정으로 세분화한다.

- **이슈 자동 분류**: 토픽 모델링(LDA + BERTopic)으로 피드백을 카테고리별 자동 분류 및 우선순위 배정. "배송", "품질", "고객 서비스", "가격", "포장" 등 업종별 맞춤 카테고리를 자동 생성하고, 빈도와 감성 점수를 결합하여 우선순위를 산정한다.

- **개선 제안 리포트**: 수집된 데이터 기반 구체적인 서비스 개선 액션 아이템 자동 생성. GPT-4 기반으로 수집된 부정 피드백 패턴을 분석하고, 실행 가능한 개선 제안을 우선순위와 함께 제시한다.

### AI 알고리즘 상세

**1. 감성 분석 모델 (KoBERT Fine-tuned)**

한국어 리뷰 데이터에 최적화된 감성 분석 모델을 구축했다. 네이버 쇼핑 리뷰, 배달의민족 리뷰, 카카오 평가 등 총 100만건의 한국어 리뷰 데이터를 수집하고, 전문 라벨러 5인이 감성 라벨링을 수행했다.

- 3단계 분류 (긍정/부정/중립): 정확도 93.5%
- 8종 감정 분류: 정확도 87.2%
- 한국어 축약어/신조어 처리: 자체 사전 12만 단어 구축
- 처리 속도: 초당 200건 분석 가능

**2. 이슈 자동 분류 (BERTopic + LDA)**

BERTopic과 LDA를 결합한 하이브리드 토픽 모델링을 통해 피드백을 자동 분류한다. 업종별로 사전 학습된 토픽 모델을 제공하며, 사용자 데이터가 축적되면 자동으로 맞춤 토픽을 재학습한다.

3. **AI 기술 대중화**: AI 감성 분석 기술이 성숙하여 중소기업도 활용 가능한 수준에 도달
4. **고객 경험 중심 경영**: CX 중심 경영 패러다임 확산으로 피드백 분석 수요 급증
