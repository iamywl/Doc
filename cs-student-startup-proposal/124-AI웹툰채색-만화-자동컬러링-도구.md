## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| 🎨 자동 채색 | 흑백 선화를 업로드하면 AI가 자동으로 컬러링 |

### 4.3 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 채색 모델 | ControlNet + LoRA | 선화 기반 조건부 채색 생성 |
| 세그멘테이션 | SAM (Segment Anything) | 캐릭터/배경/영역 자동 분리 |
| 캐릭터 인식 | Re-ID 모델 (ResNet 변형) | 같은 캐릭터 식별, 색상 일관성 |
| 스타일 학습 | Few-shot LoRA Fine-tuning | 작가별 채색 스타일 학습 (10장 내외) |
| 이미지 처리 | OpenCV, Pillow | 전처리/후처리, 레이어 분리 |
| GPU 추론 | NVIDIA A100, TensorRT | 고속 AI 추론 |
| 백엔드 | FastAPI, Celery, Redis | API, 배치 처리 큐 |
| 프론트엔드 | React, Fabric.js | 웹 에디터, 캔버스 편집 도구 |
| 플러그인 | CEP (Adobe), Clip Studio SDK | Photoshop/CSP 플러그인 |
| 인프라 | AWS (EC2 GPU, S3, CloudFront) | GPU 추론, 파일 저장/CDN |
