## 3. 문제 인식

### 3-1. 시장 현황 및 통계

| 통계 항목 | 수치 | 출처 |
|-----------|------|------|
| 미세먼지 '나쁨' 이상 일수 (서울, 2024) | 연 52일 | 한국환경공단 |
| 대기오염 조기사망자 (한국, 연) | 약 17,800명 | 세계보건기구 (WHO) |

## 4. 솔루션

### 4-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| 초소형 PM 센서 | PM2.5/PM10 레이저 산란 방식 개인 휴대 디바이스 (20g 이하) |
| 실시간 노출량 측정 | 현재 위치의 실시간 미세먼지 농도 + GPS 기반 이동 경로별 노출 기록 |
| 누적 노출량 분석 | 일간/주간/월간 개인 미세먼지 누적 노출량 추적 및 시각화 |
| AI 건강 위험 예측 | 누적 노출량 + 개인 건강 프로필 기반 건강 위험도 AI 예측 |
| 최적 경로 추천 | 미세먼지 농도가 낮은 이동 경로 추천 (산책, 통근) |
| 실내외 비교 분석 | 실내(가정, 사무실)와 실외 공기질 비교 및 환기 시점 추천 |
| 커뮤니티 공기질 맵 | 사용자 데이터 집합으로 초고해상도 공기질 지도 구축 |
| 건강 연동 | Apple Health / Samsung Health 연동 호흡량 기반 실노출량 보정 |

### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| PM 센서 | Plantower PMS7003 + 자체 보정 | 미세먼지 농도 측정 |

| 모바일 앱 | Flutter | iOS/Android 크로스 플랫폼 |
| 백엔드 | FastAPI (Python) | API 서버, 데이터 수집 |

| 데이터베이스 | TimescaleDB + PostGIS | 시계열 + 공간 데이터 |
| 지도 | Mapbox GL | 공기질 맵 시각화 |
| 인프라 | AWS IoT Core + S3 | IoT 데이터 수집·저장 |

## 9. 사회적 가치

### 9-1. 개인 건강 보호

미세먼지 민감군 1,420만 명의 개인 맞춤 건강 보호를 지원합니다. 실제 노출량 기반 행동 가이드로 호흡기 질환 예방에 기여합니다.

### 9-2. 환경 데이터 민주화

개인 사용자 크라우드소싱으로 기존 640개 측정소로는 불가능한 초고해상도 공기질 맵을 구축하여 환경 연구와 정책에 기여합니다.

### 9-3. 노동자 건강권 보장

건설, 배달 등 야외 근무자의 미세먼지 노출 기록을 확보하여 직업병 예방과 건강권 보장에 기여합니다.

### 9-4. 환경 인식 제고

개인 노출량 데이터를 통해 대기오염의 실체를 체감하게 하고, 환경 보호 의식을 높입니다.
