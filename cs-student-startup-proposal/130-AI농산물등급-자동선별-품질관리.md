## 4. 솔루션

### 4-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 외관 등급 분류 | 고해상도 카메라 + 딥러닝으로 크기, 색상, 형태, 흠집 자동 판별 |
| 내부 품질 분석 | 근적외선(NIR) 센서로 당도, 산도, 내부 결함 비파괴 측정 |

| 등급별 자동 분류 | 특/상/중/하 등급별 자동 박스 분배 |
| 품질 이력 추적 | 개별 농산물 QR 코드 부여 및 품질 데이터 블록체인 저장 |
| 예측 수확 관리 | 생육 데이터 기반 최적 수확 시점 및 예상 등급 예측 |
| 유통 최적화 | 등급·수량 데이터 기반 출하 계획 및 가격 예측 |
| 모바일 선별 키트 | 소형 카메라·센서 + 스마트폰 앱 기반 간이 선별 도구 |

### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 비전 AI | EfficientNet V2 + YOLOv8 | 외관 등급 분류, 결함 검출 |
| 스펙트럼 분석 | PLS Regression + 1D-CNN | NIR 스펙트럼 기반 내부품질 예측 |
| 엣지 추론 | NVIDIA Jetson AGX Orin | 실시간 고속 AI 추론 |
| 제어 시스템 | PLC + Modbus | 컨베이어·분류기 자동 제어 |
| 백엔드 | Django REST Framework | 품질 데이터 API |
| 프론트엔드 | Vue.js 3 + Chart.js | 관리 대시보드 |
| 이력추적 | Hyperledger Fabric | 품질 이력 블록체인 |
| 모바일 | React Native | 농가용 간이 선별 앱 |
| 클라우드 | NCloud | 데이터 저장·분석 |
| 데이터베이스 | PostgreSQL + InfluxDB | 관계형 + 시계열 데이터 |

## 9. 사회적 가치

### 9-1. 농가 소득 증대

AI 정밀 선별로 농산물의 적정 등급 판정이 이루어지면, 저평가되던 농산물의 가치를 제대로 인정받아 농가 소득이 평균 15~20% 증가할 수 있습니다.

### 9-2. 농촌 인력난 해소

고령화와 인구 감소로 심화되는 농촌 인력난을 AI 자동화로 완화하여 지속 가능한 농업 환경을 구축합니다.

### 9-3. 식품 안전성 향상

내부 결함·농약 잔류 등을 비파괴 분석으로 사전 검출하여 소비자에게 안전한 농산물을 공급합니다.

### 9-4. 식품 폐기 감소

정확한 품질 분류로 불필요한 폐기를 줄이고, 등급에 맞는 적절한 유통 채널로 연결하여 식품 손실을 최소화합니다.
