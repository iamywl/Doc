## 1. 문제 인식

### 스킨케어 시장 문제 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 화장품 구매 후 불만족률 | 45% | 대한화장품학회 (2025) |
| 성분 부작용 경험률 | 32% | 식품의약품안전처 (2025) |
| 스킨케어 월 평균 지출 | 7.2만원 | 오픈서베이 (2025) |
| 성분 확인 후 구매 비율 | 67% | 화해 트렌드 리포트 (2025) |
| 국내 화장품 시장 규모 | 15.8조원 | 한국보건산업진흥원 (2025) |
| 피부과 방문 경험률 (20대) | 58.3% | 대한피부과학회 (2025) |
| 화장품 교환/반품률 | 18.7% | 소비자원 (2025) |
| 스킨케어 정보 탐색 시간 | 월 평균 4.2시간 | 오픈서베이 (2025) |

소비자들은 수많은 화장품 중 자신의 피부에 맞는 제품을 선택하기 어려워한다. 피부 타입, 현재 상태, 성분 호환성을 종합적으로 고려한 과학적 추천이 부재하며, 구매 후 부작용으로 인한 경제적/심리적 손실이 크다.

### 스킨케어 소비자 Pain Point 분석

| Pain Point | 비율 | 상세 설명 |
|------------|------|-----------|
| 내 피부 타입을 정확히 모르겠다 | 62% | 복합성/민감성 등 자가 판단 정확도 낮음 |
| 어떤 성분이 나에게 맞는지 모르겠다 | 57% | 12,000종 이상의 화장품 성분 이해 불가 |
| 리뷰만 보고 샀는데 나에겐 안 맞았다 | 45% | 타인 리뷰 기반 구매의 한계 |
| 제품을 바꿔도 피부가 나아지지 않는다 | 38% | 루틴 전체의 시너지/상충 미고려 |
| 피부과 가기엔 부담되지만 관리는 하고 싶다 | 53% | 전문 상담의 비용/시간 장벽 |
| 계절/환경 변화에 대응하기 어렵다 | 41% | 고정된 루틴의 한계 |

### 사용자 목소리

> "화장품을 사면 항상 후회해요. 리뷰가 좋아서 샀는데 제 피부엔 트러블이 나더라고요." - 25세 대학생

> "성분표를 읽어봐도 어떤 게 좋은 건지, 서로 섞어 써도 되는 건지 모르겠어요." - 29세 직장인

> "피부과 한 번 가면 5만원인데, 매달 가기는 부담스러워요. 앱으로 관리할 수 있으면 좋겠어요." - 32세 주부

### 기존 솔루션의 한계

| 기존 솔루션 | 한계점 |
|-------------|--------|
| 화해 (성분 앱) | 성분 정보만 제공, 피부 상태 분석/맞춤 추천 없음 |
| 루미니 (피부 측정기) | 별도 하드웨어 필요 (20만원+), 일상적 사용 어려움 |
| 뷰티 유튜버/인플루언서 | 주관적 리뷰, 개인 피부 상태 미반영 |
| 피부과 상담 | 비용 부담 (회당 5만원+), 예약 불편, 빈번한 방문 어려움 |
| 브랜드 자체 진단 | 자사 제품 추천 편향, 객관성 부족 |

### 핵심 기능 상세

- **셀피 피부 진단**: 얼굴 사진으로 모공, 주름, 색소 침착, 수분도, 유분도, 민감도, 탄력, 피부결 등 8가지 항목 분석. EfficientNet-B4 모델에 한국인 피부 데이터 50만 장을 학습시켜 피부과 전문의 수준의 진단 정확도 달성 (92.3%). 조명 보정 알고리즘으로 실내/실외 다양한 환경에서도 일관된 분석 결과 제공.

- **제품 성분 분석**: 사용 중인 제품 성분 분석, 자극 성분 경고, 성분 간 궁합 체크. 12,000종 화장품 성분 데이터베이스를 기반으로 EWG 등급, 자극 지수, 성분 간 상호작용(시너지/상충)을 분석. 바코드 스캔 또는 제품명 검색으로 즉시 성분 분석 결과 확인.

- **맞춤 루틴 추천**: 피부 상태 + 계절 + 생활 패턴 기반 아침/저녁 스킨케어 루틴 추천. Collaborative Filtering(유사 피부 타입 사용자 데이터)과 Content-Based Filtering(성분 호환성)을 결합한 하이브리드 추천 모델. 날씨 API 연동으로 자외선 지수, 습도에 따른 동적 루틴 조정.

- **변화 트래킹**: 주간 셀피로 피부 상태 변화 추적, 제품 효과 데이터 기반 검증. 동일 조건(조명, 각도)에서의 셀피 촬영 가이드 제공, 8가지 피부 지표의 시계열 변화를 그래프로 시각화. "이 제품 사용 후 모공 점수가 12% 개선되었습니다" 같은 데이터 기반 피드백.

3. **데이터 기반 효과 검증**: 주간 트래킹으로 "이 제품이 내 피부에 실제로 효과가 있는가?"를 데이터로 증명.
4. **한국인 피부 특화**: 50만 장의 한국인 피부 데이터로 학습, 서양인 중심 글로벌 모델 대비 한국인 피부 특성 반영.
