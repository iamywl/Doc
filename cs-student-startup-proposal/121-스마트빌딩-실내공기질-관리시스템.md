## 3. 문제 인식

### 3.1 실내 공기질 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 현대인 실내 체류 시간 비율 | 약 87% | 환경부 생활환경 보고서 (2025) |
| 실내 공기 오염도 vs 실외 | 실외 대비 2~5배 오염 | WHO (2024) |
| 실내 공기질 기준 위반 건물 비율 | 32.4% | 환경부 다중이용시설 점검 (2025) |
| 실내 CO2 농도 1,000ppm 초과 학교 | 전체 학교의 47.2% | 교육부 학교 환경 조사 (2025) |
| 실내 공기질 관련 건강 피해 (연간) | 약 4만 3천 명 (호흡기 질환) | 질병관리청 (2025) |
| 빌딩증후군(SBS) 경험 직장인 비율 | 68.7% | 대한산업의학회 (2025) |
| HVAC 에너지 비용 (빌딩 전체의) | 약 45~55% | 에너지관리공단 (2025) |
| 실내 공기질 관리 의무 대상 시설 | 약 12만 4천 개소 | 환경부 (2025) |

### 3.2 이해관계자 인터뷰

> "회의실에서 1시간 회의하면 머리가 아프고 졸립니다. CO2 농도가 2,000ppm 넘는다고 하더라고요."
> — 판교 IT 기업 직원 한OO (31세)

> "학교 환기를 제대로 안 하면 학생들 집중력이 떨어져요. 미세먼지 나쁜 날은 창문도 못 열고..."
> — 서울 소재 초등학교 교사 이OO (39세)

> "병원 공기질 관리가 감염 예방에 핵심인데, 센서 장비가 비싸고 관리가 어렵습니다."
> — 종합병원 시설팀장 최OO (52세)

> "HVAC를 효율적으로 돌리고 싶은데, 공기질 데이터 없이 그냥 스케줄대로만 가동하고 있어요."
> — 대형 오피스빌딩 FM 담당자 박OO (44세)

| 구분 | 기술 | 용도 |
|------|------|------|

| HVAC 연동 | BACnet, Modbus, KNX | 빌딩 자동화 시스템 연동 |
| 백엔드 | FastAPI, TimescaleDB | 시계열 데이터 저장, API |
| 프론트엔드 | React, D3.js, Flutter | 대시보드, 모바일 앱 |
| 인프라 | AWS IoT Core, ECS | IoT 디바이스 관리, 서비스 배포 |
| 모니터링 | Grafana, Prometheus | 센서 상태, 시스템 모니터링 |
