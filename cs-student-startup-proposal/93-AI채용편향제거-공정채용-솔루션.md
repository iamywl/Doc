## 4. 솔루션

### 4.1 핵심 기능

#### 기능 1: AI 이력서 블라인드 분석
- 이력서/자기소개서에서 편향 유발 요소 자동 감지 및 마스킹
- 학교명, 성별, 나이, 사진, 지역 등 비역량 정보 제거
- 역량 중심 구조화 점수 산출 (기술/경험/성과 기반)
- 편향 없는 1차 서류 스크리닝 자동화

#### 기능 2: 면접 편향 실시간 감지
- 면접 중 부적절한 질문(결혼/출산/종교 등) 실시간 경고
- 면접관별 평가 패턴 분석 (특정 유형 편향 탐지)
- 구조화 면접 질문 템플릿 제공
- 면접 평가 일관성 분석 (Inter-rater reliability)

#### 기능 3: 채용 공정성 감사(Audit)
- 채용 단계별 성별/연령/학력 통과율 통계적 검정
- 차별적 영향(Disparate Impact) 자동 분석
- 4/5 규칙 등 공정성 메트릭 자동 산출
- 규제 대응 감사 보고서 자동 생성

#### 기능 4: 다양성 대시보드
- 조직 전체 다양성 현황 시각화
- 채용 파이프라인별 다양성 추이
- 벤치마크 (동종 업계 대비)
- ESG 리포트용 다양성/포용성 지표

| Frontend | React + Recharts | 대시보드, 시각화 |
| Database | PostgreSQL + Redis | 채용 데이터 + 캐싱 |
| Integration | REST API | 주요 ATS 연동 (그리팅, 잡플래닛 등) |
| Security | SOC2 Type II | 기업 데이터 보안 인증 |
