## 1. 문제 인식

### 건물 에너지 비효율의 사회적 비용

| 지표 | 수치 | 출처 |
|------|------|------|
| 건물 부문 탄소 배출 비중 | 전체의 36% | 국제에너지기구(IEA) |
| 국내 건물 에너지 사용량 | 전체 에너지의 25% | 에너지경제연구원 |
| 건물 에너지 비효율 낭비 비율 | 평균 30% | 한국에너지공단 |
| 연간 건물 에너지 비용 (상업) | 60조원 | 산업통상자원부 |
| BEMS 도입 의무화 대상 건물 | 3,000㎡ 이상 공공건물 | 녹색건축물 조성지원법 |
| BEMS 실제 도입률 | 23% (의무 대상 중) | 한국에너지공단 |
| 제로에너지건축 의무화 시행 | 2025년~ (공공 1,000㎡ 이상) | 국토교통부 |
| 에너지 절감 잠재량 (기존 건물) | 연 12조원 | 건설기술연구원 |
| 탄소중립 건물 전환 필요 수 | 250만동 | 환경부 |
| 건물 에너지 시뮬레이션 수요 | 연 15% 성장 | 한국건설기술연구원 |

2050 탄소중립을 위해 건물 부문의 에너지 효율 개선은 필수적이다. 하지만 기존 건물에너지관리시스템(BEMS)은 단순 모니터링에 그치는 경우가 많고, "어떻게 하면 에너지를 절감할 수 있는지"에 대한 시뮬레이션 기능이 부재하다. 건물 에너지 시뮬레이션 소프트웨어(EnergyPlus 등)는 전문 엔지니어만 사용 가능한 수준으로 복잡하다.

### 사용자 페인 포인트 분석

**빌딩 관리자:**
> *"BEMS를 설치했지만, 에너지 사용량 그래프만 보여줍니다. 어떤 설비를 어떻게 조정하면 얼마나 절감되는지 알려주지 않습니다."*

> *"냉난방 스케줄을 바꾸면 에너지가 얼마나 줄어드는지 사전에 알 수 있으면 좋겠습니다."*

**건설사 설계팀:**
> *"설계 단계에서 에너지 성능을 시뮬레이션하려면 EnergyPlus 전문가가 필요합니다. 비용도 비싸고 시간도 오래 걸립니다."*

**공공기관 시설 담당:**
> *"제로에너지건축 의무화에 대비해야 하는데, 기존 건물의 에너지 개선 효과를 사전에 검증할 방법이 없습니다."*

**ESCO 기업:**
> *"에너지절감 보증 사업을 하려면 정확한 절감량 예측이 필수입니다. 현재는 경험에 의존해서 리스크가 큽니다."*

## 2. 솔루션

### 핵심 기능

| 기능 | 설명 | 기술 요소 |
|------|------|-----------|
| BIM 기반 디지털 트윈 | 건물 BIM 파일(IFC)을 자동으로 디지털 트윈 3D 모델로 변환 | IFC 파서, Three.js 3D 렌더링 |
| 열역학 에너지 시뮬레이션 | 건물 열관류율, HVAC, 조명 등 에너지 소비 시뮬레이션 | EnergyPlus 엔진 + 클라우드 병렬 처리 |
| What-If 시나리오 분석 | 설비 교체, 운영 변경 시 에너지 절감 효과 사전 예측 | 파라메트릭 시뮬레이션 |
| AI 운영 최적화 | 실시간 데이터 기반 HVAC, 조명 최적 운영 전략 도출 | RL(강화학습) 최적화 에이전트 |
| IoT 실시간 연동 | 기존 BEMS/BAS 데이터를 디지털 트윈에 실시간 반영 | BACnet/Modbus 연동, MQTT |
| 탄소 배출량 산출 | 에너지 사용에 따른 탄소 배출량 자동 계산 및 리포트 | 배출 계수 DB, GHG 프로토콜 |
| 제로에너지 인증 지원 | 제로에너지건축(ZEB) 인증 시뮬레이션 및 서류 자동 생성 | ZEB 인증 기준 DB |

### 기술 스택

|------|------|-----------|

| 프론트엔드 | React + D3.js + Three.js | 3D 뷰어 및 대시보드 |

### 정성적 기대 효과

- **탄소중립 기여**: 건물 에너지 효율 개선으로 국가 탄소 배출 감축 목표 달성 지원
- **건물 관리 패러다임 전환**: 사후 대응에서 사전 예측/시뮬레이션 기반 관리로 전환
- **제로에너지건축 확산**: 설계 및 리모델링 단계에서 에너지 성능 사전 검증 가능
- **건설/에너지 산업 디지털 전환**: 건설 산업의 디지털 트윈 활용 확대 촉진
- **에너지 비용 절감**: 건물 관리비 절감을 통한 입주자/기업 경제적 부담 완화
- **기술 표준화 기여**: 국내 건물 에너지 시뮬레이션 표준 가이드라인 수립에 기여
