## 4. 솔루션

### 4.1 핵심 기능

#### 기능 1: AI 비전 기반 자동 분류
- 컨베이어 벨트 또는 수거함 상단 카메라로 폐기물 실시간 촬영
- CNN + Vision Transformer로 50종 이상 산업폐기물 분류
- 근적외선(NIR) 분광 카메라로 소재 성분 분석
- 분류 정확도 96% 이상, 초당 3건 이상 처리

#### 기능 2: IoT 계량 및 자동 기록
- 로드셀 센서로 폐기물 중량 자동 측정
- 환경부 올바로 시스템 자동 연동 (데이터 입력 자동화)
- RFID/QR 코드로 수거 용기별 추적
- 위치 정보 + 시간 + 중량 + 종류 자동 기록

#### 기능 3: 블록체인 이력 추적
- 발생 -> 수거 -> 운반 -> 처리 전 과정 블록체인 기록
- 배출자-운반자-처리자 간 투명한 이력 관리
- 불법 투기 시 즉시 경보 (경로 이탈 감지)
- ESG 리포트용 폐기물 처리 이력 증빙

#### 기능 4: 관리 대시보드
- 사업장별 폐기물 발생량/종류 현황
- 재활용률, 처리 비용 분석
- 규제 준수 체크리스트 및 자동 리포트
- 최적 처리 업체 매칭 추천
