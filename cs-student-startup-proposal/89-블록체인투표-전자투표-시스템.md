## 4. 솔루션

### 4.1 핵심 기능

#### 기능 1: 블록체인 기반 투표 기록
- 모든 투표가 분산 원장에 기록되어 위변조 불가
- Hyperledger Besu 기반 허가형 블록체인 (프라이빗)
- 합의 알고리즘: IBFT 2.0 (높은 처리량, 최종성 보장)
- 투표 결과 해시값 공개로 누구나 검증 가능

#### 기능 2: 영지식 증명(ZKP) 비밀투표
- zk-SNARK로 투표 내용을 공개하지 않고 유효성만 증명
- 투표자 신원과 투표 내용의 완전한 분리
- 이중 투표 방지 + 비밀투표 보장 동시 달성
- 개표 시에만 동형암호 복호화로 결과 산출

#### 기능 3: DID 기반 신원 인증
- 분산 신원인증(DID)으로 안전한 본인확인
- 공동인증서, PASS 앱, 생체인증 연동
- 한 번 인증으로 투표 + 검증까지 원스톱
- 신원 정보와 투표 내용 암호학적 분리

#### 기능 4: 투표 관리 대시보드
- 선거 생성, 후보/안건 등록, 유권자 관리
- 실시간 투표율 모니터링 (내용은 비공개)
- 자동 개표 및 결과 공시
- 감사 로그 및 블록체인 탐색기 제공

| Backend | Rust + Actix | 고성능 암호 연산 API |
| Backend | Node.js + Express | 관리 API, WebSocket |
| Frontend | React Native | 투표 앱 (크로스플랫폼) |
| Storage | IPFS | 투표 증빙 분산 저장 |
| Infra | AWS GovCloud / G-Cloud | 공공 클라우드 인프라 |
