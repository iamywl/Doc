## 1. 문제 인식

### 학력 위변조 및 검증 비효율

| 지표 | 수치 | 출처 |
|------|------|------|
| 연간 학력 위조 적발 건수 | 2,300건+ | 한국대학교육협의회 2025 |
| 기업 학력 검증 평균 소요 시간 | 5일 | 한국경영자총협회 2025 |
| 해외 학위 검증 소요 시간 | 2~4주 | 한국학술진흥재단 2025 |
| 검증 1건당 평균 비용 | 15,000원 | 한국경영자총협회 2025 |
| 채용 후 학력 허위 발각 비율 | 3.2% | 잡코리아 2025 |
| 블록체인 인증 도입 희망 대학 | 67.8% | 교육부 설문 2025 |
| 국제 학위 교류 건수 (연간) | 185,000건 | 교육부 2025 |
| 학력 검증 시장 연간 비용 | 1,200억원 | 한국경영자총협회 2025 |

현행 학위 인증 시스템은 종이 증명서에 의존하며, 위변조에 취약하고 검증에 수일이 소요된다. 특히 해외 학위 검증은 2~4주가 걸리며, 국제 교류가 증가하면서 이 문제는 더 심각해지고 있다. 블록체인 기반 DID(분산 신원 인증)는 이 문제를 근본적으로 해결할 수 있는 기술로 주목받고 있다.

### 이해관계자별 Pain Point

**대학교 (발급 기관)**
- "매년 졸업 시즌마다 증명서 발급 업무가 폭주합니다. 직원들이 야근해야 해요." (대학 학사 담당자)
- "해외 대학과 학점 교류 시 서류 검증에 최소 2주 이상 걸립니다." (국제교류 담당자)
- "종이 증명서 위조 사례가 매년 발생하는데, 사후 관리가 어렵습니다." (대학 총무처)

**기업 (검증 기관)**
- "1,000명 이상 지원하면 학력 검증에만 2주가 걸립니다." (대기업 채용 담당자)
- "해외 대학 학위는 검증이 거의 불가능에 가깝습니다." (글로벌 기업 HR)
- "학력 위조 발견 후 법적 대응에 추가 비용이 발생합니다." (중견기업 인사팀)

**개인 (학위 소유자)**
- "해외 취업 지원할 때마다 공증받고 번역하느라 비용이 많이 들어요." (해외 취업 준비생)
- "졸업증명서를 매번 발급받으러 학교에 가야 해서 번거로워요." (직장인)

### 핵심 기능

1. **DID 기반 학위 발급**: 대학이 블록체인에 학위 정보를 기록하고, W3C Verifiable Credentials 표준으로 디지털 학위를 발급한다. 졸업 시 학사 시스템(SIS)에서 자동으로 블록체인에 기록되며, 발급된 자격증명은 위변조가 불가능하다. 각 학위에는 고유한 DID가 부여되어 전 세계 어디서든 검증 가능하다.

2. **개인 DID 월렛**: 졸업생이 스마트폰 앱으로 학위, 성적증명서, 자격증, 수료증 등을 안전하게 관리한다. 선택적 공개(Selective Disclosure)를 지원하여 "석사 이상 학위 보유" 같은 조건만 공개하고 구체적 정보는 비공개로 유지할 수 있다.

3. **즉시 검증 API**: 기업이 지원자의 학위를 1초 내에 온라인으로 검증할 수 있다. REST API 또는 QR코드 스캔으로 검증 가능하며, 대량 검증(배치 처리)도 지원한다. 기존 5일 → 1초로 검증 시간을 단축한다.

4. **교육기관 연동**: 학사 시스템(SIS) 자동 연동으로 졸업 시 자동 발급되며, 교육부 NEIS와 호환된다. REST API 기반 연동 모듈을 제공하여 기존 학사 시스템 변경 없이 도입 가능하다.

5. **글로벌 호환**: W3C DID 표준 기반으로 해외 대학/기업과 상호 검증이 가능하다. MIT Digital Credentials Consortium, European Blockchain Services Infrastructure(EBSI) 등 국제 프로젝트와 호환을 추진한다.

6. **자격증 통합 관리**: 학위뿐 아니라 국가공인 자격증, 민간 수료증, 온라인 강좌 인증서 등을 통합 관리한다. 한국산업인력공단, 한국직업능력연구원 등과 연동하여 원스톱 자격증명 서비스를 제공한다.

| 항목 | CertiChain | ICONLOOP 브라이트 | 라온시큐어 | 학력인증 (교육부) |

| 글로벌 호환 | O | △ | X | X |
| 학사 시스템 연동 | O | X | X | O |
| 자격증 통합 관리 | O | △ | O | X |

### 경쟁 우위 상세

**ICONLOOP 브라이트 대비**: 범용 DID 플랫폼으로 학위 인증 특화 기능이 부족하다. CertiChain은 학사 시스템 연동, 교육부 NEIS 호환, 글로벌 학위 상호 인증 등 교육 도메인에 최적화되어 있다.

**라온시큐어 대비**: 신원 인증 중심으로 학위/자격증 관리 기능이 없다. CertiChain은 교육 자격증명에 특화된 Verifiable Credentials 설계를 제공한다.

**교육부 학력인증 대비**: 중앙화된 시스템으로 해외 검증이 불가능하고, API 제공이 제한적이다. CertiChain은 분산화된 블록체인 기반으로 글로벌 호환성과 실시간 API를 제공한다.
