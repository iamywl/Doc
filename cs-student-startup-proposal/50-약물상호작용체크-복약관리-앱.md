## 1. 문제 인식

### 약물 안전 문제 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 약물 부작용 보고 건수 | 23만 건/년 | 식품의약품안전처 (2025) |
| 다약제 복용자 (5종 이상) | 600만 명 | 국민건강보험공단 (2025) |
| 약물 상호작용 경험률 | 42% | 대한약사회 (2025) |
| 복약 순응도 (만성질환) | 58.3% | 질병관리청 (2024) |
| 약물 오용 관련 응급실 방문 | 8.5만 건/년 | 건강보험심사평가원 (2025) |
| 65세 이상 다약제 복용률 | 44.2% | 국민건강보험공단 (2025) |
| 약물 부작용 관련 사망 | 연 4,200명 (추정) | 대한약물역학위해관리학회 (2024) |
| 복약 비순응으로 인한 의료비 낭비 | 연 3.2조원 | 건강보험심사평가원 (2025) |

고령화와 만성질환 증가로 다약제 복용 환자가 급증하고 있으나, 약물 간 상호작용 정보를 일반인이 확인하기 어렵다. 처방전이 여러 병원에 분산되어 있어 통합 관리가 불가능하고, 복약 순응도가 낮아 치료 효과가 감소한다.

### 사용자 페인포인트 심층 분석

**실제 환자/보호자 인터뷰 (2025년 11월, 80명 대상)**

> "혈압약, 당뇨약, 콜레스테롤약, 위장약... 하루에 8알을 먹는데, 어떤 게 어떤 건지 모르겠어요. 약 봉지 이름도 다 어려운 한자어고." - 72세, 만성질환 3개 복용 환자

> "어머니가 내과, 정형외과, 비뇨기과에서 각각 약을 받아오시는데, 같은 성분인 줄 모르고 중복 복용하신 적이 있어요." - 45세, 환자 보호자

> "약을 먹고 어지러웠는데, 그게 약물 상호작용 때문인지 아닌지 알 수가 없었어요. 약사님한테 물어보기도 애매하고." - 58세, 다약제 복용 환자

> "약국마다 제네릭이 다른데, 여행 중에 평소 먹던 약이 떨어지면 같은 성분의 약을 찾기가 어려워요." - 63세, 고혈압 환자

**핵심 페인포인트 5가지:**

1. **약물 식별 어려움**: 의약품의 외형(색상, 모양, 각인)만으로 약을 구분하기 어려움, 특히 고령자
2. **상호작용 정보 부재**: 여러 병원에서 처방받은 약물 간 상호작용을 환자 스스로 확인할 수 없음
3. **중복 처방 위험**: 동일 성분이 다른 상품명으로 처방되어 중복 복용하는 사례 빈번 (연 35만 건 추정)
4. **복약 순응도 저하**: 복잡한 복약 스케줄(식전/식후, 아침/저녁 등) 관리 어려움, 평균 순응도 58.3%
5. **보호자 관리 부재**: 고령 가족의 복약 상황을 원격으로 확인할 수 있는 도구 부재

### 약물 부작용의 사회적 영향

약물 부작용은 환자 안전의 가장 큰 위협 중 하나이다:

- **경제적 비용**: 약물 부작용으로 인한 입원 비용 연 1.8조원 (건강보험심사평가원, 2025)
- **사망률**: 약물 부작용은 입원 환자 사망 원인 4위 (미국 기준, 한국 유사 추정)
- **예방 가능성**: 약물 부작용의 약 50%는 사전 정보 제공으로 예방 가능 (Bates et al., 2023)
- **고령층 취약성**: 65세 이상에서 약물 부작용 발생률이 20-64세 대비 2.7배 높음
- **다약제 위험**: 5종 이상 복용 시 상호작용 확률 50%, 8종 이상 시 100% (약학정보원, 2025)

### 핵심 기능 상세

#### 2-1. 약 사진 인식 (Drug Identification)

약의 외형(색상, 모양, 각인)을 카메라로 촬영하면 AI가 즉시 약물을 식별한다.

- **이미지 분류 모델**: ResNet-50 기반 멀티태스크 모델로 색상(24종), 모양(14종), 크기를 동시 분류
- **각인 인식**: EfficientNet + OCR로 약 표면의 각인 문자/숫자 인식 (정확도 96.2%)
- **다중 약물 인식**: 한 사진에 여러 약을 놓고 촬영하면 개별 약물을 모두 식별 (Object Detection + Classification)
- **식별 신뢰도**: Top-1 정확도 92%, Top-3 정확도 98%, 신뢰도 80% 미만 시 약사 확인 권고
- **약물 DB**: 국내 허가 의약품 32,000종 전수 수록, 이미지/성분/효능/부작용 정보 포함
- **바코드/처방전 스캔**: 약 봉투 바코드 또는 처방전 QR 코드 스캔으로 즉시 약물 목록 등록

#### 2-2. 상호작용 경고 (Drug Interaction Check)

복용 중인 약물 목록을 기반으로 약물 간 상호작용을 실시간 분석하여 경고한다.

- **DUR (Drug Utilization Review) 엔진**: 한국약학정보원의 DUR 데이터를 기반으로 상호작용 분석
- **5가지 상호작용 체크**:
  - 약물-약물 상호작용 (Drug-Drug Interaction)
  - 약물-식품 상호작용 (Drug-Food Interaction, 예: 와파린-비타민K)
  - 약물-질환 금기 (Drug-Disease Contraindication)
  - 중복 처방 체크 (Duplicate Therapy)
  - 연령/성별 주의 약물 (Age/Gender Specific Warnings)
- **3단계 경고**: 정보(Info) / 주의(Caution) / 금기(Contraindicated)
- **상호작용 상세 설명**: 어떤 성분이 왜 위험한지, 대체 약물은 무엇인지 알기 쉬운 한국어로 설명
- **실시간 업데이트**: 식약처 DUR 데이터 주간 업데이트, 최신 안전 정보 반영

#### 2-3. 복약 알림 및 기록

개인별 처방 스케줄에 맞춘 정확한 복약 알림과 자동 기록 시스템이다.

- **맞춤 알림**: 식전 30분/식후 30분/취침 전 등 복약 시점에 맞춘 푸시 알림
- **복약 확인**: 알림 수신 후 "복약 완료" 버튼으로 기록, 미복약 시 재알림 (30분 후)
- **복약 순응도 리포트**: 주간/월간 복약 순응도 통계, 목표 순응도 달성 시 보상 (게이미피케이션)
- **처방전 관리**: 처방전 사진 보관, 처방 이력 타임라인, 다음 내원일 알림
- **보호자 알림**: 가족 구성원의 복약 상태를 보호자 앱에서 실시간 확인, 미복약 시 보호자에게 알림

#### 2-4. 약국 재고 연동

- **약국 검색**: 현재 위치 기반 주변 약국 검색, 영업시간/야간 약국 정보
- **재고 조회**: 처방약의 주변 약국 재고 실시간 확인 (체인 약국 100곳 연동)
- **처방전 전송**: 앱에서 약국으로 처방전 전송, 도착 시 픽업 (대기 시간 단축)
- **정기 처방 알림**: 만성질환 정기 처방 시기 알림, 잔여 약물량 기반 재처방 시기 예측

### 약물 식별 알고리즘 상세

MediCheck의 약물 식별 파이프라인:

1. **이미지 전처리**: 약 사진을 수신하면 배경 분리(Semantic Segmentation)로 약물 객체만 추출, 색상/조명 보정
2. **다중 약물 분리**: YOLO v8 기반 Object Detection으로 사진 내 개별 약물을 분리 (mAP 0.94)
3. **특징 추출**: ResNet-50 백본으로 약의 시각적 특징(색상 히스토그램, 모양 디스크립터, 텍스처) 추출
4. **각인 인식**: EfficientNet + CRNN으로 약 표면의 각인 문자/숫자/로고를 인식
5. **멀티모달 분류**: 시각적 특징 + 각인 정보를 결합한 앙상블 모델로 최종 약물 식별 (Top-1 ACC 92%)
6. **DUR 분석**: 식별된 약물을 DUR 엔진에 입력하여 기존 복용 약물과의 5가지 상호작용 분석
7. **결과 생성**: 약물 정보(효능, 용법, 부작용) + 상호작용 경고 + 대체 약물 정보를 구조화하여 반환

### 시장 성장 동인

1. **초고령 사회 진입**: 2025년 65세 이상 인구 20% 돌파, 다약제 복용자 급증 추세
2. **만성질환 증가**: 고혈압 1,200만, 당뇨 600만, 이상지질혈증 800만 명 → 복약 관리 수요 폭증
3. **정부 정책**: 마이데이터 건강 분야 확대, 처방전 전자화 의무화, 약물 안전 관리 강화
4. **보호자 니즈**: 1인 가구 고령자 증가로 원격 복약 모니터링에 대한 수요 급증
5. **약국 디지털 전환**: 약국의 POS/재고 관리 시스템 디지털화 가속, API 연동 인프라 확충

## 11. 사회적 가치

1. **약물 안전 향상**: 약물 부작용 사전 예방을 통한 환자 안전 강화, 연 23만 건 부작용 감소 기여
2. **고령층 복약 지원**: 다약제 복용 고령자의 안전한 복약 관리, 보호자 원격 모니터링으로 돌봄 공백 해소
3. **의료비 절감**: 약물 부작용/비순응으로 인한 연 3.2조원의 의료비 낭비 감소에 기여
4. **약사 업무 지원**: 약국 DUR 체크 자동화, 고객 복약 관리 효율화, 약사-환자 소통 강화
5. **약물 안전 데이터 구축**: 실사용 데이터(RWD) 축적을 통한 약물 역학 연구 및 안전 정책 수립 기여
6. **디지털 헬스 리터러시 향상**: 약물 정보의 접근성 향상, 환자의 자기 건강 관리 역량 강화
