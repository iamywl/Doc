## 4. 솔루션

### 4.1 핵심 기능

**① 블록체인 이력 추적 엔진**
- Hyperledger Fabric 기반 허가형 블록체인 네트워크
- 원산지 → 가공 → 물류 → 판매 전 과정 불변 기록
- 참여자별 역할 기반 접근 제어 (생산자/가공자/유통자/소비자)

**② IoT 자동 데이터 수집**
- 온도·습도 IoT 센서로 콜드체인 자동 모니터링
- GPS 기반 물류 위치 자동 추적
- 수동 입력 최소화로 데이터 신뢰성 확보

**③ 소비자 투명성 앱**
- QR 코드 스캔으로 전체 이력 확인 (원산지, 가공일, 유통경로, 온도 기록)
- 친환경·유기농 인증 블록체인 검증
- 식품 안전 알림 (리콜, 주의보)

**④ 기업 관리 대시보드**
- 공급망 전체 시각화 및 실시간 모니터링
- 리콜 발생 시 영향 범위 즉시 파악 (14일 → 수 시간)
- 식품 이력추적 규정 자동 준수 리포트

### 4.3 기술 스택

| 구분 | 기술 | 선정 사유 |
|------|------|-----------|
| 블록체인 | Hyperledger Fabric 2.5 | 허가형, 높은 TPS, 기업 친화적 |
| 스마트 컨트랙트 | Go (Chaincode) | Fabric 네이티브, 고성능 |
| 백엔드 API | Node.js (Express) | Fabric SDK 네이티브 지원 |
| 오프체인 DB | PostgreSQL | 대용량 부가 데이터 저장 |
| IoT 연동 | MQTT, ESP32 센서 | 경량 IoT 메시지 프로토콜 |
| 프론트엔드 | React, Next.js | 기업 대시보드, 반응형 UI |
| 모바일 | Flutter | 생산자/소비자 크로스 플랫폼 |
| QR 인증 | 자체 QR 생성 + 블록체인 검증 | 위변조 불가 QR 인증 |
| 인프라 | AWS (EKS, ECR) | 블록체인 노드 컨테이너 운영 |
| 모니터링 | Hyperledger Explorer, Grafana | 블록체인 네트워크 상태 모니터링 |
