## 2. 창업 아이템 개요

| 항목 | 내용 |
|------|------|
| 아이템명 | 폰트미 (FontMe) |

## 3. 문제 인식

### 3.1 폰트 시장 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 폰트 시장 규모 | 약 2,800억 원 | 한국콘텐츠진흥원 (2025) |
| 개인 크리에이터 수 (유튜브, 인스타 등) | 약 320만 명 | 방송통신위원회 (2025) |
| 폰트 제작 시 평균 소요 기간 (한글) | 6~12개월 | 한국타이포그래피학회 (2025) |
| 한글 폰트 1종 제작 비용 | 3,000~5,000만 원 | 산돌커뮤니케이션 (2024) |
| 개인 폰트 제작 수요 (연간 검색량) | 전년 대비 187% 증가 | 네이버 트렌드 (2025) |
| 소상공인 브랜드 폰트 수요 | 78.3%가 "원한다" 응답 | 소상공인시장진흥공단 (2025) |

| 연간 폰트 라이선스 분쟁 건수 | 약 2,300건 | 저작권위원회 (2025) |

| 기능 | 설명 |
|------|------|
| ✍️ 손글씨 스캔 | 16자만 손글씨로 쓰면 AI가 전체 한글 폰트 자동 생성 |

### 4.3 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 생성 모델 | Stable Diffusion XL (커스텀) | 글자 이미지 생성 기반 모델 |
| 스타일 추출 | CLIP + LoRA Fine-tuning | Few-shot 스타일 학습 (16자 → 스타일 벡터) |
| 벡터 변환 | Potrace, FontForge | 래스터 이미지를 벡터 글리프로 변환 |
| 폰트 엔진 | OpenType.js, FontTools | OTF/TTF 폰트 파일 생성 및 힌팅 |
| 백엔드 | FastAPI, Celery | API 서버, 비동기 폰트 생성 작업 큐 |
| GPU 인프라 | NVIDIA A100 (클라우드) | 모델 학습 및 추론 |
| 프론트엔드 | Next.js, Canvas API | 웹 에디터, 실시간 미리보기 |
| 스토리지 | AWS S3, CloudFront | 폰트 파일 저장 및 CDN 배포 |
| 데이터베이스 | PostgreSQL, MongoDB | 사용자/폰트 메타데이터, 생성 이력 |
| 결제 | Stripe, 토스페이먼츠 | 구독 결제, 폰트 마켓 정산 |

## 5. 시장 분석

### 5.1 시장 규모 (TAM/SAM/SOM)

| 구분 | 규모 | 산출 근거 |
|------|------|-----------|
| TAM (전체 시장) | 약 1조 2,000억 원 | 글로벌 폰트 시장 (아시아 한정) |
| SAM (유효 시장) | 약 2,800억 원 | 국내 폰트 시장 (라이선스 + 제작) |
| SOM (목표 시장) | 약 120억 원 | AI 개인 맞춤 폰트 생성 시장 (3년 내) |

### 5.2 경쟁사 분석

| 경쟁사 | 강점 | 약점 | 폰트미 차별점 |
|--------|------|------|---------------|
| 산돌구름 | 대형 폰트 라이브러리 | 기존 폰트 구독, 커스텀 제작 불가 | 개인 손글씨 → 폰트 자동 생성 |
| Calligraphr (해외) | 손글씨 폰트 제작 | 한글 미지원, 수동 작업 많음 | 한글 11,172자 AI 자동 완성 |
