## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| 소액 AI 포트폴리오 | 월 1만원부터 소수점 주식/ETF 분산투자, AI가 리스크 성향별 포트폴리오 자동 구성 |
| 자동 리밸런싱 | 시장 상황에 따라 AI가 주기적으로 포트폴리오 자동 조정 |
| 금융교육 모듈 | 투자 진행에 맞춘 단계별 금융 교육 콘텐츠 (게이미피케이션) |
| 투자 시뮬레이션 | 가상 자금으로 투자 연습, 전략 백테스팅 |
| 감정 관리 코칭 | 공포/탐욕 지수 기반 매매 시점 조언, 감정적 투자 방지 |
| 또래 벤치마킹 | 동일 연령대 익명 투자 성과 비교 (프라이버시 보호) |
| 목표 기반 투자 | 여행/학자금/비상금 등 목표 설정 후 자동 투자 플랜 |
| 금융사기 경보 | AI가 의심 투자 정보 감별, 사기 유형 알림 |

### 4.3 기술 스택

| 분류 | 기술 | 용도 |
|------|------|------|
| 포트폴리오 최적화 | Modern Portfolio Theory, 강화학습(PPO) | 리스크-수익 최적 배분 |
| 시장 분석 | Temporal Fusion Transformer, LSTM | 자산 가격 예측, 시장 트렌드 분석 |
| 리스크 관리 | VaR, CVaR, Monte Carlo 시뮬레이션 | 손실 확률 계산, 리스크 경고 |
| NLP | KoBERT, 감성분석 | 뉴스/SNS 시장 심리 분석, 사기 감별 |
| 모바일 | Flutter (iOS/Android) | 크로스플랫폼 투자 앱 |
| 백엔드 | Spring Boot (Java), gRPC | 금융 트랜잭션, 고신뢰성 API |
| 데이터베이스 | PostgreSQL, Redis, ClickHouse | 금융 데이터, 캐시, 분석 |
| 보안 | AES-256, TLS 1.3, FIDO2 | 금융 데이터 암호화 |
| 클라우드 | AWS (금융 전용 리전) | 금융 컴플라이언스 준수 인프라 |
