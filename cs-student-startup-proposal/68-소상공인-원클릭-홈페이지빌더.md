## 1. 문제 인식

### 소상공인 디지털 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 홈페이지 미보유 소상공인 비율 | 66% | 소상공인시장진흥공단 (2025) |
| 홈페이지 보유자 중 관리 포기 비율 | 47% | 한국인터넷진흥원 (2025) |
| 홈페이지 제작 평균 비용 | 150~500만원 | 웹 에이전시 평균 (2025) |
| 디지털 전환 의향 있으나 미실행 비율 | 71% | 중소벤처기업부 (2025) |
| 네이버 플레이스만 운영하는 소상공인 | 58% | 네이버 비즈니스 (2025) |
| 모바일 검색 후 방문하는 고객 비율 | 76% | 구글 코리아 (2025) |
| 자체 홈페이지 보유 매장의 매출 증가율 | +22% | 대한상공회의소 (2025) |

소상공인의 대다수가 자체 홈페이지의 필요성을 인식하면서도 높은 제작 비용, 기술적 어려움, 관리 부담으로 인해 포기하고 있다. 네이버 플레이스에만 의존하는 경우 브랜딩 한계와 플랫폼 종속 문제가 발생한다.

**반찬 가게 대표 박반찬 (1인 사업자)**
> "인스타그램에 사진 올리는 것도 벅찬데, 메뉴가 바뀔 때마다 홈페이지를 업데이트하는 건 불가능해요. 메뉴판 사진만 찍으면 자동으로 홈페이지에 반영되면 좋겠어요."

### 핵심 기능

- **AI 기반 자동 디자인**: 업종별 최적화 디자인 템플릿을 AI가 자동 선택 및 커스터마이징. 사업자번호만 입력하면 공공데이터 API에서 업종, 상호명, 주소를 자동으로 가져오고, 네이버 플레이스에 등록된 사진과 정보를 크롤링하여 홈페이지를 구성한다. Stable Diffusion 기반으로 로고와 배너 이미지도 자동 생성한다.

- **메뉴/상품 관리**: 사진 촬영만으로 메뉴판/상품 목록 자동 생성. Tesseract OCR 엔진으로 메뉴판 사진에서 메뉴명, 가격, 설명을 자동 인식하여 디지털 메뉴판을 생성한다. 가격 변경 시 사진만 다시 촬영하면 자동으로 업데이트된다.

- **예약 시스템**: 네이버 예약, 카카오 예약과 연동되는 통합 예약 관리. 자체 예약 시스템도 제공하여 단골 고객 관리가 가능하며, 예약 알림, 노쇼 방지, 예약 통계 등의 부가 기능을 포함한다.

- **네이버/카카오 연동**: 네이버 플레이스, 카카오맵, 인스타그램 자동 동기화. 홈페이지에서 수정한 정보(영업시간, 메뉴, 공지 등)가 연동된 플랫폼에 자동으로 반영되어 멀티 플랫폼 관리 부담을 제거한다.

### AI 기술 상세

**1. 사업자 정보 자동 수집**
사업자등록번호 API(국세청)를 통해 상호명, 업종, 사업장 주소를 자동 수집하고, 네이버 플레이스 API로 매장 사진, 영업시간, 메뉴 정보를 추가로 크롤링한다.

**2. AI 디자인 생성 (Stable Diffusion + GPT-4)**
업종 분류 결과에 따라 사전 학습된 디자인 템플릿 풀에서 최적 디자인을 선택하고, Stable Diffusion으로 업종 맞춤형 배너 이미지를 생성한다. GPT-4는 매장 소개 문구, SEO 메타 태그, 소셜 미디어 공유 텍스트를 자동 작성한다.

**3. OCR 메뉴 인식 (Tesseract + Custom Model)**
메뉴판 사진에서 텍스트를 추출하는 자체 OCR 모델을 개발했다. 한국어 메뉴판에 특화하여 세로 쓰기, 다양한 폰트, 손글씨까지 인식 가능하며 인식 정확도 95.2%를 달성했다.
