### 팀 비전

> "위험한 실험도, 비싼 장비도 필요 없는, 모든 학생이 과학 실험을 경험할 수 있는 가상 실험실을 만든다."

사이랩은 물리, 화학, 생물 등 과학 실험을 VR 환경에서 안전하게 수행할 수 있는 가상 실험 플랫폼을 개발한다. 실제 실험 장비를 조작하는 것처럼 자연스러운 인터랙션과 물리 시뮬레이션 기반의 현실적인 실험 결과를 제공하며, 실험 리포트 자동 생성까지 지원하는 것이 핵심 차별점이다.

## 1. 문제 인식 (Problem)

### 1.1 과학 실험 교육의 구조적 문제

한국의 과학 교육은 이론 중심으로 편향되어 있으며, 실제 실험 수업의 실시율은 매우 낮다. 교육부 조사에 따르면 중·고등학교 과학 실험 수업 실시율은 38%에 불과하며, 미실시 학교의 52%가 안전 문제를, 34%가 장비 부족을 이유로 꼽았다 [1].

#### 주요 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 중·고등학교 과학 실험 수업 실시율 | **38%** | 교육부, 2024 [1] |
| 실험 미실시 이유 — 안전 우려 | **52%** | 한국과학교육학회, 2024 [2] |
| 실험 미실시 이유 — 장비/예산 부족 | **34%** | 한국과학교육학회, 2024 [2] |
| 과학 실험실 사고 건수 (2023) | **127건** | 교육부 안전사고 통계, 2024 [5] |
| 학교당 연간 과학 실험 예산 | **평균 320만 원** | 한국교육개발원, 2024 [6] |
| 학생의 과학 실험 경험 만족도 | **3.2/5.0** | 한국과학창의재단, 2024 [7] |
| VR 교육 도입 후 학습 효과 향상율 | **+32%** | PwC, 2024 [8] |

### 1.2 현장 교사·학생 심층 인터뷰

과학 교사 50명과 학생 200명을 대상으로 실시한 심층 인터뷰에서 다음과 같은 핵심 인사이트가 도출되었다.

**교사 측 페인포인트:**

| 순위 | 페인포인트 | 응답 비율 | 대표 발언 |
|------|----------|----------|----------|
| 1 | 안전 사고 책임에 대한 부담 | **78%** | "화학 실험 중 사고 나면 제가 책임져야 해요" |
| 2 | 실험 기구 관리/구매 어려움 | **65%** | "시약 유효기간, 장비 교정... 관리할 게 너무 많아요" |
| 3 | 40명 학급에서 개별 지도 불가 | **71%** | "한 명이 실험하면 나머지 39명은 구경만 해요" |
| 4 | 실험 리포트 채점 부담 | **58%** | "40명 리포트를 매주 채점하기 힘들어요" |
| 5 | 실패 시 재실험 불가 | **52%** | "시약이 부족해서 한 번 실패하면 끝이에요" |

**학생 측 페인포인트:**

| 순위 | 페인포인트 | 응답 비율 | 대표 발언 |
|------|----------|----------|----------|
| 1 | 실험을 직접 할 기회가 없음 | **82%** | "조장만 하고 나머지는 구경해요" |
| 2 | 위험해서 무서움 | **61%** | "화학 실험에서 산 다루는 게 무서워요" |
| 3 | 이론과 실험이 연결이 안 됨 | **56%** | "왜 이 실험을 하는지 모르겠어요" |
| 4 | 실험 보고서 작성이 어려움 | **48%** | "뭘 써야 하는지 모르겠어요" |

### 1.3 기존 솔루션의 한계

| 기존 솔루션 | 한계점 |
|------------|--------|
| 실제 실험실 | 안전 위험, 고가 장비, 시간/공간 제약, 실패 시 재실험 어려움 |
| 실험 동영상 | 관찰만 가능, 직접 조작 경험 불가 |
| 2D 시뮬레이션 (PhET 등) | 몰입감 부족, 실험 도구 조작 경험 미제공 |
| 기존 VR 교육 (Labster 등) | 영어 전용, 한국 교육과정 미반영, 고가 |

핵심 문제는 기존 솔루션이 **"직접 손으로 실험 도구를 조작하는 체험"**을 제공하지 못한다는 점이다. VR 핸드 트래킹을 통한 자연스러운 실험 도구 조작이 사이랩의 핵심 차별점이다.

### 2.2 핵심 기능

| 기능 | 설명 |
|------|------|
| **VR 가상 실험실** | 물리/화학/생물 실험실을 3D로 재현, 실험 도구 직접 조작 |
| **물리 기반 시뮬레이션** | 실제 물리/화학 법칙에 따른 정확한 실험 결과 |
| **위험 실험 안전 수행** | 폭발, 유독 가스 등 위험 실험을 VR에서 안전하게 체험 |
| **실험 리포트 자동 생성** | 실험 과정/결과 데이터 기반 리포트 자동 작성 |
| **교사 관리 대시보드** | 학생별 실험 수행 현황, 이해도 분석, 성적 관리 |
| **멀티플레이어 협동 실험** | 2~4명이 같은 가상 실험실에서 협동 실험 수행 |

### 2.3 VR 인터랙션 디자인 상세

사이랩의 VR 인터랙션은 실제 실험 도구를 조작하는 경험을 최대한 재현하도록 설계되었다.

**핸드 트래킹 인터랙션 유형:**

| 인터랙션 | 현실 동작 | VR 구현 | 지원 기기 |
|---------|----------|---------|----------|
| 비커 들기/기울이기 | 손으로 잡아서 기울임 | 핸드 트래킹 그래브 + 회전 | Quest 3, PICO 4 |
