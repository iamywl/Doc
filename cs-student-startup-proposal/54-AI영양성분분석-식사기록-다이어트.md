### 핵심 기능 상세

- **사진 기반 음식 인식**: 식사 사진 촬영 → AI가 음식 종류, 양 자동 인식 (한식 800종 이상). EfficientDet으로 사진 내 개별 음식 영역을 감지(Object Detection)하고, ViT(Vision Transformer)로 각 음식을 분류. 양 추정은 참조 객체(수저, 그릇)와의 상대적 크기로 추정하여 오차율 15% 이내.

- **영양소 자동 기록**: 인식된 음식 기반 칼로리, 탄수화물, 단백질, 지방, 미량 영양소 자동 계산. 식품의약품안전처 식품영양성분 DB와 자체 구축 한식 영양 DB를 결합하여 총 5만 종 이상의 음식 영양 정보 제공. 일간/주간/월간 영양소 섭취 트렌드 시각화.

- **목표 기반 식단 조절**: 체중 감량/근육 증가/건강 유지 목표별 일일 권장 섭취량 설정 및 코칭. 사용자의 체중, 신장, 활동량, 목표를 기반으로 TDEE(총 일일 에너지 소비량) 자동 계산. 매 끼니 촬영 후 남은 칼로리와 부족 영양소를 알려주는 실시간 코칭.

- **전문가 상담**: 영양사/다이어트 전문가와 식단 리뷰 및 1:1 맞춤 상담. 앱 내 식사 기록을 영양사가 직접 확인하고 피드백, 주간 식단 리포트 기반 상담으로 효과적인 개선.

### AI 음식 인식 모델 상세

**2단계 파이프라인**:

| 단계 | 모델 | 역할 | 성능 |
|------|------|------|------|
| 1단계: 감지 | EfficientDet-D4 | 사진 내 개별 음식 영역 바운딩박스 감지 | mAP 89.2% |
| 2단계: 분류 | ViT-B/16 (Fine-tuned) | 감지된 영역의 음식 종류 분류 | Top-1 정확도 94.1% |

**한식 특화 학습 데이터**:
- 총 학습 이미지: 120만 장 (한식 80만 + 양식/일식/중식 40만)
- 카테고리: 한식 800종, 전체 1,200종
- 데이터 소스: 식약처 음식 이미지 DB, 자체 수집, 크라우드소싱
- 데이터 증강: 조명, 각도, 플레이팅 변화 적용

**양 추정 알고리즘**:
- 참조 객체 기반: 밥그릇, 수저, 접시 크기를 기준으로 상대적 크기 추정
- 깊이 추정: 단안 깊이 추정(Monocular Depth Estimation)으로 음식 높이 추정
- 보정 학습: 사용자가 "양 수정" 시 개인별 보정 모델 학습
- 양 추정 오차: 평균 15% 이내 (밥류), 20% 이내 (국/찌개류)

**황서진 (대표/PM)**: 대학원 컴퓨터 비전 연구실에서 음식 이미지 분류 프로젝트 1년 수행, ECCV 워크샵 포스터 발표. 푸드테크 해커톤 2회 수상. 다이어트 앱 5개 사용 경험에서 불편함을 느끼고 NutriSnap 아이디어 착상.

**임도윤 (CV 엔지니어)**: 음식 이미지 인식 석사 논문으로 학위 취득, 한식 데이터셋 구축 및 EfficientDet Fine-tuning 전문. 식약처 음식 이미지 DB 구축 프로젝트 참여 경험. 한식 인식 Top-1 정확도 94% 달성.
