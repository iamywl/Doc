## 1. 문제 인식

### 암호화폐 과세의 복잡성

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 암호화폐 투자자 수 | 630만명 | 금융위원회 2025 |
| 세금 계산 어려움 호소 비율 | 87.3% | 한국블록체인협회 2025 |
| 복수 거래소 이용 비율 | 62.4% | 업비트 리서치 2025 |
| DeFi/해외 거래소 이용 비율 | 38.7% | 코인데스크코리아 2025 |
| 세무 상담 비용 (1회) | 30~50만원 | 한국세무사회 2025 |
| 자진 신고 의향 | 71.2% | 자체 설문 (n=2,000) |
| 연간 거래 건수 (활성 투자자 평균) | 847건 | 업비트 리서치 2025 |
| 과세 대상 투자자 추정 | 약 120만명 | 국세청 2025 |

2025년 암호화폐 과세가 본격 시행되면서, 수백~수천 건의 거래 내역에서 과세 대상 금액을 정확히 계산하는 것이 투자자들의 최대 고민이 되었다. 복수 거래소 이용, DeFi 거래, 에어드롭, 스테이킹 수익 등 복잡한 거래 유형은 전문가도 계산이 어렵다. 세무사 상담 비용도 고가여서 소액 투자자에게는 부담이 크다.

### 사용자 인터뷰 주요 발언

> "업비트, 빗썸, 바이낸스 세 곳에서 거래했는데, 취득가액 산출이 정말 복잡해요. 각 거래소에서 CSV 다운받고 합치는 것만으로도 며칠이 걸립니다." — 투자자 A (32세, 연 거래 2,000건)

> "DeFi에서 유동성 풀 제공한 수익, 에어드롭으로 받은 토큰... 이런 것도 과세 대상인지 아닌지부터 모르겠어요." — 투자자 B (28세, DeFi 활성 사용자)

> "세무사 찾아가면 50만원인데, 내 코인 수익이 100만원이에요. 세금 계산 비용이 수익보다 비쌀 수도 있어요." — 투자자 C (24세, 소액 투자자)

> "세무사로서 암호화폐 고객이 늘고 있는데, 거래 내역 정리가 가장 시간이 많이 걸립니다. 자동화 도구가 절실합니다." — 세무사 D (38세)

### Pain Point 상세 분석

| Pain Point | 영향 범위 | 현재 대안 | 한계점 |
|------------|-----------|-----------|--------|
| 복수 거래소 내역 통합 | 62.4% 투자자 | 수동 CSV 병합 | 수일 소요, 오류 빈번 |
| 취득가액 산출 복잡성 | 전체 투자자 | 엑셀 수동 계산 | FIFO/이동평균 혼동 |
| DeFi/해외 거래소 처리 | 38.7% 투자자 | 대부분 미신고 | 과태료/가산세 위험 |
| 신고서 작성 난이도 | 전체 과세 대상 | 세무사 상담 | 건당 30~50만원 |
| 연중 세금 추적 불가 | 전체 투자자 | 연말 일괄 계산 | 절세 기회 상실 |
| 손익 상계 비인지 | 대다수 투자자 | 모름 | 과세 최적화 불가 |

### 핵심 기능

1. **거래소 자동 연동**: 업비트, 빗썸, 코인원, 바이낸스, 코빗 등 주요 거래소 API 연동, CSV 업로드 지원. ccxt 라이브러리를 활용하여 통합 거래소 인터페이스를 구현한다. API 키는 읽기 전용(Read-only) 권한만 요구하며, AWS KMS로 암호화하여 저장한다. 연동 후 과거 거래 내역을 자동 수집하고, 이후 실시간으로 신규 거래를 동기화한다.

2. **과세 자동 계산**: 선입선출(FIFO)/이동평균법 적용, 취득가 자동 산출, 250만원 기본공제 반영. 한국 소득세법 시행령에 맞춰 가상자산 기타소득 과세 표준을 정확히 반영한다. 양도차익 = 양도가액 - 취득가액 - 부대비용(수수료) 공식을 적용하되, FIFO와 이동평균법 중 유리한 방식을 자동 비교 제안한다. 원화(KRW) 기준 환산, 크로스 거래(BTC→ETH) 시 KRW 환산 처리도 자동화한다.

3. **신고서 자동 생성**: 국세청 양식에 맞는 기타소득 신고서 PDF 자동 생성, 홈택스 연동. 종합소득세 신고 기간(5월)에 맞춰 기타소득(가상자산) 항목의 신고서를 국세청 양식 그대로 PDF로 생성한다. 홈택스 직접 연동을 통해 원클릭 제출도 준비 중이다.

4. **절세 전략 AI**: 세금 예상액 실시간 표시, 손익 상계 기회 알림, 연말 절세 매도 추천. 연중 실시간으로 예상 세금을 추적하며, "지금 손실 중인 코인을 매도하면 X원의 세금을 절감할 수 있습니다"와 같은 절세 알림을 제공한다. Tax-Loss Harvesting 전략을 자동 분석하여 최적의 매도 타이밍을 추천한다.

5. **DeFi/NFT 지원**: 유니스왑 등 DeFi 거래, NFT 매매, 에어드롭, 스테이킹 수익 처리. Etherscan/Covalent API를 통해 온체인 거래를 자동 분석한다. 유동성 풀 제공 수익, 스왑, 에어드롭, 스테이킹 보상 등 30+ 유형의 DeFi 거래를 자동 분류하고 과세 여부를 판단한다.

### 세금 계산 알고리즘 상세

**선입선출법 (FIFO) 구현**:
```
취득 원장: [(날짜1, 수량1, 단가1), (날짜2, 수량2, 단가2), ...]
매도 시: 가장 오래된 취득분부터 순차 차감
양도차익 = 매도가 × 매도수량 - Σ(취득단가i × 차감수량i) - 수수료
```

**이동평균법 구현**:
```
가중평균 취득단가 = (기존 보유액 + 신규 취득액) / (기존 수량 + 신규 수량)
양도차익 = (매도가 - 가중평균 취득단가) × 매도수량 - 수수료
```

### CryptoTax의 경쟁 우위

1. **한국 세법 완벽 반영**: 가상자산 기타소득 과세, 250만원 기본공제, FIFO/이동평균법 선택, 크로스 거래 KRW 환산 등 한국 세법 특수 사항을 100% 반영한다.

2. **절세 전략 AI**: 단순 세금 계산을 넘어 Tax-Loss Harvesting, 손익 상계, 연말 최적 매도 등 능동적 절세 전략을 추천하는 유일한 국내 서비스다.

3. **DeFi 완전 지원**: Etherscan/Covalent API를 통한 온체인 분석으로 30+ 유형의 DeFi 거래를 자동 분류하는 기능은 경쟁사 대비 압도적이다.

4. **세무사 생태계**: 파트너 세무사 네트워크를 구축하여, CryptoTax 데이터 기반으로 50% 할인된 세무 상담을 연결한다. 세무사에게는 고객 확보 채널을, 사용자에게는 저렴한 전문 상담을 제공한다.
