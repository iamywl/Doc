### 4.3 핵심 기능

| 기능 | 설명 |
|------|------|
| 실시간 수질 모니터링 | 수온, DO, pH, 염도, 탁도, 암모니아 24시간 실시간 측정 |
| AI 수질 예측 | 6~24시간 앞 수질 변화 예측으로 선제적 대응 |
| 질병 조기 감지 | 수중 카메라 영상 + 수질 이상 패턴으로 질병 72시간 전 경보 |
| 자동 급이 최적화 | 어류 성장 단계, 섭이 패턴, 수질 상태에 따른 자동 급이량 조절 |
| 자동 환수 제어 | 수질 악화 시 환수 밸브·에어레이터 자동 가동 |
| 성장 추적 | 수중 카메라로 어체 크기 자동 측정, 출하 시기 예측 |
| 기상 연동 경보 | 태풍, 폭염, 한파 등 기상 이벤트에 따른 사전 대응 가이드 |
| 생산 이력 관리 | 사료 투입, 수질 기록, 약품 사용 이력 자동 기록 (HACCP 대응) |

- **질병 조기 감지**: 수질+영상 복합 분석으로 72시간 전 질병 경보 (업계 최초)
- **자동 급이 최적화**: 사료비 30% 절감 실현
- **고령 사용자 UI**: 직관적인 모바일 앱으로 고령 양식인도 쉽게 사용
