## 4. 솔루션

### 4-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 수요예측 엔진 | 시계열 분석 + 외부 변수(날씨, 유행성 질환, 지역 인구) 결합 수요예측 |
| 자동 발주 시스템 | 적정 재고 수준 자동 산출 및 도매상 연동 자동 발주 |
| 유통기한 관리 | 유통기한 임박 의약품 자동 경고 및 소진 전략 제안 |
| 처방 트렌드 분석 | 건보 청구 데이터 기반 처방 패턴 변화 조기 감지 |
| 재고 대시보드 | 실시간 재고 현황, 과잉/부족 알림, 재무 영향 분석 |
| 도매상 가격 비교 | 복수 도매상 가격 비교 및 최적 발주처 추천 |
| 폐기 예측 모듈 | 폐기 가능성 높은 품목 사전 식별 및 대응 방안 제시 |
| 약국 간 재고 교환 | 인근 약국 간 과잉 재고 교환 매칭 네트워크 |
