## 4. 솔루션

### 4.1 핵심 기능

**① 음소 단위 발음 분석 AI**
- 한국어 40개 음소별 정밀 발음 평가 (정확도 / 유창성 / 운율)
- 입모양·혀 위치 3D 가이드 애니메이션
- 원어민 발음과 사용자 발음 파형 비교 시각화

**② 모국어별 맞춤 교정**
- 영어, 중국어, 일본어, 베트남어, 스페인어 등 14개 모국어별 오류 패턴 DB
- 모국어 간섭 현상 기반 취약 음소 집중 학습
- 개인화된 학습 경로 AI가 자동 설계

**③ AI 대화 파트너**
- 상황별 롤플레이 (카페, 병원, 은행, 회사 등 50개+ 시나리오)
- 실시간 발음 평가 + 자연스러운 대화 유지
- 격식체/비격식체 전환 연습

**④ TOPIK 말하기 시험 대비**
- TOPIK 말하기 모의시험 AI 평가
- 문제 유형별 연습 및 전략 가이드
- 등급별 맞춤 학습 콘텐츠

### 4.3 기술 스택

| 구분 | 기술 | 선정 사유 |
|------|------|-----------|
| 음성 인식 | Wav2Vec2 (한국어 Fine-tuned) | 음소 단위 정밀 인식 |
| 발음 평가 | GOP (Goodness of Pronunciation) | 음소별 정확도 정량 평가 |
| 운율 분석 | Praat + Parselmouth | 억양, 강세, 리듬 분석 |
| 대화 AI | GPT-4 API (한국어 교육 프롬프트) | 자연스러운 한국어 대화 생성 |
| TTS | VITS (한국어 Fine-tuned) | 고품질 원어민 발음 음성 합성 |
| 백엔드 | FastAPI + WebSocket | 실시간 음성 스트리밍 처리 |
| 모바일 | React Native | 글로벌 사용자 대상 크로스 플랫폼 |
| DB | MongoDB + Redis | 유연한 학습 이력 관리 |
| 인프라 | GCP (GPU T4) | 음성 AI 추론 최적화 |
| 3D 가이드 | Three.js, Rive | 입모양·혀 위치 애니메이션 |
