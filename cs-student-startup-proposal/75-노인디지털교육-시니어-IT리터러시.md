## 1. 문제 인식

### 시니어 디지털 격차 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 고령층 디지털 정보화 수준 (일반 국민 대비) | 69.9점 | 과학기술정보통신부 (2025) |
| 키오스크 이용 불편 경험 (65세 이상) | 74% | 한국소비자원 (2025) |
| 스마트폰 기본 기능 사용 어려움 | 58% | 한국정보화진흥원 (2025) |
| 디지털 금융 서비스 이용 불가율 (70세 이상) | 63% | 금융감독원 (2025) |
| 65세 이상 인구 비율 (2026년) | 19.2% | 통계청 (2025) |
| 디지털 교육 참여 경험률 (65세 이상) | 15% | 한국노인인력개발원 (2025) |
| 디지털 소외로 인한 우울감 경험률 | 38% | 서울시 복지재단 (2025) |
| 자녀에게 디지털 도움 요청 부담감 | 72% | 한국정보화진흥원 (2025) |

키오스크 확산, 은행 창구 축소, 비대면 행정 서비스 전환으로 디지털 소외 문제가 심화되고 있다. 기존 디지털 교육은 일회성 강좌 중심으로 실제 사용 능력 향상에 한계가 있으며, 어르신 눈높이에 맞춘 지속적 학습 도구가 절실하다.

### 사용자 페인 포인트 분석

**어르신 당사자의 목소리:**
- *"패스트푸드점에서 키오스크로 주문하라는데 뒤에 줄이 서 있으니 당황해서 그냥 나왔어요."* - 70세 김할머니
- *"은행에서 모바일 앱으로 하라고 하는데 어떻게 하는지 모르겠어요. 창구는 줄이 너무 길고."* - 73세 이할아버지
- *"손녀가 보내준 사진을 카톡에서 저장하고 싶은데 아들한테 물어보기가 미안해요. 이미 여러 번 물어봤거든요."* - 68세 박할머니
- *"병원 예약을 인터넷으로만 받는다는데 어떻게 해야 하는지..."* - 75세 최할아버지

**가족의 어려움:**
- 부모님의 디지털 문의에 매번 대응하기 어려움 (직장인 82% 응답)
- 원격으로 설명하는 것이 극도로 비효율적 ("왼쪽 위 버튼"이 어디인지 설명이 안 됨)
- 부모님이 디지털 사기에 노출될까 걱정 (피싱/보이스피싱)

**복지관/지자체의 과제:**
- 디지털 교육 강사 부족, 1회성 교육 후 학습 유지가 어려움
- 어르신별 수준 차이가 커서 단체 교육의 효과가 낮음
- 교육 효과 측정 및 관리 도구 부재

### 핵심 기능 상세

#### 2.1 AI 대화형 학습
- **음성 대화 교육**: "할머니, 오늘은 카카오톡으로 사진 보내는 법을 배워볼까요?" 같은 친근한 음성 대화
- **맞춤형 속도**: 어르신의 학습 속도에 맞춰 자동 조절, 천천히 반복 설명
- **단계별 가이드**: 화면의 어디를 터치해야 하는지 시각/음성/진동으로 동시 안내
- **반복 학습**: 어르신이 원하는 만큼 같은 내용 반복 가능, 부담 없는 학습 환경
- **성취 보상**: 학습 완료 시 스티커/배지 획득, 가족에게 성취 알림 발송
- **KoGPT 기반**: 자연스러운 한국어 대화, 어르신의 질문에 맞춤 응답

#### 2.2 큰 글씨 UI (시니어 접근성 특화)
- **기본 글씨 18pt 이상**: 모든 텍스트 최소 18pt, 제목 24pt 이상
- **고대비 색상**: WCAG AAA 기준 충족, 배경 대비 7:1 이상
- **큰 터치 영역**: 최소 48dp 이상 터치 영역, 오조작 방지
- **음성 안내**: 모든 버튼과 기능에 음성 설명 탑재
- **간결한 네비게이션**: 한 화면에 최대 4개 선택지, 뒤로가기 항상 표시
- **돋보기 모드**: 화면 부분 확대 기능, 핀치 줌 간편 지원

#### 2.3 키오스크 시뮬레이터
- **실제 키오스크 1:1 재현**: 실제 키오스크 화면을 그대로 재현한 연습 모드
- **10종 이상 키오스크**: 패스트푸드(맥도날드, 롯데리아), 영화관(CGV, 메가박스), 관공서, ATM, 병원, 카페, 기차역, 공항 체크인
- **단계별 가이드**: 음성+시각으로 "여기 빨간 버튼을 누르세요" 안내
- **무한 반복**: 실수해도 걱정 없는 시뮬레이션 환경
- **난이도 조절**: 초급(음성 가이드 전체) → 중급(힌트만) → 고급(혼자서)

#### 2.4 원격 도우미
- **화면 공유**: 어르신의 스마트폰 화면을 가족/봉사자에게 실시간 공유
- **원격 포인터**: 도우미가 "여기를 누르세요"라고 화면에 포인터 표시
- **음성 통화**: 화면 공유와 동시에 음성 통화로 설명
- **1:1 매칭**: 자원봉사 대학생과 어르신 1:1 정기 매칭 서비스
- **가족 연동**: 자녀가 원격으로 부모님의 스마트폰 문제 즉시 해결

### 시장 성장 동인

1. **초고령사회 진입**: 2025년 65세 이상 인구 20% 돌파, 매년 고령 인구 증가
2. **비대면 전환 가속**: 키오스크, 모바일 뱅킹, 온라인 행정 등 디지털 전환 불가피
3. **정부 디지털 포용 정책**: 과기정통부 디지털 포용 예산 연간 20% 이상 확대
4. **가족 부양 부담**: 디지털 문의에 대한 가족 부담 증가, 전문 교육 서비스 수요
5. **사회적 이슈화**: 키오스크 차별 사례 언론 보도 증가, 사회적 관심 고조
