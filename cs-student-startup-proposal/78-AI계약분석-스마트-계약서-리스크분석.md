## 1. 문제 인식

### 중소기업 계약 리스크의 현실

| 지표 | 수치 | 출처 |
|------|------|------|
| 중소기업 법률 검토 없이 계약 체결 비율 | 72% | 중소벤처기업부 |
| 연간 계약 분쟁 건수 | 48만건 (2025년) | 법원행정처 |
| 계약 분쟁 평균 피해액 | 건당 8,700만원 | 대한상사중재원 |
| 연간 총 계약 분쟁 피해액 | 4.2조원 | 한국경제연구원 |
| 변호사 계약서 검토 비용 | 건당 30~100만원 | 대한변호사협회 |
| 프리랜서 계약 분쟁 경험률 | 43% | 한국콘텐츠진흥원 |
| 부동산 계약 분쟁 | 연 12만건 | 한국부동산원 |
| 중소기업 법무 인력 보유율 | 8% | 중소기업중앙회 |
| 계약서 평균 검토 소요 시간 | 3~5일 | 법률신문 |
| 기업 간 계약 분쟁 소송 평균 기간 | 18개월 | 법원행정처 |

중소기업과 스타트업, 프리랜서의 대다수가 법률 전문가 없이 계약서를 검토하고 체결하고 있다. 계약서의 불리한 조항이나 법적 리스크를 사전에 파악하지 못해 분쟁이 발생하면, 금전적 피해뿐 아니라 사업 존속까지 위협받는 경우가 빈번하다.

변호사에게 계약서 검토를 의뢰하면 건당 30~100만원의 비용과 3~5일의 시간이 소요되어, 수많은 계약을 처리해야 하는 기업에게는 현실적이지 않다. 특히 표준계약서를 약간 변형한 '함정 조항'이나 상대방에게 일방적으로 유리한 조건을 비전문가가 발견하기란 매우 어렵다.

> *"특약 조항의 법적 효력을 항상 확신하기 어렵습니다. 빠르게 검토해주는 도구가 있으면 전문성을 높일 수 있을 것 같습니다."*

## 2. 솔루션

### 핵심 기능

| 기능 | 설명 | 기술 요소 |
|------|------|-----------|
| 계약서 자동 분석 | PDF/이미지 업로드 시 계약서 내용 자동 추출 및 구조화 | OCR + LLM 기반 문서 파싱 |
| 리스크 스코어링 | 조항별 위험도를 점수화하여 시각적으로 표시 | Fine-tuned LLM, 법률 규칙 엔진 |
| 불리한 조항 탐지 | 일방적으로 불리한 조건, 함정 조항 자동 하이라이트 | NER + 법률 패턴 매칭 |
| 누락 조항 분석 | 계약 유형별 필수 조항 누락 여부 점검 | 업종별 표준 계약서 비교 엔진 |
| 법률 해설 제공 | 어려운 법률 용어를 쉬운 말로 해설 | LLM 기반 법률 용어 해설 생성 |
| 계약서 비교 | 이전 버전, 표준계약서와의 차이점 자동 비교 | Diff 알고리즘 + 시맨틱 비교 |
| 수정 제안 | 리스크 조항에 대한 대안 문구 자동 제안 | 법률 문서 생성 LLM |
| 변호사 연결 | 심층 검토 필요 시 전문 변호사 즉시 매칭 | 법무법인 네트워크 연계 |

### 기술 스택

| 구분 | 기술 | 선정 이유 |
|------|------|-----------|
| LLM | GPT-4 Fine-tuned + Claude API | 법률 문서 이해 및 리스크 분석 |
| NLP/NER | spaCy + 한국어 법률 NER 모델 | 당사자, 금액, 기간, 조건 등 개체 추출 |
| OCR | Naver Clova OCR + Tesseract | 한국어 문서 이미지 텍스트 추출 |
| 벡터 DB | Pinecone / Milvus | 유사 계약서 검색, 판례 매칭 |
| 백엔드 | FastAPI (Python) | 비동기 문서 처리 파이프라인 |
| 데이터베이스 | PostgreSQL + Redis | 분석 이력 및 캐싱 |
| 프론트엔드 | Next.js + TypeScript | SSR 기반 웹 애플리케이션 |
| 인프라 | AWS (ECS, S3, Lambda) | 서버리스 문서 처리 및 확장성 |
| 문서 파싱 | HWP 파서 (Python) + PDF.js | 한글(HWP) 및 PDF 문서 파싱 |
