## 4. 솔루션

### 4-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| UEBA 이상행위 탐지 | 사용자 행동 패턴 학습 후 비정상 데이터 접근·전송 실시간 감지 |
| 실시간 DLP 연동 | 기존 DLP 솔루션과 연계하여 유출 시도 차단 및 로깅 |
| 포렌식 증거 자동 수집 | 유출 탐지 시 법적 증거력 있는 형태로 자동 증거 보전 |
| 퇴직자 위험 스코어링 | 퇴직 예정자의 이상 접근 패턴 사전 감지 및 등급 부여 |
| 클라우드 모니터링 | SaaS/IaaS 환경의 데이터 흐름 모니터링 (M365, Google, AWS) |
| AI 오탐 최소화 | 컨텍스트 인식 AI로 오탐률을 기존 대비 80% 감소 |
| 포렌식 리포트 자동화 | 법적 절차에 활용 가능한 표준 포렌식 보고서 자동 생성 |
| 위협 인텔리전스 | 다크웹 모니터링, 유출 데이터 탐지, 외부 위협 조기 경보 |

### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 엔드포인트 에이전트 | Rust + eBPF | 경량 시스템 모니터링 에이전트 |
| UEBA 엔진 | PyTorch + Isolation Forest | 사용자 행동 이상 탐지 |
| 로그 수집 | Fluentd + Apache Kafka | 대용량 로그 실시간 수집 |
| 데이터 저장 | Elasticsearch + MinIO | 로그 검색 + 증거 저장 |
| 백엔드 | Go + gRPC | 고성능 이벤트 처리 API |
| 프론트엔드 | React + D3.js | 보안 대시보드 시각화 |
| 포렌식 엔진 | Python + Sleuth Kit | 디지털 증거 분석 자동화 |
| 클라우드 연동 | CASB 아키텍처 | M365/Google/AWS API 연동 |
| 인프라 | 온프레미스 + 하이브리드 | 고객사 데이터 센터 배포 |
| 보안 | HashiCorp Vault | 암호키 관리, 증거 무결성 |

## 9. 사회적 가치

### 9-1. 산업 기술 보호

국내 핵심 산업기술의 해외 유출을 방지하여 국가 기술 경쟁력을 보호합니다. 연간 28건에 달하는 산업기술 해외유출 적발 건수의 사전 예방에 기여합니다.

### 9-2. 중소기업 보안 민주화

대기업 전유물이었던 고도화된 보안 솔루션을 합리적 가격의 SaaS로 중소기업에 제공하여 보안 격차를 해소합니다.

### 9-3. 디지털 신뢰 경제 구축

기업 데이터의 안전한 관리를 통해 디지털 경제의 신뢰 기반을 강화하고, 기업 간 안전한 데이터 교류를 촉진합니다.

### 9-4. 일자리 창출

보안 분석가, 포렌식 전문가 등 고급 보안 인력의 수요를 창출하고, 청년 보안 전문가 양성에 기여합니다.
