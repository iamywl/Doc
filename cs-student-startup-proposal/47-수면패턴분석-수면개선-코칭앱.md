## 1. 문제 인식

### 한국인 수면 문제 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 한국인 평균 수면 시간 | 7시간 22분 (OECD 최하위) | OECD (2025) |
| 불면증 진료 인원 | 120만 명 | 건강보험심사평가원 (2025) |
| 수면 부족 자각률 | 53.7% | 국민건강영양조사 (2024) |
| 수면 장애로 인한 사회적 비용 | 연 12조원 | 수면연구학회 (2024) |
| 수면 관련 앱 사용 경험률 | 18.3% | 한국갤럽 (2025) |
| 20대 불면증 진료 증가율 | 전년 대비 32% | 건강보험심사평가원 (2025) |
| 수면제 처방 건수 | 연 890만 건 | 식품의약품안전처 (2025) |
| 수면 부족 관련 교통사고 | 연 21,000건 | 도로교통공단 (2024) |

수면 부족은 집중력 저하, 우울증, 면역력 약화 등 광범위한 건강 문제를 유발한다. 기존 수면 앱은 단순 기록에 그치며, 수면 환경(온도, 소음, 조도)과 생활 패턴을 종합 분석하여 실질적인 개선을 코칭하는 서비스는 부재하다.

### 사용자 페인포인트 심층 분석

**실제 사용자 인터뷰 (2025년 11월, 직장인/대학생 80명 대상)**

> "수면 앱 3개 써봤는데, 다 '어젯밤 5시간 잤네요'만 알려주고 끝이에요. 어떻게 해야 잘 잘 수 있는지를 알고 싶은 건데." - 32세 IT기업 개발자

> "잠들기 전에 방이 너무 덥거나 밖에서 소리가 나면 잠을 못 자는데, 이런 환경 요인을 추적해주는 앱은 없더라고요." - 24세 대학원생

> "야근 후 새벽에 귀가하면 수면 리듬이 완전히 깨져요. 교대근무자를 위한 맞춤 수면 가이드가 필요해요." - 28세 간호사

> "수면제에 의존하기 싫은데, 비약물적 방법으로 수면을 개선할 방법을 잘 모르겠어요." - 35세 회사원

**핵심 페인포인트 5가지:**

1. **기록 vs 개선의 괴리**: 기존 앱은 데이터를 보여주지만, 구체적인 행동 변화를 유도하지 않음
2. **환경 요인 무시**: 수면의 질에 결정적 영향을 미치는 온도(18-20도), 소음(40dB 이하), 조도(0.5lux 이하) 등의 환경 데이터를 활용하지 않음
3. **개인차 미반영**: 크로노타입(아침형/저녁형), 생활 패턴, 건강 상태에 따른 개인화 부재
4. **지속 동기 부족**: 초기 흥미 후 2주 내 사용 이탈률 68%, 행동 변화 유지를 위한 게이미피케이션 미비
5. **전문 상담 부재**: 심각한 수면 문제 시 수면 전문의 연결이 어려움

### 수면 부족의 연쇄적 영향

수면 부족은 단순 피로를 넘어 심각한 건강/사회 문제를 유발한다:

- **인지 기능**: 6시간 이하 수면 시 인지 기능이 혈중 알코올 농도 0.1%와 동일한 수준으로 저하 (Walker, 2017)
- **정신 건강**: 만성적 수면 부족은 우울증 발생률 2.5배 증가 (대한수면연구학회, 2024)
- **생산성 손실**: 수면 부족 근로자의 연간 생산성 손실 인당 약 280만원 (한국생산성본부, 2025)
- **만성 질환**: 7시간 미만 수면 시 당뇨 위험 1.5배, 심혈관 질환 위험 1.4배 증가

### 핵심 기능 상세

#### 2-1. 수면 사이클 추적

스마트워치의 가속도계, 심박 센서, SpO2 센서 데이터를 결합하여 수면 단계를 정밀 분류한다.

- **4단계 수면 분류**: 각성(Wake), 렘 수면(REM), 얕은 수면(Light NREM), 깊은 수면(Deep NREM)
- **LSTM 기반 분류 모델**: 30초 에포크(epoch) 단위로 수면 단계를 분류, 정확도 87% (PSG 대비)
- **수면 효율 점수**: 총 침대 시간 대비 실제 수면 시간 비율, 수면 잠복기, 중간 각성 횟수 등을 종합한 0-100 점수
- **수면 부채 계산**: 개인 최적 수면 시간 대비 누적 부족량을 계산하여 회복 일정 제안
- **크로노타입 판별**: 2주 데이터 분석을 통해 아침형/중간형/저녁형 크로노타입을 판별하고, 최적 취침/기상 시간 제안

#### 2-2. 수면 환경 분석

IoT 센서 또는 스마트폰 내장 센서를 활용하여 수면 환경을 실시간 모니터링한다.

- **온도 모니터링**: 실내 온도 측정, 최적 수면 온도(18-20도) 대비 편차 알림
- **소음 측정**: 데시벨 측정 및 소음 이벤트(교통, 이웃 소음 등) 기록, 소음 패턴 분석
- **조도 감지**: 수면 전후 조도 변화 추적, 블루라이트 노출 시간 분석
- **습도 모니터링**: 실내 습도 측정, 최적 습도(40-60%) 유지 알림
- **환경-수면 상관 분석**: 환경 요인과 수면 품질 간 상관관계를 AI가 분석하여 최적 환경 조건 제안

#### 2-3. AI 수면 위생 코칭

인지행동치료-불면증(CBT-I) 기반의 개인화된 수면 위생 코칭을 제공한다.

- **수면 제한 요법**: 수면 효율이 낮을 때 침대에 있는 시간을 제한하여 수면 압력을 높이는 기법 적용
- **자극 통제 요법**: '침대는 잠자는 곳'이라는 연합을 강화하는 행동 지침 제공
- **카페인/알코올 가이드**: 개인 대사 속도를 고려한 마지막 카페인 섭취 시간 제안 (평균 취침 8시간 전)
- **운동 타이밍**: 크로노타입과 수면 패턴에 맞는 최적 운동 시간대 제안
- **수면 전 루틴**: 개인 선호와 효과 데이터를 기반으로 한 수면 전 이완 루틴 커스터마이징

#### 2-4. 스마트 알람

- **얕은 수면 기상**: REM 또는 Light NREM 단계에서 설정 시간 +-20분 내 최적 기상 시점 탐지
- **점진적 기상**: 설정 시간 10분 전부터 점진적으로 알람 강도를 높여 자연스러운 기상 유도
- **일출 시뮬레이션**: IoT 조명과 연동하여 자연 일출을 모방한 점진적 조명 변화로 기상 보조

### 수면 분석 알고리즘 상세

SleepWise의 수면 분석 파이프라인은 다음과 같이 구성된다:

1. **원시 데이터 수집**: 스마트워치에서 가속도(3축), 심박수(BPM), SpO2(%) 데이터를 1초 간격으로 수집
2. **전처리**: 노이즈 필터링(밴드패스 필터 0.5-5Hz), 정규화, 30초 에포크 윈도우 분할
3. **특징 추출**: 시간/주파수 도메인 특징 42개 추출 (HRV, 움직임 강도, 심박 변동성 등)
4. **수면 단계 분류**: Bi-LSTM 모델이 에포크별 수면 단계 분류 (정확도 87%, Cohen's Kappa 0.72)
5. **수면 품질 평가**: 수면 효율, 수면 잠복기, 각성 횟수, 렘/깊은 수면 비율을 종합하여 0-100 점수 산출
6. **환경 데이터 통합**: IoT 센서의 온도/소음/조도 데이터와 수면 품질 간 다변량 회귀 분석
7. **코칭 액션 생성**: 강화학습(RL) 기반 에이전트가 개인 데이터를 고려한 최적 수면 위생 행동을 추천
