## 3. 문제 인식

### 3-1. 시장 현황 및 통계

| 통계 항목 | 수치 | 출처 |
|-----------|------|------|
| 전국 공공화장실 수 | 약 52,000개소 | 행정안전부 (2025) |
| 공공화장실 불만족률 | 43.7% | 한국소비자원 |
| 화장실 관련 민원 (연간) | 약 84,000건 | 국민권익위원회 |
| 공공화장실 관리 예산 (연) | 전국 약 2,800억 원 | 행정안전부 |
| 화장실 1회 청소 비용 | 평균 8,500원 | 한국환경공단 |
| 청소 인력 부족률 | 28.3% | 고용노동부 |
| 화장실 범죄 발생 건수 (연) | 약 3,200건 | 경찰청 |
| 화장실 불결로 인한 감염병 | 연 24,000건 추정 | 질병관리청 |
| 고속도로 휴게소 화장실 수 | 약 4,200개소 | 한국도로공사 |
| 외국인 관광객 화장실 만족도 | 52.3점 (100점) | 한국관광공사 |

### 3-2. 현장 목소리

> "하루에 세 번 정해진 시간에 청소를 하지만, 이용이 몰리는 시간 직후에는 엉망이 됩니다. 효율적인 청소 타이밍을 알 수 있으면 좋겠어요."
> — 서울 강남구 공공화장실 청소 담당자 김OO

> "화장지 없음, 비누 없음 같은 기본적인 민원이 전체의 60% 이상이에요. 실시간으로 소모품 상태를 알 수 있으면 좋겠습니다."
> — 경기도 시설관리공단 담당자 이OO

> "고속도로 휴게소 화장실 때문에 SNS에 악평이 올라오면 이미지가 크게 실추됩니다."
> — 한국도로공사 휴게시설팀 박OO

> "관광지 화장실이 불결하면 재방문 의사가 확 떨어져요. 외국인 관광객 평가에서도 화장실이 항상 하위입니다."
> — 제주특별자치도 관광정책과 최OO

## 4. 솔루션

### 4-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| 이용량 카운팅 | 비접촉 센서로 출입 인원 실시간 카운팅 (프라이버시 보호) |
| 소모품 잔량 감지 | 화장지, 비누, 핸드타올 잔량 IoT 센서 실시간 모니터링 |

| 설비 이상 감지 | 변기 막힘, 수도 누수, 조명 고장 자동 감지 |
| AI 청소 스케줄링 | 이용 패턴·환경 데이터 기반 최적 청소 시점 자동 배정 |

| 이용자 피드백 | 터치 패널 간편 평가 (좋음/보통/나쁨) 실시간 수집 |

### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| IoT 센서 | 자체 설계 MCU + BLE 5.0 | 각종 환경·상태 센서 |
| 통신 | LoRaWAN + LTE Cat-M1 | 저전력 장거리 센서 데이터 전송 |
| 게이트웨이 | Raspberry Pi + Custom | 현장 데이터 수집·전송 |
| 백엔드 | Node.js + Express | API 서버, 실시간 이벤트 |
| AI 분석 | Python + scikit-learn | 이용 패턴 예측, 스케줄 최적화 |
| 데이터베이스 | InfluxDB + PostgreSQL | 시계열 센서 + 관계형 데이터 |
| 프론트엔드 | React + Ant Design | 관제 대시보드 |
| 모바일 | Flutter | 청소원 모바일 앱 |
| 디스플레이 | Android 키오스크 모드 | 위생 점수 공개 패널 |
| 인프라 | NCloud | 공공 클라우드 인프라 |

## 9. 사회적 가치

### 9-1. 공중위생 향상

실시간 위생 모니터링으로 공공화장실의 청결도를 크게 높여 감염병 예방에 기여합니다. 특히 노로바이러스, 대장균 등의 전파 경로를 차단합니다.

### 9-2. 시민 삶의 질 개선

화장실은 도시 생활의 기본 인프라입니다. 깨끗한 공공화장실은 시민의 외출 편의성과 삶의 질을 직접적으로 향상시킵니다.

### 9-3. 관광 경쟁력 강화

외국인 관광객의 화장실 만족도를 높여 한국 관광의 전반적 이미지를 개선하고 재방문율을 높입니다.

### 9-4. 청소 노동자 처우 개선

데이터 기반 효율적 청소 스케줄링으로 불필요한 과도 노동을 줄이고, 청소 노동의 가치를 가시화합니다.
