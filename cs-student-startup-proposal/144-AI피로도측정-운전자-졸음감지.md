### 4.3 핵심 기능

| 기능 | 설명 |
|------|------|
| 얼굴 랜드마크 분석 | 68개 포인트 추적으로 눈 깜빡임 빈도, PERCLOS, 하품 감지 |
| 시선 추적 | 운전 중 시선 이탈 시간·방향 분석 (주의 분산 감지) |
| HRV 피로도 분석 | 심박변이로 자율신경계 피로 수준 정량 측정 |
| 운전 패턴 분석 | 핸들 미세 조향, 차선 이탈 빈도, 속도 변화로 졸음 전조 감지 |
| 다중 센서 융합 | 카메라+HRV+운전패턴 3중 융합으로 오탐률 5% 이하 |
| 3단계 경보 | 주의→경고→비상 단계별 차등 경보 (음성, 진동, 경고음) |
| 피로도 점수 | 0~100 실시간 피로도 점수 + 안전 운전 권고 |
| 관제 대시보드 | 운수업체 전 차량 실시간 모니터링 + 운전 리포트 |
| 보험 연동 | 안전 운전 데이터 보험사 공유로 보험료 할인 혜택 |
| 야간/IR 모드 | 적외선 카메라로 야간에도 정확한 얼굴 분석 |
