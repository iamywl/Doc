## 3. 문제 인식

### 3.1 핵심 문제 통계

| 지표 | 수치 | 출처 |
|------|------|------|

## 4. 솔루션

### 4.1 핵심 기능

#### 기능 1: 비주얼 SLAM 기반 실내 측위
- 스마트폰 카메라로 주변 환경을 인식하여 정밀 위치 파악 (1m 이내 오차)
- 별도 비콘/센서 설치 최소화 (기존 대비 도입 비용 80% 절감)
- 3D 공간 맵 자동 구축 및 업데이트
- UWB/BLE 비콘 하이브리드로 정확도 보강

#### 기능 2: AR 경로 오버레이
- 카메라 화면 위에 화살표, 목적지 표시가 AR로 오버레이
- 계단/엘리베이터/에스컬레이터 안내
- 남은 거리, 예상 소요 시간 표시
- 시각장애인을 위한 음성 + 진동 안내 모드

#### 기능 3: 스마트 POI 검색
- "내과 3번 진료실", "A-127 탑승구" 등 자연어 검색
- 가장 가까운 화장실/ATM/카페 등 주변 POI 검색
- 즐겨찾기, 최근 방문 경로 저장
- 다국어 지원 (한/영/중/일)

#### 기능 4: 건물 관리자 대시보드
- 실내 맵 편집 도구 (매장/시설 변경 반영)
- 방문객 동선 히트맵 분석
- 혼잡도 실시간 모니터링
- 위치 기반 프로모션/알림 설정

### 4.3 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| AR | ARKit (iOS) / ARCore (Android) | AR 렌더링, 평면 감지 |
| CV | ORB-SLAM3 + SuperPoint | 비주얼 SLAM, 특징점 매칭 |
| 측위 | UWB + BLE 5.1 | 보조 실내 측위 (하이브리드) |
| 3D | Three.js + WebGL | 3D 실내 맵 시각화 |
| Backend | Go + gRPC | 경로 탐색 API, 실시간 통신 |
| Frontend | React Native | 크로스플랫폼 모바일 앱 |
| Frontend | React + Deck.gl | 관리자 대시보드 |
| Database | PostgreSQL + PostGIS | 공간 데이터, POI 관리 |
| Database | Redis | 비주얼 특징 벡터 캐싱 |
| Infra | AWS EKS + S3 + CloudFront | 3D 맵 배포, 서비스 인프라 |
