## 1. 문제 인식

### 소규모 팀 프로젝트 관리 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 스타트업 수 (20인 이하) | 약 12만개 | 중소벤처기업부 (2025) |
| 기존 PM 도구 활용 포기율 | 68% | 자사 설문 (2025) |
| 카카오톡/메신저로 업무 관리 비율 | 54% | 잡코리아 (2025) |
| 프로젝트 지연 원인 중 커뮤니케이션 문제 | 43% | PMI Korea (2025) |
| 소규모 팀 월 PM 도구 예산 | 팀당 10만원 미만 | 자사 설문 (2025) |
| PM 도구 온보딩에 소요되는 평균 시간 | 4.2시간 (Jira 기준) | 자사 측정 (2025) |
| 기존 PM 도구 기능 중 실제 사용 비율 | 23% | Gartner (2025) |

Jira, Asana, Monday.com 등 기존 PM 도구는 대규모 조직을 위해 설계되어 기능이 과도하게 복잡하다. 소규모 팀은 초기 설정에만 수시간이 소요되며, 결국 카카오톡이나 엑셀로 회귀하는 경우가 대부분이다. 한국어 최적화 부족과 높은 요금도 진입 장벽이다.

### 사용자 페인포인트 분석

**소규모 팀 리더/PM 인터뷰 (N=250, 2025년 자체 조사)**

> "Jira를 도입했는데 워크플로우 설정만 3일 걸렸어요. 결국 5명이 쓰는데 너무 과한 도구예요." - 스타트업 CTO 김OO (32세)

> "카톡으로 업무 지시하면 대화에 묻혀서 뭐가 진행 중인지 아무도 몰라요." - 디자인 에이전시 팀장 이OO (29세)

> "프리랜서팀인데 주간 리포트를 쓰는 데 2시간이 걸려요. AI가 자동으로 만들어주면 좋겠어요." - 프리랜서 PM 박OO (35세)

> "Notion은 자유도가 너무 높아서 프로젝트 관리 용으로 쓰기엔 구조가 안 잡혀요." - 스타트업 대표 정OO (27세)

| 페인포인트 | 응답 비율 | 심각도 (5점) |
|-----------|----------|-------------|
| PM 도구 복잡성 (과도한 기능/설정) | 68.4% | 4.6 |
| 카카오톡에 업무 지시가 묻힘 | 54.2% | 4.3 |
| 프로젝트 현황 파악 어려움 | 62.1% | 4.4 |
| 리포트/회고 작성 시간 부담 | 48.7% | 3.8 |
| 시간 투입 대비 성과 측정 불가 | 45.3% | 3.6 |
| 해외 도구의 한국어 미지원 | 41.8% | 3.4 |
| 높은 요금 (멤버당 $10+) | 55.6% | 4.1 |

### 기존 PM 도구의 한계

| 도구 | 주요 문제점 | 소규모 팀 부적합 이유 |
|------|-----------|---------------------|
| Jira | 초기 설정 복잡, 학습 곡선 높음 | 5명 팀에 엔터프라이즈급 기능 불필요 |
| Asana | 영어 중심, 높은 가격, 기능 과잉 | 한국어 미흡, 멤버당 $10.99 |
| Monday.com | 자유도 높지만 PM 특화 부족 | 프로젝트 관리보다 일반 업무 관리에 가까움 |
| Notion | PM 전용 도구가 아님, 구조화 어려움 | 매번 직접 설계 필요, 시간 추적 없음 |
| Linear | 영어만 지원, 개발팀 한정 | 비개발 직군 사용 어려움, 한국어 미지원 |

### 주요 기능 상세

- **칸반/스프린트 보드**: 드래그앤드롭으로 직관적인 태스크 관리, 칸반과 스프린트 모드 자유 전환. 5개의 프리셋 템플릿(개발팀, 디자인팀, 마케팅팀, 에이전시, 대학팀)으로 초기 설정 시간을 최소화한다. 스프린트 계획, 백로그 관리, 번다운 차트를 기본 제공한다.

- **이슈 트래킹**: 이슈 등록, 담당자 배정, 우선순위, 라벨, 마감일 관리를 최소한의 클릭으로 처리. 키보드 단축키로 빠른 이슈 등록, 마크다운 에디터로 상세 설명 작성, 이미지/파일 첨부를 지원한다. 이슈 간 의존성(블로커) 관계를 설정하여 병목을 시각화한다.

- **시간 추적**: 태스크별 타이머로 실제 소요 시간 기록, 프로젝트/멤버별 시간 투입 분석. 자동 추적 모드(브라우저 활동 기반)와 수동 입력 모드를 지원하며, 프리랜서 팀의 클라이언트 보고서 생성에 활용할 수 있다. Pomodoro 타이머도 내장하여 집중력 관리를 돕는다.

- **AI 자동 리포트**: GPT-4 API 기반으로 일간 진행 요약, 주간 성과 리포트, 스프린트 회고 자료를 AI가 자동 생성. 단순 통계 나열이 아니라 "이번 주 병목은 디자인 검수 단계", "김OO 워크로드 120% 과부하" 등 실행 가능한 인사이트를 제공한다.

- **Slack/GitHub 연동**: Slack에서 직접 이슈를 등록/업데이트하고, GitHub PR/커밋과 이슈를 자동 연결한다. Google Calendar와 연동하여 마감일을 캘린더에 자동 반영한다.

- **타임라인 뷰**: 간트 차트 형태의 타임라인 뷰로 전체 프로젝트 일정을 한눈에 파악한다. 이슈 간 의존성과 마감일을 시각적으로 표현하며, 일정 지연 시 자동 알림을 발송한다.

### AI 리포트 생성 기능 상세

TeamPulse의 핵심 차별화 기능인 AI 자동 리포트는 3가지 유형으로 제공된다.

| 리포트 유형 | 생성 주기 | 내용 | 전달 채널 |
|------------|----------|------|----------|
| 일간 진행 요약 | 매일 오전 9시 | 어제 완료 이슈, 오늘 예정 이슈, 블로커 현황 | Slack, 이메일 |
| 주간 성과 리포트 | 매주 월요일 | 완료율, 속도(velocity), 멤버별 기여도, 병목 분석 | PDF, Slack |
| 스프린트 회고 | 스프린트 종료 시 | 달성/미달성 목표, 개선점, 다음 스프린트 추천 | Notion, PDF |

AI 리포트는 단순 데이터 집계가 아니라, 패턴 분석을 통해 구체적인 개선 제안을 포함한다. 예를 들어 "지난 3개 스프린트 연속으로 백엔드 이슈가 오버플로우됨. 백엔드 리소스 보강 또는 스프린트 범위 축소 권고" 같은 actionable insight를 생성한다.

### 시장 성장 동력

원격/하이브리드 근무 확산과 프리랜서 경제 성장으로 소규모 팀 PM 도구 시장이 연 28% 성장하고 있다. 특히 한국어에 최적화된 경량 PM 도구에 대한 수요가 높지만, 적합한 제품이 부재한 상황이다.

| 성장 동력 | 상세 내용 | 영향도 |
|----------|----------|--------|
| 원격/하이브리드 근무 | 코로나 이후 원격 근무 정착, 비동기 협업 도구 수요 | 매우 높음 |
| 프리랜서 경제 성장 | 프리랜서/긱워커 460만명(2025), 프로젝트 기반 팀 증가 | 높음 |
| 스타트업 생태계 확대 | 연 2만개+ 신규 스타트업 설립, 초기 팀 관리 도구 필요 | 높음 |
| AI 생산성 도구 트렌드 | AI 자동화 리포트/분석에 대한 수요 급증 | 높음 |
| 한국어 도구 부재 | 기존 글로벌 PM 도구의 한국어 지원 미흡 | 중간 |
| SaaS 구매 문화 확산 | 국내 기업의 SaaS 도입 연 35% 증가 | 중간 |

| 리포트 작성 시간 절감 | 주당 평균 2시간 절약/팀 |
| 대학생 팀 무료 지원 | 2,000팀+ |

TeamPulse는 소규모 팀의 생산성 향상을 통해 스타트업 생태계 활성화에 기여한다. 특히 대학 프로젝트 팀과 초기 스타트업에 무료/저가로 PM 도구를 제공하여 디지털 리터러시와 체계적 프로젝트 관리 역량 향상에 기여한다. 프리랜서 경제 확산에 따른 분산된 팀의 효율적 협업을 지원하여, 위치와 고용 형태에 관계없이 누구나 체계적으로 프로젝트를 관리할 수 있는 환경을 만든다.
