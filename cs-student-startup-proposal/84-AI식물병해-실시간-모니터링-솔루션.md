## 1. 문제 인식

### 병해충으로 인한 농업 피해의 심각성

| 지표 | 수치 | 출처 |
|------|------|------|
| 연간 병해충 피해액 | 3.2조원 | 농촌진흥청 |
| 병해충 발견 지연 평균 기간 | 발생 후 7~14일 | 국립식물검역원 |
| 조기 발견 시 피해 감소율 | 60~80% | 농촌진흥청 |
| 농약 사용량 (OECD 평균 대비) | 2.3배 | OECD |
| 농약 비용 (농가 평균) | 연 350만원 | 한국농촌경제연구원 |
| 잔류 농약 기준 초과 농산물 | 3.2% | 식품의약품안전처 |
| 기후변화로 신종 병해 증가 | 연 12% | 국립농업과학원 |
| 농업인 병해 진단 역량 부족 | 57% | 농촌진흥청 |
| 식물 전문의 1인당 담당 면적 | 3,000ha | 국립농업과학원 |
| 스마트 방제 시스템 도입률 | 4.5% | 한국농업기술진흥원 |

병해충은 작물 수확량과 품질을 크게 좌우하지만, 대부분의 농가에서는 육안으로 이상 증상이 뚜렷해진 후에야 발견한다. 이 시점에서는 이미 감염이 상당히 진행되어 방제 효과가 떨어지고, 농약 사용량이 증가한다. 전문 식물병리학자의 수는 한정적이어서 농가가 즉시 진단을 받기 어렵다.

### 사용자 페인 포인트 분석

**시설원예 농가 (딸기):**
> *"잿빛곰팡이병이 하우스 전체로 퍼진 뒤에야 발견했습니다. 수확량의 40%를 잃었어요. 초기에 알았다면 국소 방제로 충분했을 텐데..."*

> *"비슷한 증상인데 탄저병인지 역병인지 구분이 안 됩니다. 잘못된 약을 뿌리면 돈만 낭비입니다."*

**과수원 농가:**
> *"사과 점무늬낙엽병을 겹무늬썩음병으로 오진해서 효과 없는 약제를 사용했습니다."*

**농업기술센터:**
> *"농가에서 사진을 찍어 보내면 전문가가 진단하는데, 답변까지 2~3일 걸립니다. 그 사이 병이 퍼집니다."*

**귀농 초보 농가:**
> *"식물 병을 진단할 지식이 없습니다. 스마트폰으로 찍으면 바로 알려주는 서비스가 있으면 좋겠어요."*

## 2. 솔루션

### 핵심 기능

| 기능 | 설명 | 기술 요소 |
|------|------|-----------|
| AI 병해 자동 진단 | 작물 이미지 촬영 시 병해 종류 자동 분류 (96.5% 정확도) | Vision Transformer (ViT) + EfficientNet |
| 초기 단계 탐지 | 인간 육안으로 보이지 않는 초기 감염 징후 포착 | 하이퍼스펙트럴/근적외선 이미지 분석 |
| 감염 확산 예측 | 현재 감염 범위에서 향후 확산 경로/속도 예측 | SIR 감염 모델 + 기상 데이터 |
| 맞춤 약제 처방 | 진단 결과에 따른 최적 약제, 희석 비율, 살포 시기 안내 | 약제 DB + 작물/병해 매칭 엔진 |
| 실시간 모니터링 | 하우스 내 AI 카메라가 24시간 작물 상태 촬영/분석 | 엣지 AI 카메라 + 스케줄 촬영 |
| 병해 이력 관리 | 농가별 병해 발생 이력, 방제 기록 자동 관리 | 타임라인 DB, 대시보드 |
| 모바일 진단 앱 | 스마트폰 카메라로 즉시 진단 (인터넷 없이도 가능) | On-device 경량 모델 (TFLite) |
| 지역 병해 경보 | 지역 단위 병해 발생 현황 및 확산 경보 | 크라우드소싱 데이터 + 기상 연동 |

### 기술 스택

| 구분 | 기술 | 선정 이유 |
|------|------|-----------|
| 이미지 분류 | Vision Transformer (ViT) + EfficientNet V2 | 높은 분류 정확도, 전이학습 효율 |
| 객체 탐지 | YOLOv8 | 병해 부위 실시간 탐지 및 위치 표시 |
| 엣지 AI | NVIDIA Jetson Nano + TFLite | 현장 실시간 추론, 오프라인 지원 |
| 데이터 증강 | Albumentations + Diffusion 합성 | 학습 데이터 확장, 소수 병해 대응 |
| 백엔드 | FastAPI + PostgreSQL | 진단 API, 이력 관리 |
| 모바일 | Flutter + TFLite | 크로스플랫폼, 온디바이스 추론 |
| 클라우드 | AWS (SageMaker, S3, Lambda) | 모델 학습/서빙, 이미지 저장 |
| GIS | Leaflet + PostGIS | 병해 발생 지도 시각화 |
| IoT | MQTT + Raspberry Pi | AI 카메라 제어, 데이터 전송 |
