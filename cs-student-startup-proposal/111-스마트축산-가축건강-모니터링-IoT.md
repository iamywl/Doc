## 3. 문제 인식

### 3.1 축산업 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 한우 사육 두수 (2025) | 약 350만 두 | 통계청 |
| 국내 젖소 사육 두수 | 약 39만 두 | 통계청 |
| 국내 돼지 사육 두수 | 약 1,100만 두 | 통계청 |
| 가축 질병으로 인한 연간 피해액 | 약 1조 2,000억원 | 농림축산식품부 |
| 구제역/ASF 발생 시 살처분 비용 | 건당 평균 3,000억원+ | 농림축산식품부 |
| 농가당 평균 수의사 방문 빈도 | 월 1~2회 | 대한수의사회 |
| 한우 송아지 폐사율 | 약 5.2% | 한우자조금 |
| 축산농가 고령화 (60세 이상) | 52.3% | 통계청 농업총조사 |
| 스마트축산 도입률 | 약 8.7% | 농촌진흥청 |
| 축산물 수출 목표 (2027) | 5억 달러 | 농식품부 |

### 3.2 이해관계자 인터뷰

> "소 한 마리가 갑자기 폐사하면 1,000만원 이상 손실입니다. 미리 알 수 있었다면 수의사를 불러 치료했을 텐데요."
> — 충북 한우 농장주 (사육 두수 150두)

> "농장 규모가 커지면서 개체별 관리가 불가능합니다. 500두를 사람 눈으로 일일이 확인할 수 없어요."
> — 경북 대규모 한우 농장 관리인

> "가축 질병을 초기에 발견하면 치료 비용이 1/10로 줄어듭니다. 하지만 대부분 증상이 뚜렷해진 뒤에야 발견합니다."
> — 충남 공공수의사

> "구제역, ASF 같은 법정 전염병은 조기 발견이 생사를 가릅니다. IoT로 실시간 모니터링하면 방역 대응이 훨씬 빨라질 겁니다."
> — 농림축산검역본부 방역 담당관

## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| 개체별 바이탈 모니터링 | 체온, 심박, 활동량, 반추량을 웨어러블 센서로 24시간 측정 |
| AI 질병 조기진단 | 바이탈 데이터 패턴 분석으로 질병 발생 3~5일 전 경보 |
| 행동 패턴 분석 | 음수량, 사료 섭취, 이동 패턴, 기립/횡와 비율 AI 분석 |
| 발정 감지 | 체온/활동량 변화 기반 발정 시기 자동 감지 (정확도 92%) |
| 분만 예측 | 체온/행동 변화로 분만 12시간 전 알림 |
| 수의사 원격 연계 | 개체 건강 데이터를 수의사에게 실시간 공유, 원격 상담 지원 |
| 축사 환경 모니터링 | 축사 내 온습도, 암모니아, CO2 자동 모니터링 및 환기 제어 |
| 이력 추적 관리 | 개체별 건강/치료/백신 이력 자동 관리, 축산물이력제 연동 |

### 4.3 기술 스택

| 분류 | 기술 | 용도 |
|------|------|------|
| IoT 센서 | MAX30102(심박), DS18B20(체온), ADXL345(가속도) | 가축 바이탈 데이터 수집 |
| 통신 | LoRaWAN, BLE 5.0 | 넓은 축사/방목지 저전력 장거리 통신 |

| 이상탐지 | Isolation Forest, DBSCAN | 비정상 바이탈/행동 감지 |
| 백엔드 | FastAPI, PostgreSQL, TimescaleDB | API, 관계형/시계열 데이터 |
| 프론트엔드 | Flutter (모바일), React (웹) | 농장주 앱, 수의사 포털 |
| 클라우드 | AWS IoT Core, Lambda | IoT 관리, 서버리스 처리 |
| 메시지 | MQTT, Kafka | 센서 데이터 스트리밍 |

## 5. 시장 분석

### 5.1 시장 규모 (TAM/SAM/SOM)

| 시장 구분 | 규모 | 산출 근거 |
|-----------|------|-----------|
| TAM (전체 시장) | 3조원 | 글로벌 스마트축산 기술 시장 중 아태 지역 |
| SAM (유효 시장) | 4,200억원 | 국내 축산 ICT 및 건강관리 솔루션 시장 |
| SOM (목표 시장) | 130억원 | 초기 3년 내 AI 가축건강 모니터링 점유 목표 |

### 5.2 경쟁사 분석

| 경쟁사 | 유형 | 강점 | 약점 | 차별점 |
|--------|------|------|------|--------|
| 아이씨티케이 (국내) | 축산 IoT | 시장 경험, 제품 라인업 | AI 분석 미약 | 데이터 수집 vs AI 진단 |
| Allflex (글로벌) | 이어태그 | 글로벌 점유율 | 고가, 한국 지원 미약 | 글로벌 vs 국내 최적화 |
| 스마트팜코리아 | 스마트팜 | 정부 지원, 인지도 | 축산 건강 특화 부족 | 환경관리 vs 건강관리 전문 |
| 씨앤팜 (국내) | 축산관리 SW | 농가 데이터 | IoT 없음, 수동 입력 | 수동 기록 vs 자동 IoT |
| 축산물품질평가원 | 공공 서비스 | 이력 추적 인프라 | 건강 모니터링 없음 | 이력관리 vs 건강 AI |
