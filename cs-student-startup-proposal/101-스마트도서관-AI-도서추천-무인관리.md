## 4. 솔루션

### 4.1 핵심 기능

**① AI 맞춤형 도서 추천**
- 대출 이력, 독서 패턴, 관심 분야 기반 하이브리드 추천 (협업필터링 + 콘텐츠 기반)
- 연령대, 직업, 시즌, 트렌드 반영 동적 추천
- "이 책을 읽은 사람이 다음에 읽은 책" 경로 추천

**② RFID 기반 무인 대출/반납**
- RFID 태그 + 자동 대출/반납기 통합 솔루션
- 모바일 앱 QR 기반 간편 대출
- 미반납 자동 알림 및 연체 관리

**③ 스마트 도서관 관리 대시보드**
- 실시간 좌석·열람실 이용 현황 (IoT 센서)
- 장서 활용도 분석 및 구입 추천 AI
- 이용자 통계, 인기 도서, 트렌드 리포트 자동 생성

**④ 독서 커뮤니티 플랫폼**
- AI 기반 독서 모임 매칭
- 독서 기록 및 리뷰 관리
- 도서관별 프로그램 참여 및 예약

### 4.3 기술 스택

| 구분 | 기술 | 선정 사유 |
|------|------|-----------|
| 추천 AI | Implicit (CF), SentenceTransformers (NLP) | 하이브리드 추천으로 정확도 향상 |
| NLP | KoBERT, KoGPT | 한국어 도서 콘텐츠 분석 |

| IoT | ESP32 + BLE 비콘 | 좌석 이용 현황 감지 |
| 프론트엔드 | React, Next.js | 관리 대시보드 및 웹 서비스 |
| 모바일 | Flutter | iOS/Android 크로스 플랫폼 |
| 인프라 | AWS (EC2, RDS, S3) | 안정적 클라우드 인프라 |
| 데이터 분석 | Apache Airflow, Pandas | 주기적 데이터 분석 파이프라인 |
