## 2. 창업 아이템 개요

| 항목 | 내용 |
|------|------|
| 아이템명 | 탭커넥트 (TapConnect) |

## 3. 문제 인식

### 3.1 명함/네트워킹 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 연간 종이 명함 인쇄량 | 약 100억 장 | 대한인쇄정보산업협동조합 (2025) |
| 명함 교환 후 연락하는 비율 | 11.3% | 한국경영학회 조사 (2025) |
| 받은 명함 분실/방치 비율 | 약 87% | 잡코리아 직장인 설문 (2025) |
| 연간 비즈니스 네트워킹 행사 수 | 약 4만 2천 건 | MICE 산업통계 (2025) |
| MZ세대 직장인 디지털 명함 관심도 | 76.8% | 사람인 설문 (2025) |
| 프리랜서/1인 기업 수 | 약 890만 명 | 통계청 (2025) |
| 영업직 종사자의 명함 관리 불편 응답률 | 82.4% | 한국세일즈포스 조사 (2024) |

## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| 📲 NFC 탭 공유 | NFC 카드를 상대방 폰에 터치하면 디지털 프로필 즉시 표시 (앱 불필요) |
| 🌐 디지털 프로필 | 이름, 직함, 연락처, SNS, 포트폴리오를 하나의 프로필 페이지에 통합 |
| 🔄 실시간 업데이트 | 프로필 정보 변경 시 이미 공유된 링크도 자동 업데이트 |

| 🤖 AI 후속 관리 | 교환 후 적절한 타이밍에 후속 연락 알림 및 메시지 초안 제안 |
| 🎪 이벤트 모드 | 컨퍼런스/전시회 전용 대량 네트워킹 + 참가자 매칭 |
| 🏢 팀 관리 | 기업 직원 명함 일괄 관리, 브랜드 통일, CRM 연동 |
| 🎨 카드 커스터마이징 | NFC 카드 디자인 맞춤 제작 (색상, 로고, 소재) |
