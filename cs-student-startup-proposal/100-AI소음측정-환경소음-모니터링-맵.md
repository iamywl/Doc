## 3. 문제 인식

### 3.1 환경 소음 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 소음 관련 민원 건수 | 연간 약 18만 건 (환경 민원 1위) | 환경부 (2025) |
| 층간소음 민원 건수 | 연간 약 6.5만 건 | 한국환경공단 (2025) |
| 공사장 소음 기준 초과율 | 약 38% | 국립환경과학원 (2025) |
| 소음으로 인한 수면 장애 비율 | 성인의 23.5% | 국민건강영양조사 (2025) |
| 소음 관련 건강 비용 | 연간 약 1.8조원 | 한국환경정책평가연구원 (2025) |
| 도시 소음 기준 초과 지역 | 전국 주거지역의 31% | 환경부 소음지도 (2025) |
| 소음 측정 인프라 | 전국 측정소 320개소 (부족) | 국립환경과학원 (2025) |
| 부동산 거래 시 소음 정보 제공 | 5% 미만 | 한국부동산원 (2025) |
| 소음 민원 처리 평균 기간 | 약 45일 | 국민권익위원회 (2025) |
| AI 소음 모니터링 도입 지자체 | 전체의 8% 미만 | 환경부 (2025) |

## 4. 솔루션

### 4.1 핵심 기능

**① AI 소음원 자동 분류**
- 딥러닝 기반 12종 소음원 자동 분류 (교통, 공사, 생활, 유흥, 항공 등)
- 94% 이상 분류 정확도
- 시간대별 소음 패턴 분석 및 주요 원인 리포트

**② 실시간 소음지도**
- GIS 기반 전국 소음지도 실시간 시각화
- 크라우드소싱 + IoT 센서 하이브리드 데이터 수집
- 시간대별, 소음원별 필터링 및 히스토리

**③ 스마트 민원 시스템**
- 소음 기준 초과 시 자동 기록 및 증거 생성
- 관할 기관에 자동 민원 접수 연동
- 민원 처리 현황 추적 및 알림

**④ 부동산/건설 연계 서비스**
- 부동산 매물별 소음 등급 정보 제공 API
- 건설 현장 소음 실시간 모니터링 및 규제 준수 관리
- 아파트 층간소음 상시 모니터링
