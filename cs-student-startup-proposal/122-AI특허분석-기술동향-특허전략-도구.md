## 3. 문제 인식

### 3.1 특허 분석 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 연간 특허 출원 건수 | 약 24만 건 | 특허청 (2025) |
| 중소기업 특허 분쟁 연간 피해액 | 약 2조 3,000억 원 | 중소벤처기업부 (2025) |
| 특허 분석 없이 출원하는 비율 (스타트업) | 약 62% | 한국지식재산연구원 (2025) |
| 특허 전문 분석 비용 (1건) | 300~1,000만 원 | 대한변리사회 (2025) |
| 특허 무효화 비율 | 약 31.4% | 특허심판원 (2025) |

| AI/SW 특허 출원 증가율 (전년 대비) | +28.3% | 특허청 (2025) |

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| 🔍 AI 선행기술 조사 | 자연어로 기술 설명 입력 시 관련 특허 자동 검색·분류 |
| 📊 특허 맵 생성 | 기술 분야별 특허 분포를 2D/3D 맵으로 자동 시각화 |
