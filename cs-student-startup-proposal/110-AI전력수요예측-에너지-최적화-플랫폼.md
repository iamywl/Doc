## 3. 문제 인식

### 3.1 국내 에너지 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 연간 전력소비량 (2025) | 약 5,800억 kWh | 한국전력공사 |
| 산업용 전력 비중 | 전체의 52.3% | 에너지통계연보 |
| 건물 에너지 비중 | 전체의 21.7% | 에너지관리공단 |
| 전력 피크 관리 실패 비용 (연간) | 약 4조원 | 전력거래소 |
| 기업 에너지비용 비중 (제조업) | 매출의 3~8% | 한국에너지공단 |
| 전력요금 인상률 (최근 3년) | 누적 약 32% | 한국전력공사 |
| RE100 참여 국내 기업 | 약 95개사 | RE100 이니셔티브 |
| 탄소배출권 가격 | 톤당 약 2.5만원 | 한국거래소 |
| 에너지 관리 시스템 도입률 (중소기업) | 약 12% | 에너지관리공단 |
| 전력 수요예측 오차율 (기존) | 평균 8~15% | 전력거래소 |

### 3.2 이해관계자 인터뷰

> "전기요금이 3년 사이 30% 이상 올랐는데, 어디서 전력을 줄일 수 있는지 분석할 도구가 없습니다."
> — 경기도 중견 제조업체 시설관리 팀장

> "피크시간대 전력 사용을 분산하면 요금을 크게 줄일 수 있는데, 수요예측이 안 되니 사후 대응만 합니다."
> — 대형 쇼핑몰 에너지 관리 담당자

> "RE100 참여를 선언했지만, 우리 공장의 정확한 에너지 사용 패턴도 파악하지 못하고 있습니다."
> — 대기업 ESG 담당 임원

> "중소기업이 사용할 수 있는 합리적 가격의 에너지 관리 솔루션이 거의 없습니다."
> — 에너지관리공단 중소기업 지원 담당

## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 전력수요예측 | 날씨, 생산계획, 과거패턴 등 다변량 데이터로 24시간~7일 전력수요 예측 (정확도 97%) |
| 피크 부하 관리 | 피크 시간대 예측 및 부하 분산 스케줄링으로 기본요금 절감 |
| 설비 운전 최적화 | 강화학습 기반 HVAC, 조명, 생산설비 최적 운전 스케줄 자동 생성 |
| 에너지 이상 탐지 | 전력 사용 패턴 이상 실시간 감지, 설비 고장/누전 조기 경고 |
| 탄소 배출 관리 | 전력 소비 기반 탄소 배출량 자동 산출, RE100/탄소배출권 관리 |
| 전력시장 대응 | 전력 거래 시장 가격 예측, 최적 매매 시점 추천 |
| 비용 시뮬레이션 | 요금제 변경, 설비 교체, 재생에너지 도입 시 비용 시뮬레이션 |
| 다사업장 통합관리 | 여러 사업장의 에너지 현황 통합 대시보드, 벤치마킹 |

| 시장 구분 | 규모 | 산출 근거 |

| SAM (유효 시장) | 8,500억원 | 국내 에너지 관리 솔루션 및 수요관리 시장 |
| SOM (목표 시장) | 250억원 | 초기 3년 내 AI 에너지 최적화 SaaS 점유 목표 |

### 5.2 경쟁사 분석

| 경쟁사 | 유형 | 강점 | 약점 | 차별점 |
|--------|------|------|------|--------|
| 에너지엑스 (국내) | BEMS | 국내 시장 점유, 건물 특화 | AI 예측 미약, 건물 한정 | 건물 한정 vs 산업+건물 통합 |
| 인코어드테크 (국내) | 에너지 AI | AI 기술력, 한전 협업 | 대기업 중심, 고가 | 대기업 전용 vs 중소기업 포함 |
| Schneider Electric | 글로벌 EMS | 글로벌 브랜드, 하드웨어 | 고가, 커스터마이징 복잡 | 고가 HW+SW vs 클라우드 SaaS |
| 그리드위즈 (국내) | DR 사업자 | 수요관리 인프라 | SaaS 미약, 예측 한정 | DR 수행 vs 통합 최적화 |
| 자체 EMS | 사내 시스템 | 맞춤형 | 개발/유지비 고가, AI 없음 | 고비용 vs SaaS 합리적 가격 |
