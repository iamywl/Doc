## 4. 솔루션

### 4.1 핵심 기능

#### 기능 1: AI 동작 분석 및 실시간 자세 교정
- 스마트폰/웹캠으로 사용자의 재활 운동 동작을 실시간 촬영
- MediaPipe + 자체 학습 모델로 33개 관절 포인트 추적
- 올바른 동작 대비 각도, 속도, 범위를 분석하여 즉시 피드백
- 음성 가이드로 "무릎을 5도 더 구부려주세요" 같은 구체적 교정

#### 기능 2: 맞춤형 재활 프로그램 생성
- 진단명, 수술 유형, 회복 단계에 따른 AI 자동 프로그램 설계
- 물리치료사가 원격으로 프로그램 조정 가능
- 통증 수준, 운동 수행도에 따른 난이도 자동 조절
- 500개 이상의 재활 운동 라이브러리

#### 기능 3: 원격 치료사-환자 화상 상담
- 1:1 화상 재활 세션 (화면 공유로 동작 지도)
- 치료사가 환자의 운동 데이터를 실시간 모니터링
- 예약/결제 시스템 통합
- 다중 환자 동시 모니터링 대시보드

#### 기능 4: 치료 진행도 분석 대시보드
- 운동 수행률, 정확도 추이 그래프
- 관절 가동 범위(ROM) 변화 추적
- 통증 일지 및 회복 지표 시각화
- 병원/보험사 제출용 리포트 자동 생성

- 보호자 부담 경감 (통원 시간/비용 절감)
