## 3. 문제 인식

### 3.1 국내 화재 현황 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| 연간 화재 발생 건수 (2025) | 약 42,000건 | 소방청 |
| 화재로 인한 연간 사망자 | 약 310명 | 소방청 화재통계 |
| 화재로 인한 연간 재산피해 | 약 6,500억원 | 소방청 |
| 전기 화재 비율 | 전체 화재의 28.3% | 한국전기안전공사 |
| 노후 건축물(30년 이상) 비율 | 전체 건축물의 37.2% | 국토교통부 |
| 화재 감지기 오경보율 | 약 72% | 한국소방안전원 |
| 화재 초기 진압 실패율 | 43.7% | 소방청 |
| 건물 화재 인명피해 비율 | 전체 화재 사망의 67% | 소방청 |
| 화재 발견~신고 평균 시간 | 약 8.2분 | 소방청 |
| 소방차 도착 평균 시간 | 약 7.1분 | 소방청 |

### 3.2 이해관계자 인터뷰

> "건물 내 화재 감지기가 요리 연기에도 울려서 입주민들이 경보를 무시하게 되었습니다. 정작 진짜 화재 때는 아무도 대피하지 않았습니다."
> — 서울시 강남구 상가건물 관리소장 (경력 15년)

> "30년 넘은 건물의 전기배선 상태를 하나하나 점검하기 어렵습니다. AI로 이상 징후를 미리 감지할 수 있다면 엄청난 도움이 될 것입니다."
> — 경기도 소방서 화재예방과 소방관

> "노후 전통시장에서 매년 화재가 발생하는데, 사전 예방이 핵심입니다. 기존 감지기는 화재가 이미 발생한 후에야 작동합니다."
> — 전통시장활성화추진단 담당자

> "건물 화재보험료가 매년 올라가고 있어요. 화재 예방 시스템이 인증되면 보험료 할인도 가능할 텐데요."
> — 대형 상업시설 운영팀장

## 4. 솔루션

### 4.1 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 전기 이상 감지 | 전기 아크, 과부하, 누전 등 전기 화재 전조 징후를 AI로 실시간 분석 |
| 열화상 AI 분석 | 건물 내 열화상 카메라 영상을 딥러닝으로 분석하여 과열 지점 자동 탐지 |
| 복합 가스 센서 융합 | CO, CO2, VOC, 미세 입자 등 다중 가스 센서 데이터를 AI로 융합 분석 |
| 예측형 경보 | 화재 발생 30분~2시간 전 3단계 예측 경보 (주의/경고/위험) |
| 스마트 대피 안내 | 건물 구조와 화재 위치를 고려한 최적 대피 경로 실시간 생성 |
| 소방 연계 자동 신고 | 위험 단계 도달 시 119 자동 신고 및 건물 정보 자동 전송 |
| 건물 이력 관리 | 센서 데이터 기반 건물 안전 이력 관리 및 위험도 점수화 |
| IoT 디바이스 자가진단 | 센서 및 디바이스 상태 자동 모니터링, 고장 시 즉시 알림 |

## 5. 시장 분석

### 5.1 시장 규모 (TAM/SAM/SOM)

| 시장 구분 | 규모 | 산출 근거 |
|-----------|------|-----------|
| TAM (전체 시장) | 2조 5,000억원 | 글로벌 화재 감지 및 예방 시장 중 아시아 시장 |
| SAM (유효 시장) | 4,500억원 | 국내 건물 화재 안전관리 솔루션 시장 |
| SOM (목표 시장) | 180억원 | 초기 3년 내 AI 화재 예측 솔루션 점유 목표 |

### 5.2 경쟁사 분석

| 경쟁사 | 유형 | 강점 | 약점 | 차별점 |
|--------|------|------|------|--------|
| 기존 화재감지기 (동양엘리베이터 등) | 하드웨어 | 시장 점유율, 인증 | 감지만 가능, 예측 불가 | 사후 감지 vs 사전 예측 |
| 아이파이어 | 영상 AI | 불꽃 감지 기술 | 발화 후 감지, 실외 중심 | 발화 전 예측 vs 발화 후 감지 |
