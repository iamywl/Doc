## 3. 문제 인식

### 3-1. 시장 현황 및 통계

| 통계 항목 | 수치 | 출처 |
|-----------|------|------|
| 전국 어린이놀이시설 수 | 78,462개소 | 행정안전부 (2025) |
| 놀이시설 안전사고 (연간) | 26,847건 | 한국소비자원 (2024) |
| 중대 부상 사고 비율 | 12.3% (골절 이상) | 소방청 통계 |

> "아파트 놀이터를 관리하라는데 인력이 부족합니다. 사고가 나면 관리 책임이 우리한테 돌아와요."
> — 경기 용인시 아파트 관리소장 이OO

> "놀이시설 안전점검을 연 2회 하지만, 그 사이에 노후화되거나 파손되는 기구가 많습니다."
> — 대전광역시 놀이시설 안전관리자 조OO

> "아이들이 놀이터에서 놀 때 항상 불안합니다. 잠깐 눈을 뗀 사이에 일어나는 사고가 대부분이에요."
> — 인천 부평구 학부모 한OO (자녀 4세, 7세)

## 4. 솔루션

### 4-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| AI 낙상 감지 | 엣지 AI 카메라로 아동의 낙상·추락 상황 실시간 감지 |
| 충돌 위험 알림 | 놀이기구 주변 아동 밀집도 분석 및 충돌 위험 사전 경고 |
| 이탈 감지 | 놀이터 경계 밖 이동 아동 자동 감지 및 보호자 알림 |
| 놀이기구 상태 모니터링 | IoT 센서(진동, 기울기, 하중)로 기구 안전 상태 실시간 감시 |
| 보호자 앱 알림 | 실시간 푸시 알림, 아동 위치 확인, 사고 영상 즉시 전송 |
| 환경 모니터링 | 기온, 자외선, 미세먼지 측정 및 놀이 적합도 안내 |
| 안전 통계 대시보드 | 사고 패턴 분석, 위험 시간대 파악, 월간 안전 리포트 |
| 자동 신고 시스템 | 중대 사고 감지 시 119 자동 신고 및 현장 정보 전송 |

### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 엣지 AI | NVIDIA Jetson Orin Nano | 현장 실시간 AI 추론 |
| 비전 모델 | YOLOv8 + MediaPipe | 객체 탐지 및 자세 추정 |
| 낙상 감지 | ST-GCN (시공간 그래프 CNN) | 낙상·이상행동 분류 |
| IoT 프로토콜 | MQTT + LoRaWAN | 센서 데이터 저전력 전송 |
| 백엔드 | Go + gRPC | 고성능 실시간 이벤트 처리 |
| 데이터베이스 | InfluxDB + PostgreSQL | 시계열 센서 + 관계형 데이터 |
| 프론트엔드 | Vue.js 3 + Leaflet | 관제 대시보드·지도 시각화 |
| 모바일 | Flutter | 보호자 앱 (iOS/Android) |
| 클라우드 | NCloud (네이버 클라우드) | 공공 클라우드 인프라 |
| 알림 | Firebase Cloud Messaging | 실시간 푸시 알림 |

## 9. 사회적 가치

### 9-1. 아동 안전 보호

연간 26,847건의 놀이시설 사고 중 AI 실시간 감지를 통해 중대사고를 50% 이상 예방하는 것을 목표로 합니다. 골든타임 내 구조율을 현재 65.2%에서 95% 이상으로 향상시킬 수 있습니다.

### 9-2. 보호자 양육 부담 경감

맞벌이 가정이 64.2%에 달하는 상황에서, 아이가 놀이터에서 안전하게 놀 수 있다는 신뢰를 제공하여 양육 스트레스를 완화합니다.

### 9-3. 지자체 안전관리 효율화

수동 점검에 의존하던 놀이시설 안전관리를 IoT 기반 자동화로 전환하여, 제한된 인력과 예산으로도 효과적인 안전관리가 가능합니다.

### 9-4. 저출산 시대 아동 친화 환경

안전한 놀이 환경 구축은 아동 친화 도시 조성에 기여하며, 저출산 시대에 아이 키우기 좋은 사회 환경을 만드는 데 일조합니다.
