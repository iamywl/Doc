### 4.1 제품 개요

메디박스는 시간대별 자동 약 분배, 복약 알림(음성/LED/진동), 복약 여부 자동 체크, 보호자 실시간 알림, 약물 상호작용 검증을 통합한 IoT 스마트 약상자입니다.

| 기능 | 설명 |

| 데이터 보안 | AES-256, ISMS-P, PHI 비식별화 |
| 모니터링 | Grafana + CloudWatch |
