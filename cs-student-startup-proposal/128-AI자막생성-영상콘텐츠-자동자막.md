|------|------|

### 4-3. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 음성 인식 | OpenAI Whisper (large-v3) fine-tuned | 한국어 특화 STT |
| 화자 분리 | pyannote.audio | 다중 화자 식별 |
| 음성 분리 | Demucs | BGM/노이즈 분리 |
| 번역 | LLM API + 자체 NMT | 문맥 기반 다국어 번역 |
| 백엔드 | FastAPI + Celery | 비동기 작업 처리 |
| 프론트엔드 | Next.js + TypeScript | 웹 앱 및 자막 에디터 |
| 데이터베이스 | PostgreSQL + Redis | 작업 큐, 사용자 데이터 |
| 영상 처리 | FFmpeg + MoviePy | 영상 인코딩, 자막 하드코딩 |
| 인프라 | AWS (EC2 GPU + S3 + Lambda) | 클라우드 GPU 추론 |
| CI/CD | GitHub Actions + Docker | 자동 배포 파이프라인 |

### 9-4. 교육 형평성

온라인 강의의 자막 자동 생성으로 청각장애 학생, 외국인 유학생 등 교육 취약계층의 학습 접근성을 높입니다.
