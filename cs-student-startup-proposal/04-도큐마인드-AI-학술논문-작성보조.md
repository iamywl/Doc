## 창업 아이템 개요

### 한줄 소개

> **"AI가 함께 쓰는 학술 글쓰기 — 검색부터 인용, 표절 검사까지 원스톱"**

도큐마인드는 RAG(Retrieval Augmented Generation) 기술과 학술 인용 그래프(Citation Graph)를 결합하여, 한국 대학생이 논문·보고서를 작성할 때 겪는 **문헌 검색, 구조화된 글쓰기, 참고문헌 관리, 표절 방지**를 하나의 플랫폼에서 해결하는 AI 학술 글쓰기 보조 서비스이다.

### 핵심 기능 4가지

| # | 기능 | 설명 | 기술 기반 |
|:---:|:---|:---|:---|
| 1 | **스마트 문헌 검색** | 연구 주제 입력 시 관련 논문·자료를 자동 검색·요약 | RAG + Semantic Search |
| 2 | **AI 구조화 글쓰기** | 서론-본론-결론 구조 제안, 문단별 AI 코칭 | Fine-tuned Korean Academic LLM |
| 3 | **자동 인용·참고문헌** | 드래그앤드롭으로 인용 삽입, APA/MLA/Chicago 자동 생성 | Citation Graph + CSL Engine |
| 4 | **실시간 표절·윤리 검사** | 작성 중 표절률 실시간 모니터링, AI 생성 텍스트 탐지 | Fingerprinting + AI Detector |

## 1. 문제 인식

### 1.1 학술 글쓰기의 구조적 어려움

대한민국 대학생 약 **328만명**, 대학원생 약 **33만명**이 매 학기 다수의 리포트와 논문을 작성한다 [1]. 대학내일 20대연구소(2024)에 따르면, 대학생 학업 스트레스 요인 중 **"리포트/논문 작성"이 시험 다음으로 2위**를 차지했다 [4]. 이는 단순 글쓰기가 아닌, 문헌 검색 → 비판적 분석 → 구조화 → 인용 → 교정이라는 복합적 프로세스 때문이다.

#### 학술 글쓰기 프로세스별 Pain Point

| 단계 | Pain Point | 소요 시간 비중 | 스트레스 강도 |
|:---|:---|:---:|:---:|
| **1. 주제 선정** | 방향 설정 어려움, 선행연구 파악 부족 | 10% | ★★★☆☆ |
| **2. 문헌 검색/수집** | 적절한 논문 찾기 어려움, RISS/KCI 사용법 미숙 | 25% | ★★★★☆ |
| **3. 문헌 분석/정리** | 핵심 내용 추출, 비교·분석 역량 부족 | 20% | ★★★★★ |
| **4. 구조화/초안 작성** | 서론-본론-결론 논리 구성 어려움 | 20% | ★★★★☆ |
| **5. 인용/참고문헌 관리** | 인용 형식(APA 등) 혼동, 수작업 비효율 | 15% | ★★★★☆ |
| **6. 표절 검사/교정** | 의도치 않은 표절, 자기표절 인식 부족 | 10% | ★★★☆☆ |

> **핵심 발견**: 전체 프로세스의 약 **60%가 "검색·분석·인용" 단계**에 소요되며, 이 영역이야말로 AI가 가장 효과적으로 보조할 수 있는 지점이다.

### 1.2 표절 문제의 심각성

한국연구재단에 따르면 대학 내 표절 적발 건수는 매년 약 **15%씩 증가**하고 있다 [5]. 이는 학생들의 윤리 의식 부재뿐 아니라, **올바른 인용 방법을 배울 기회가 부족**한 구조적 문제에 기인한다.

| 표절 유형 | 비율 | 주요 원인 |
|:---|:---:|:---|
| **직접 복사(Copy-Paste)** | 38% | 인용 표기법 미숙, 시간 부족 |
| **패러프레이징 실패** | 27% | 바꿔 쓰기 역량 부족, 원문 이해 미흡 |
| **자기 표절** | 18% | 기존 과제 재활용, 자기인용 규정 무지 |
| **출처 미기재** | 12% | 참고문헌 관리 부재 |
| **AI 생성물 무비판 사용** | 5% | ChatGPT 등 AI 도구 윤리 기준 미비 |

출처: 한국연구재단 학술윤리백서(2024) [5], 교육부 학사관리 실태조사(2024) [7]

## 2. 실현 가능성

### 2.1 핵심 기술 3요소

도큐마인드의 기술적 차별화는 다음 세 가지 핵심 기술의 결합에 있다.

#### 기술 1: RAG (Retrieval Augmented Generation)

| 구분 | 내용 |
|:---|:---|
| **정의** | 외부 지식베이스에서 관련 문서를 검색(Retrieve)한 뒤, 이를 LLM의 컨텍스트로 주입하여 생성(Generate)하는 기법 [11] |
| **할루시네이션 해결** | 기존 ChatGPT의 가장 큰 문제인 "그럴듯하지만 틀린 정보 생성"을 실제 논문 데이터 기반으로 해결 |
| **구현 방법** | RISS/KCI/Google Scholar API → Vector DB(Milvus/Chroma) 색인 → 질의 시 Top-K 유사 문서 검색 → LLM 프롬프트 주입 |
| **핵심 지표** | Retrieval Accuracy 95%+, Hallucination Rate <3% (SQuAD 벤치마크 기준) |
| **컴공 학생 구현 난이도** | ★★★☆☆ (LangChain/LlamaIndex 프레임워크 활용) |

#### 기술 2: Citation Graph (인용 그래프)

| 구분 | 내용 |
|:---|:---|
| **정의** | 논문 간 인용-피인용 관계를 그래프 구조로 모델링하여 핵심 논문·연구 흐름 파악 |
| **활용** | 사용자가 입력한 주제에서 가장 영향력 있는 논문(High PageRank) 자동 추천 |
| **데이터** | Semantic Scholar API(무료, 200M+ 논문), CrossRef API, KCI Open API |
| **구현 방법** | Neo4j 그래프DB + PageRank 알고리즘 + Community Detection |
| **컴공 학생 구현 난이도** | ★★★☆☆ (그래프 알고리즘 수업 프로젝트 확장) |

#### 기술 3: Fine-tuned Korean Academic LLM

| 구분 | 내용 |
|:---|:---|
| **정의** | 한국어 학술 문체에 특화된 경량 언어모델 Fine-tuning |
| **베이스 모델** | Solar-10.7B 또는 EXAONE-7.8B (한국어 성능 우수 오픈소스) |
| **Fine-tuning 데이터** | KCI 등재 논문 초록 50만건 + 학위논문 10만건 (공개 데이터) |
| **기법** | QLoRA (4bit Quantization + LoRA) → 단일 GPU(A100 40GB)에서 학습 가능 |
| **출력** | 학술 문체 생성, 논리적 문단 구조 제안, 한국어 패러프레이징 |
| **컴공 학생 구현 난이도** | ★★★★☆ (HuggingFace + PEFT 라이브러리 활용) |

| **RAG 프레임워크** | LangChain + LlamaIndex | 가장 성숙한 RAG 생태계, 활발한 커뮤니티 |
| **LLM Serving** | vLLM | PagedAttention으로 높은 throughput, 오픈소스 |
| **벡터 DB** | Milvus (or ChromaDB for MVP) | 대규모 벡터 검색 최적화, 오픈소스 |
| **그래프 DB** | Neo4j Community Edition | Citation Graph 모델링 최적, 무료 |
| **RDB** | PostgreSQL 16 | JSON 지원, Full-text Search, 안정성 |
| **캐시** | Redis | 세션 관리, API 응답 캐시 |
| **인프라** | Docker + Docker Compose (MVP) | 로컬 개발 환경 통일, 배포 용이 |

| **CI/CD** | GitHub Actions | 무료, GitHub 생태계 통합 |

| **Should** | AI 텍스트 탐지 | 학술 윤리 트렌드 부합 |

### 3.2 TAM-SAM-SOM 분석

| 구분 | 정의 | 산출 | 규모 |
|:---|:---|:---|---:|
| **TAM** | 글로벌 AI Writing + 표절 검사 시장 | $6.5B + $1.2B [3][8] | **$7.7B** (약 10조원) |
| **SAM** | 국내 대학(원)생 학술 글쓰기 시장 | 361만명 × ₩59,000/년 (연간 ARPU) | **약 2,130억원** |
| **SOM** | 3년 내 확보 가능 시장 | SAM의 3% (약 10.8만명) | **약 64억원** |

> **SAM 산출 근거**: 국내 대학생 328만 + 대학원생 33만 = 361만명 [1]. 연간 ARPU는 Pro 구독 기준 ₩4,900 × 12개월 = ₩58,800 ≈ ₩59,000. 단, 무료 사용자 포함 blended ARPU 적용.

> **SOM 산출 근거**: 국내 에듀테크 스타트업의 대학생 시장 평균 침투율 2~5% (1~3년) [2]. 보수적으로 3% 적용. 10.8만명 × ₩59,000 = 약 64억원.

## 5. 사회적 가치

### 5.1 사회적 가치 프레임워크

도큐마인드는 예비창업패키지 **사회문제해결형**에 부합하는 다음 세 가지 사회적 가치를 창출한다.

| 사회적 가치 | 관련 SDGs | 측정 지표 |
|:---|:---:|:---|
| **교육 격차 해소** | SDG 4 (양질의 교육) | 비수도권 대학생 접근성, 무료 Tier 이용률 |
| **연구 윤리 제고** | SDG 16 (정의와 제도) | 표절 적발률 감소, 올바른 인용 사용률 증가 |
| **디지털 리터러시 향상** | SDG 9 (혁신과 인프라) | AI 도구 윤리적 활용 교육 효과 |

### 5.2 교육 격차 해소

#### 문제 현황

| 지표 | 수도권 대학 | 비수도권 대학 | 격차 |
|:---|:---:|:---:|:---:|
| 학교 도서관 학술 DB 보유 수 | 평균 15개 | 평균 7개 | **2.1배** |
| 학술 글쓰기 교과목 개설 비율 | 78% | 43% | **1.8배** |
| 논문 작성 튜터링 프로그램 | 65% | 28% | **2.3배** |
| 학생 1인당 학술 지원 예산 | 42만원 | 18만원 | **2.3배** |

출처: 한국대학교육협의회(2024) [12], 교육부 대학정보공시(2024) [9]

#### DocuMind의 해결 방안

- **무료 Tier 제공**: 모든 대학생에게 기본 문헌 검색·인용 기능 무료 제공
- **비수도권 대학 할인**: Campus 라이선스 30% 할인 (비수도권 특별가)
- **학술 글쓰기 가이드**: AI 코칭 모드를 통한 체계적 학술 글쓰기 교육
- **RISS/KCI 통합 접근**: 단일 인터페이스에서 모든 학술 DB 접근 가능

### 5.3 연구 윤리 제고

#### 기대 효과

| 항목 | 현재 (도입 전) | 목표 (도입 3년 후) | 개선 효과 |
|:---|:---:|:---:|:---:|
| 의도치 않은 표절 비율 | 32% | 12% | **-62.5%** |
| 올바른 인용 사용률 | 45% | 80% | **+77.8%** |
| 자기표절 인식률 | 28% | 75% | **+167.9%** |
| AI 생성물 윤리적 사용률 | 15% | 60% | **+300%** |

> 도큐마인드는 단순 "표절 검사" 도구가 아니라, **작성 과정에서 올바른 인용 습관을 형성**하는 교육적 도구이다. "검사 후 적발"이 아닌 "작성 중 예방" 패러다임으로 전환하여 학술 윤리 문화를 근본적으로 개선한다.
