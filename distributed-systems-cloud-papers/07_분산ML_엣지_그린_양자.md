# 분산 ML, 엣지 컴퓨팅, 그린 컴퓨팅, 양자 클라우드 핵심 논문 모음

> 총 40편의 핵심 논문을 7개 대분류, 15개 소분류로 정리

---

## 1. 분산 머신러닝 학습 (Distributed ML Training)

### 1.1 서베이 논문

#### [1] A Survey on Distributed Machine Learning
- **저자**: Joost Verbraeken, Matthijs Baez, Wannes Meert, Cees Snoek, Max Welling, Jan van Rijn
- **연도/학회**: 2020, ACM Computing Surveys
- **요약**: 분산 머신러닝의 전반적인 기술 현황을 포괄적으로 정리한 서베이 논문. 중앙 집중식 대비 분산 ML의 도전과 기회, 사용 기술, 가용 시스템을 체계적으로 분류하여 설명한다.
- **중요도**: 분산 ML 입문자가 전체 그림을 파악하기 위한 필수 참고 논문. 데이터 병렬화, 모델 병렬화, 통신 전략 등 모든 핵심 개념을 아우른다.

#### [2] Communication-Efficient Distributed Deep Learning: A Comprehensive Survey
- **저자**: Zhenheng Tang, Shaohuai Shi, Xiaowen Chu, Wei Wang, Bo Li
- **연도/학회**: 2020, arXiv (2003.06307)
- **요약**: 분산 딥러닝에서 통신 효율성을 높이는 알고리즘을 종합적으로 정리. 통신 동기화, 시스템 아키텍처, 압축 기술, 통신-연산 병렬화의 4개 차원으로 분류한다.
- **중요도**: 분산 학습에서 가장 큰 병목인 통신 오버헤드 해결 방법을 체계적으로 이해하는 데 핵심적인 논문.

#### [3] Distributed Training of Large Language Models: A Survey
- **저자**: Various authors
- **연도/학회**: 2025, ScienceDirect
- **요약**: 대규모 언어 모델(LLM) 분산 학습의 기반 기술을 심층 분석. 데이터 병렬화, 모델 병렬화, 3D 병렬화 전략과 프레임워크, 최적화 기법을 포괄적으로 다룬다.
- **중요도**: LLM 시대에 맞춘 최신 분산 학습 기술 동향을 파악하는 데 필수적인 최신 서베이.

### 1.2 Parameter Server

#### [4] Scaling Distributed Machine Learning with the Parameter Server
- **저자**: Mu Li, David G. Andersen, Jun Woo Park, Alexander J. Smola, Amr Ahmed, Vanja Josifovski, James Long, Eugene J. Shekita, Bor-Yiing Su
- **연도/학회**: 2014, OSDI (USENIX)
- **요약**: 파라미터 서버 프레임워크의 기념비적 논문. 워커 노드에 데이터와 워크로드를 분산하고, 서버 노드가 전역 공유 파라미터를 관리하며, 비동기 데이터 통신과 유연한 일관성 모델을 지원한다.
- **중요도**: 분산 ML 시스템 아키텍처의 근간이 된 논문. 이후 등장한 거의 모든 분산 학습 프레임워크의 설계 철학에 영향을 주었다.

#### [5] Communication Efficient Distributed Machine Learning with the Parameter Server
- **저자**: Mu Li, David G. Andersen, Alexander Smola, Kai Yu
- **연도/학회**: 2014, NeurIPS
- **요약**: 3세대 파라미터 서버 프레임워크를 제안하여 분산 ML에서의 통신 효율성을 크게 개선. 대규모 머신러닝 워크로드를 효율적으로 처리하기 위한 시스템적 최적화를 포함한다.
- **중요도**: PS 아키텍처의 통신 최적화 기법을 이해하는 핵심 논문.

---

### 1.3 데이터 병렬화 및 AllReduce

#### [6] Horovod: Fast and Easy Distributed Deep Learning in TensorFlow
- **저자**: Alexander Sergeev, Mike Del Balso
- **연도/학회**: 2018, arXiv
- **요약**: Uber에서 개발한 분산 딥러닝 프레임워크. Baidu가 개척한 Ring-AllReduce 알고리즘을 활용하여 GPU 간 효율적인 그래디언트 평균화를 수행하며, 사용자 코드 수정을 최소화한다.
- **중요도**: Ring-AllReduce 기반 데이터 병렬 학습의 실질적 표준이 된 프레임워크. 간단한 API로 분산 학습의 진입 장벽을 획기적으로 낮추었다.

#### [7] BytePS: A Unified Architecture for Accelerating Distributed DNN Training in Heterogeneous GPU/CPU Clusters
- **저자**: Yimin Jiang, Yibo Zhu, Chang Lan, Bairen Yi, Yong Cui, Chuanxiong Guo
- **연도/학회**: 2020, OSDI (USENIX)
- **요약**: 이종 GPU/CPU 클러스터에서 분산 DNN 학습을 가속하는 통합 아키텍처 제안. 기존 AllReduce와 PS를 BytePS의 두 특수 케이스로 통합하며, CPU-GPU 및 GPU-GPU 통신의 최적 비율을 조정한다.
- **중요도**: AllReduce와 PS의 장점을 통합한 최적 통신 프레임워크. 256 GPU 환경에서 기존 대비 최대 84~245% 성능 향상을 달성했다.

### 1.4 모델 병렬화 및 파이프라인 병렬화

#### [8] Megatron-LM: Training Multi-Billion Parameter Language Models Using Model Parallelism
- **저자**: Mohammad Shoeybi, Mostofa Patwary, Raul Puri, Patrick LeGresley, Jared Casper, Bryan Catanzaro
- **연도/학회**: 2019, arXiv (NVIDIA Research)
- **요약**: 수십억 파라미터 규모 트랜스포머 모델 학습을 위한 효율적인 레이어 내 모델 병렬화(텐서 병렬화) 기법 제안. 512개 GPU로 83억 파라미터 모델을 학습하며 76% 스케일링 효율을 달성했다.
- **중요도**: 텐서 병렬화의 사실상 표준이 된 논문. 새로운 컴파일러나 라이브러리 변경 없이 PyTorch에서 간단한 통신 연산 삽입만으로 구현 가능하다.

#### [9] Efficient Large-Scale Language Model Training on GPU Clusters Using Megatron-LM
- **저자**: Deepak Narayanan, Mohammad Shoeybi, Jared Casper, Patrick LeGresley, Mostofa Patwary, Vijay Anand Korthikanti, Dmitri Vainbrand, Prethvi Kashinkunti, Julie Bernauer, Bryan Catanzaro, Amar Phanishayee, Matei Zaharia
- **연도/학회**: 2021, SC (Supercomputing Conference)
- **요약**: 텐서, 파이프라인, 데이터 병렬화를 결합한 3D 병렬화 기법을 GPU 클러스터에서 효율적으로 구현하는 방법을 제시. 1조 파라미터 규모 모델 학습을 가능하게 한다.
- **중요도**: 3D 병렬화(TP + PP + DP)의 실질적 구현 가이드. 현대 대규모 LLM 학습의 핵심 참고 논문.

#### [10] GPipe: Efficient Training of Giant Neural Networks using Pipeline Parallelism
- **저자**: Yanping Huang, Youlong Cheng, Ankur Bapna, Orhan Firat, Dehao Chen, Mia Chen, HyoukJoong Lee, Jiquan Ngiam, Quoc V. Le, Yonghui Wu, Zhifeng Chen
- **연도/학회**: 2019, NeurIPS (Google)
- **요약**: 마이크로배치 분할 기반 파이프라인 병렬화 라이브러리. 동기식 그래디언트 업데이트를 사용하면서도 디바이스 수에 거의 선형적인 속도 향상을 달성한다.
- **중요도**: 파이프라인 병렬화의 선구적 논문. 동기식 학습의 안정성과 파이프라인의 효율성을 동시에 달성하는 방법을 제시했다.

#### [11] PipeDream: Generalized Pipeline Parallelism for DNN Training
- **저자**: Deepak Narayanan, Aaron Harlap, Amar Phanishayee, Vivek Seshadri, Nikhil R. Devanur, Gregory R. Ganger, Phillip B. Gibbons, Matei Zaharia
- **연도/학회**: 2019, SOSP
- **요약**: 인터배치 파이프라이닝과 인트라배치 병렬화를 결합하여 학습 처리량을 극대화. 1F1B(1-Forward-1-Backward) 스케줄링으로 하드웨어 활용률을 높이면서 데이터 병렬화와 유사한 의미론을 유지한다.
- **중요도**: 비동기 파이프라인 병렬화의 대표 논문. GPipe 대비 1.7배 빠르며, 실제 대규모 학습 시스템에서 널리 채택된 기법을 제안했다.

### 1.5 메모리 최적화

#### [12] ZeRO: Memory Optimizations Toward Training Trillion Parameter Models
- **저자**: Samyam Rajbhandari, Jeff Rasley, Olatunji Ruwase, Yuxiong He
- **연도/학회**: 2020, SC (Supercomputing Conference) / Microsoft Research
- **요약**: 데이터 및 모델 병렬 학습에서 메모리 중복을 제거하는 ZeRO(Zero Redundancy Optimizer) 제안. 가중치, 그래디언트, 옵티마이저 상태를 디바이스 간 분할하여 메모리 소비를 획기적으로 줄인다.
- **중요도**: 1000억~1조 파라미터 모델 학습을 가능하게 한 핵심 기술. DeepSpeed 라이브러리에 통합되어 LLM 학습의 사실상 필수 도구가 되었다.

---

### 1.6 병렬화 전략 자동 탐색

#### [13] Beyond Data and Model Parallelism for Deep Neural Networks (FlexFlow)
- **저자**: Zhihao Jia, Matei Zaharia, Alex Aiken
- **연도/학회**: 2019, MLSys
- **요약**: SOAP(Sample, Operation, Attribute, Parameter) 차원의 포괄적 병렬화 탐색 공간을 정의하고, FlexFlow 프레임워크를 통해 자동으로 최적 병렬화 전략을 탐색. 실행 시뮬레이터로 기존 대비 1000배 빠르게 전략을 평가한다.
- **중요도**: 데이터/모델 병렬화를 넘어선 다차원 병렬화 개념을 제시. 자동 병렬화 전략 탐색 분야의 기초를 놓았다.

## 2. Mixture of Experts (MoE) 분산 학습

#### [14] GShard: Scaling Giant Models with Conditional Computation and Automatic Sharding
- **저자**: Dmitry Lepikhin, HyoukJoong Lee, Yuanzhong Xu, Dehao Chen, Orhan Firat, Yanping Huang, Maxim Krikun, Noam Shazeer, Zhifeng Chen
- **연도/학회**: 2020, ICLR (Google)
- **요약**: 조건부 계산 기반의 희소 MoE 트랜스포머를 6000억 파라미터까지 확장. 2048개 TPU v3에서 4일 만에 다국어 번역 모델을 학습하며, All-to-All 통신 패턴과 자동 샤딩 API를 확립했다.
- **중요도**: MoE의 대규모 분산 학습을 실증한 최초의 논문. Top-2 라우팅과 자동 파티셔닝 방식이 후속 MoE 연구의 기반이 되었다.

#### [15] Switch Transformers: Scaling to Trillion Parameter Models with Simple and Efficient Sparsity
- **저자**: William Fedus, Barret Zoph, Noam Shazeer
- **연도/학회**: 2022, JMLR (Google)
- **요약**: Top-1 라우팅으로 MoE 모델을 단순화하고 bfloat16 학습을 최초로 가능하게 함. 1조 파라미터 규모의 희소 모델을 효율적으로 학습할 수 있는 방법을 제시한다.
- **중요도**: MoE 아키텍처의 실용성을 크게 높인 논문. 라우팅 단순화와 안정적 학습 기법이 현대 MoE 모델(Mixtral 등)의 토대가 되었다.

## 3. 분산 추론 및 서빙 (Distributed Inference & Serving)

#### [16] Orca: A Distributed Serving System for Transformer-Based Generative Models
- **저자**: Gyeong-In Yu, Joo Seong Jeong, Geon-Woo Kim, Soojeong Kim, Byung-Gon Chun
- **연도/학회**: 2022, OSDI (USENIX)
- **요약**: 반복(iteration) 수준 스케줄링과 선택적 배칭을 제안하여 트랜스포머 기반 생성 모델의 서빙 효율을 획기적으로 개선. GPT-3 175B 모델에서 FasterTransformer 대비 36.9배 처리량 향상을 달성했다.
- **중요도**: 현대 LLM 서빙 시스템의 근간이 된 논문. Continuous batching 개념의 원조로, vLLM 등 후속 시스템에 지대한 영향을 주었다.

#### [17] Efficient Memory Management for Large Language Model Serving with PagedAttention (vLLM)
- **저자**: Woosuk Kwon, Zhuohan Li, Siyuan Zhuang, Ying Sheng, Lianmin Zheng, Cody Hao Yu, Joseph E. Gonzalez, Hao Zhang, Ion Stoica
- **연도/학회**: 2023, SOSP
- **요약**: 운영체제의 가상 메모리 및 페이징 기법에서 영감을 받은 PagedAttention 알고리즘 제안. KV 캐시 메모리 낭비를 거의 0으로 줄이고, 요청 간 KV 캐시 공유를 가능하게 한다.
- **중요도**: LLM 서빙의 게임 체인저. 기존 대비 2~4배 처리량 향상을 달성하며, 오픈소스 LLM 서빙의 사실상 표준(vLLM)이 되었다.

#### [18] Parallax: Efficient Distributed LLM Inference on Heterogeneous Hardware
- **저자**: Gradient Network Research Team
- **연도/학회**: 2024, Technical Report
- **요약**: 지리적으로 분산된 이종 하드웨어에서 파이프라인 병렬화, 동적 KV 캐시 관리, 연속 배칭을 통해 고처리량, 저지연 LLM 추론을 실현. End-to-end 지연 3.1배, 토큰 간 지연 5.3배 개선을 달성했다.
- **중요도**: 에지-클라우드 환경에서의 분산 추론 실용성을 입증한 최신 연구.

## 4. 연합 학습 (Federated Learning)

#### [19] Communication-Efficient Learning of Deep Networks from Decentralized Data (FedAvg)
- **저자**: H. Brendan McMahan, Eider Moore, Daniel Ramage, Seth Hampson, Blaise Agüera y Arcas
- **연도/학회**: 2017, AISTATS (Google)
- **요약**: 연합 학습의 창시 논문. 모바일 디바이스에 학습 데이터를 분산 유지한 채 로컬 업데이트를 집계하여 공유 모델을 학습하는 FedAvg 알고리즘을 제안. 통신 라운드를 획기적으로 줄이면서 데이터 프라이버시를 보장한다.
- **중요도**: 연합 학습 분야를 창시한 기념비적 논문. 프라이버시 보호 분산 학습의 새로운 패러다임을 열었으며, 수천 편의 후속 논문을 탄생시켰다.

#### [20] Federated Optimization in Heterogeneous Networks (FedProx)
- **저자**: Tian Li, Anit Kumar Sahu, Manzil Zaheer, Maziar Sanjabi, Ameet Talwalkar, Virginia Smith
- **연도/학회**: 2020, MLSys
- **요약**: FedAvg의 일반화 프레임워크로, 시스템 이질성(디바이스별 성능 차이)과 통계적 이질성(비IID 데이터)을 동시에 해결. 디바이스별 가변적 작업량을 허용하며, 이질적 환경에서 평균 22% 정확도 향상을 달성한다.
- **중요도**: 실제 연합 학습 환경의 핵심 과제인 이질성 문제를 정면으로 다룬 논문. FedAvg의 한계를 이론적, 실험적으로 보완한다.

#### [21] Advances and Open Problems in Federated Learning
- **저자**: Peter Kairouz, H. Brendan McMahan, et al. (58명 공동 저자)
- **연도/학회**: 2021, Foundations and Trends in Machine Learning
- **요약**: 연합 학습의 미해결 문제와 향후 연구 방향을 포괄적으로 정리한 대규모 서베이. 최적화, 프라이버시, 공정성, 시스템 설계 등 다양한 측면을 다룬다.
- **중요도**: FL 분야의 연구 로드맵 역할을 하는 필수 참고 논문.

## 5. 엣지 컴퓨팅 (Edge Computing)

### 5.1 모바일 엣지 컴퓨팅 (MEC)

#### [22] Mobile Edge Computing: A Survey
- **저자**: Nasir Abbas, Yan Zhang, Amir Taherkordi, Tor Skeie
- **연도/학회**: 2018, IEEE Internet of Things Journal
- **요약**: 모바일 네트워크 엣지에서 클라우드 서비스를 확장하는 MEC 아키텍처에 대한 포괄적 서베이. MEC의 정의, 아키텍처, 장점, 응용, 도전과제를 체계적으로 정리한다.
- **중요도**: MEC 분야의 대표적 서베이 논문. 엣지 컴퓨팅의 개념적 기초를 확립하는 데 기여했다.

#### [23] A Survey on Mobile Edge Computing: The Communication Perspective

#### [31] Energy Efficiency in Cloud Computing Data Centers: A Survey on Software Technologies
- **저자**: Various authors
- **연도/학회**: 2022, Cluster Computing (Springer)

- **중요도**: 데이터센터 에너지 효율 개선을 위한 소프트웨어적 접근법의 최신 현황을 파악하는 필수 논문.

- **연도/학회**: 2024, Software: Practice and Experience (Wiley)

- **중요도**: 클라우드 컴퓨팅의 지속 가능성 연구를 선도하는 Buyya 교수의 최신 비전 논문.

#### [34] Energy and Carbon-Aware Distributed ML Tasks Scheduling for Multi-Renewable Energy-Based Edge-Cloud Continuum

- **중요도**: 그린 AI와 엣지 컴퓨팅의 교차점을 다루는 최신 연구. 20~40% 탄소 배출 감소를 실증했다.

### 8.1 분산 강화학습

- **저자**: Lasse Espeholt, Hubert Soyer, Remi Munos, Karen Simonyan, Vlad Mnih, Tom Ward, Yotam Doron, Vlad Firoiu, Tim Harley, Iain Dunning, Shane Legg, Koray Kavukcuoglu

#### [45] Tiresias: A GPU Cluster Manager for Distributed Deep Learning
